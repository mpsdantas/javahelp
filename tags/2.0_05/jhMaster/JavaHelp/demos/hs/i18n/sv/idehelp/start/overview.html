Introduktion av Java Development Environment

I det här avsnittet beskriver vi begrepp och verktyg som används i Java Development Environments utvecklingsmiljö.
  • Verktyg i Java Development Environment introducerar huvudfunktionerna i Java WorkShop.
  • Utvecklingsstadier i projekts visar processen för att utveckla ett projekt med hjälp av verktygen i Java Development Environment.
  • Begrepp i Java Development Environment introducerar de termer som används för att beskriva komponenterna i Java Development Environment. Andra termer som är specifika för Java Development Environment (t.ex. miniprogram och tråd) förklaras när du träffar på dem för första gången i självstudien.
