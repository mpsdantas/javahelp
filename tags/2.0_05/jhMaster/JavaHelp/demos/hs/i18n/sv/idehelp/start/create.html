Övning 1: Skapa Blink-projektet

Den här övningen visar hur du skapar ett miniprogramprojekt, ett Java-program som utformas för att köras i en webbläsare. Projektet som du skapar, Blink, är ett litet miniprogram som visar blinkande text i olika färger. Du skapar projektet med en befintlig källfil, Blink.java, som har installerats i katalogen Blink.
  1. Klicka på projektknappen i verktygsfältet i Java Development Environments huvudfönster.
    Då visas fönstret Projekthanteraren.
  2. Leta efter namnet på den aktuella portfolio-filen i projekthanteraren. Den aktuella filen visas med blå text.
    Om den aktuella portfolio-filen inte är personlig dubbelklickar du på det personliga portfolionamnet. Om du inte har någon egen portfolio finns det mer information i Lägga till dinpersonliga portfolio.
  3. Välj Arkiv -> Nytt -> Projekt (var noga med att gå till Arkiv-menyn i projekthanteraren).
    Så öppnas guiden Skapa projekt.
  4. Markera namnlös och skriv Blink i rutan för projektnamnet.
    Var noga med att skriva ett versalt "B" i Blink. Java är skriftlägeskänsligt. Projektnamnet Blink överensstämmer med namnet på källfilen, Blink.java, som följer Java-reglerna för initial versal.
  5. Klicka på alternativknappen Miniprogram som motsvarar typen av tillämpning.
  6. Klicka på envals knappen Inget GUI.
    Det här projektet kommer inte att ha något grafiskt användargränssnitt.
  7. Klicka på Nästa.
  8. Du bör spara Blink-projektet i standardkatalogen.
  9. Klicka på Ja för att lägga till de befintliga Java-filerna i Blink-projektet.
  10. Klicka på Nästa.
  11. Klicka på knappen Lägg till alla i katalogen (under listan med projektfiler).
    Filnamnet Blink.java visas i listan med projektfiler.
  12. Klicka på Nästa.
    Ordet Blink infogas automatiskt i rutan Miniprogramklass. Huvudklassfilen är filen .class som utökar klassen Applet i Java-biblioteket.
  13. Klicka på Nästa.
  14. Du bör använda html-sidan som genereras av Java Development Environment.
  15. Klicka på Klar.
    Källfilen Blink.java visas i källredigeraren och filens namn visas i titelraden i Java Development Environment. I projekthanteraren visar den egna portfolio-filen Blink-projektet som det aktuella projektet. Det aktuella projektet anger sammanhanget för alla verktygen i Java Development Environment.

    I det här exemplet skapade du Blink-projektet i filen Blink.prj i katalogen Blink. När du skapar projekt i framtiden kan du skapa det i vilken katalog som helst i filsystemet.
