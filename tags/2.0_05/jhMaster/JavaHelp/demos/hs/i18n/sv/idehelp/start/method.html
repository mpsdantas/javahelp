Övning 2: Titta på metoderna i ditt projekt

I den här övningen kommer du att lista de metoder som definierats i klassen Blink. I denna lista kan du granska källkoden för metoden. För att kunna göra klart denna övning måste du vara färdig med föregående övning, Granska klasshierarkin.
  1. Rulla nedåt till etiketten Index konstruktör i klassläsaren.
    En konstruktör är en specialmetod som initialiserar ett objekt när det skapas. Klassläsaren listar en konstruktör, Blink, som definierats i klassen Blink.
  2. Klicka på Blink.
    Källredigeraren visar källkoden för konstruktören Blink. Konstruktörens första rad markeras i blått.
  3. Rulla nedåt till etiketten Metodindex i klassläsaren.
    Klassen Blink innehåller fem metoder: init, paint, start, stop, och kör.
  4. Klicka på paint.
    Källredigeraren visar källkoden för metoden paint. Den första raden markeras i blått.
