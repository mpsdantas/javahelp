Övning 1: Titta på klasshierarkin

I den här övningen kommer du att lära dig hur klasserna i projektet Blink härleds
  1. Kontrollera i Projekthanteraren efter namnet på aktuellt projekt.
    Om aktuellt projekt inte är Blink dubbelklickar du på projektnamnet Blink.
  2. Välj Bläddra -> Visa klasser.
    Java Development Environment öppnar klassläsaren, som listar klasser, metoder och kommentarer som ingår i projektet Blink. Projektet har en källfil: Blink.java.
  3. Klicka på Blink.java.
    Klassläsaren öppnar klasserna där Blink härleds ifrån.
    Blink härleds från Applet, som härleds från Panel, osv.

    Informationen i Klassläsaren härleds från javadoc, Java API dokumentationsgenerator. Javadoc analyserar förklaringen och kommentarerna i projektets källkod och formaterar API till HTML-kod.
