Inställningsmeny

Proxies Använd dialogrutan Proxies för att ändra proxy-servrar (proxies).
Versionskontroll Väljer ett versionskontrollsystem som ska användas med Java Development Environment.
Källredigerare Anger inställningar för följande funktioner vid källredigering:
  • Indrag/tabbar
  • Syntaxmarkering och färger
  • Välja teckensnitt
Tangentbord Granska, lägga till eller ta bort tangenttilldelning (kopplingar) för funktionella grupper inom Java Development Environment. Se Inställningar tangentbord för ytterligare information.
Indata i systemfönstret Bestämmer hur inmatningen av kommandoradsbaserade program tolkas i systemfönstret. Se dialogrutan Inställningar programbyggare ang. ytterligare information.
Programbyggare Bestämmer vilken Javakompilator som Java Development Environment använder för att kompilera dina projekt. Se dialogrutan Inställningar programbyggare ang. ytterligare information.
Visa etiketter i verktygsfältet Lägger till etiketter nedanför verktygsfältets knappar.
