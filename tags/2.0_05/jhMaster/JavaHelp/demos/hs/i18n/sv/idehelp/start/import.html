Lägga till din personliga portfolio

Första gången du kör Java Development Environment så skapas en portfolio med namnet "personlig" och till den har du skrivrättigheter. Om du tar bort din personliga portfolio, men inte raderar ut själva filen (.psf) från filsystemet, kan du importera den tillbaka till Java Development Environment.

I den här självstudien kan du dock använda vilken portfolio som helst där du har skrivrättigheter. Eller också kan du skapa en ny portfolio med hjälp av Skapa en portfolio.

Så här lägger du till din personliga portfolio:

  1. I Java Development Environments Projekthanterare väljer du Arkiv -> Lägg till -> Portfolio.
    Dialogrutan Lägg till portfolio visas.
  2. Skriv in /your_installation_directory/jws/personal.psf i filrutan för portfolio.
    Alla filnamn för portfolio slutar med filnamnstillägget .psf.
  3. Klicka på Lägg till.
    Java Development Environment låter nu din personliga portfolio bli den aktuella och visar dess namn i blåfärgad text.
