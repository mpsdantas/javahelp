Java Development Environment Självstudier

Syftet med självstudierna är att visa hur du kan använda de grundläggandee funktionerna i Java Development Environment för att skapa, redigera, bygga, felsöka och köra projekt.

I det första avsnittet får du bekanta dig med begreppen, verktygen och utvecklingsproceduren som används i Java Development Environment. En beskrivning av de grundläggandee funktionerna i Java Development Environment finns i självstudierna 1-5. I avsnitten 6 och 7 beskrivs avancerade funktioner i Java Development Environment.

  • Introduktion tilll Java Development Environment tar upp hur du kan använda Java Development Environment för att skapa Internet-tillämpningar och webbsidor.
  • Självstudie ett: Skapa Blink-projektet visar hur du skapar, bygger och kör miniprogram i Java Development Environment.
  • Självstudie två: Redigera projektattribut går igenom hur du redigerar projektet.
  • Självstudie tre: Åtgärda fel i källkod beskriver hur du rättar fel när du bygger projektet, och hur du bygger om dina projekt.
  • Självstudie fyra: Bläddra igenom ett projekt visar hur du går igenom klasserna och metoderna som utgör projektet. Här får du också lära dig att söka efter strängar i källkoden.
  • Självstudie fem: Felsöka ett projekt förklarar hur du letar upp och korrigerar fel som gör att projektet inte körs korrekt. Här använder du felsökaren för att granska trådar, ange brytpunkter och utvärdera uttryck.
  • Självstudie sex: Anpassa en körsida beskriver hur du redigerar HTML-filen som Java WorkShop skapar åt dig när du kör ett projekt.
  • Självstudie sju: Skapa ett GUI tar upp hur du skapar ett användargränssnitt till ett projekt.
  • Självstudie åtta: Skapa Clock-projektet handlar om hur du skapar ett bönprojekt och importerar JavaBeans till GUI-byggaren.
