Begrepp i Java Development Environment

Följande ord och begrepp används i online-hjälpen. Det är lättare att gå igenom självstudien om du förstår dessa begrepp. Andra termer som är specifika för Java Development Environment (exempelvis miniprogram och tråd) förklaras där de förekommer i självstudien.

När du går igenom självstudien bör du hålla ett öga på den animerade ikonen längst upp till höger i läsaren. Kuben snurrar när Java WorkShop öppnar en hjälpsida.

Knapp
Ett element i ett verktygsfält eller fönster som du kan klicka på för att utföra en åtgärd. Om du exempelvis klickar på knappen Projekt visas projekthanteraren i Java Development Environment.

Det visas verktygstips till alla knapparna i verktygsfälten. Om du inte är säker på vad en viss verktygsknapp har för funktion kan du placera pekaren över knappen för att få en beskrivning av den. Om du exempelvis placerar pekaren över knappen visas texten "Sök sträng".

Meny
En lista med poster där du kan välja en viss uppgift eller ett visst objekt. Menyerna visas i läsaren mellan titelraden och huvudverktygsfältet i Java Development Environment. Hänvisningar till menyposterna sker ofta enligt formen Meny -> Menypost. Exempel: Projekt -> Bygg betyder "välj posten Bygg på Projekt-menyn".
Flik
En sida med flikar (som liknar en dialogruta) som visas när du klickar på en flik i ett fönster. Fönstret Redigera projekt innehåller exempelvis följande flikar: Allmänt, Bygg, Felsök/bläddra och Kör.
