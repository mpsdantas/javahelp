Tutorial Uno: Creación de un proyecto

Este tutorial está diseñado para los nuevos usuarios de Java Development Environment. En este tutorial, aprenderá qué es un proyecto y cómo crear, construir, y ejecutar un proyecto en Java Development Environment.

El tutorial Uno está dividido en cinco lecciones. Para sacar el máximo partido del tutorial, deberá realizar los ejercicios de las lecciones siguiendo el orden en que se presentan.

  • Conceptos importantes: Proyectos y carteras ofrece una visión general completa sobre el funcionamiento de los proyectos y carteras. Deberá entender bien estos conceptos antes de pasar a otra lección de este tutorial.
  • Ejercicio 1: Creación del proyecto Blink muestra cómo crear un proyecto que contiene un applet. Un applet es un pequeño programa que se ejecuta en un explorador que soporta el lenguaje Java.
  • Ejercicio 2: Construcción del proyecto muestra cómo construir el proyecto del applet.
  • Ejercicio 3: Ejecución del proyecto muestra cómo ejecutar el proyecto del applet y cómo verlo en un visualizador de applets.
  • Ejercicio 4: Recopilación de datos de perfil muestra cómo recopilar información de perfil.
