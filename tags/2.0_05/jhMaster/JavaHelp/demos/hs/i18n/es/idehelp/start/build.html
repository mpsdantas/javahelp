Ejercicio 2: Construcción del proyecto

En este ejercicio se explica cómo construir el proyecto Blink. Deberá crear un proyecto para poder ejecutarlo. Cuando construye un proyecto, Java WorkShop compila el código fuente Java del proyecto en un archivo llamado archivo.class.

Si no ha creado todavía el proyecto Blink, deberá hacerlo ahora siguiendo los pasos que se indican en Creación del proyecto Blink.

  1. Consulte en la ventana del administrador de proyectos de Java Development Environment el nombre del proyecto actual.
    Si el proyecto actual no es Blink, pulse dos veces en el nombre de proyecto Blink.
  2. Pulse en el botón "Construir" de la barra de herramientas principal de Java Development Environment.
    Java Development Environment compila el programa Blink.java en el archivo Blink.class de código de bytes. Observe la pestaña "Construir" (debajo del editor de código fuente) mientras Java Development Environment construye el proyecto. Cuando el código fuente se haya terminado de construir y compilar, aparecerá el mensaje de estado "Terminado" en la pestaña "Construir".
