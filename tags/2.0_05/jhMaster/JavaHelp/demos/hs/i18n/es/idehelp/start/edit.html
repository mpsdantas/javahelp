Tutorial Dos: Edición de atributos de proyecto

En este tutorial, aprenderá qué son los atributos de proyecto, y cómo editar los atributos de ejecución y construcción. Este tutorial está dividido en tres lecciones. Para sacar el máximo partido del tutorial, revise primero la descripción de los atributos de proyecto y, a continuación, haga los dos ejercicios que se proponen.
  • Conceptos importantes: Atributos de proyecto explica qué son los atributos de proyecto y cómo se utilizan en Java WorkShop. Deberá entender bien estos conceptos antes de pasar al Ejercicio 1.
  • Ejercicio 1: Edición de atributos de ejecución muestra cómo especificar la forma en que se ejecuta el applet.
  • Ejercicio 2: Edición de atributos de construcción muestra cómo especificar diferencias de construcción.
