Cadena de búsqueda
Activa el cuadro de diálogo Cadena de búsqueda. Puede buscar un texto específico dentro de un proyecto o directorio. Este comando también tiene un botón de acceso rápido en la barra de herramientas principal.
Parar búsqueda Para la búsqueda actual.
Coincidencia siguiente Muestra la siguiente cadena coincidente con la del resultado de la búsqueda. La siguiente coincidencia es la que aparece en la lista después de la coincidencia actual en la pestaña Cadena de búsqueda. Si es necesario, se carga un nuevo archivo en el editor.
Coincidencia previa Muestra la cadena anterior coincidente con la del resultado de la búsqueda. La coincidencia anterior es la que aparece en la lista antes de la coincidencia actual en la pestaña "Cadena de búsqueda". Si es necesario, se carga un nuevo archivo en el editor.
Ver clases Abre el visualizador de clases. Puede utilizar el visualizador de clases para examinar las clases, métodos y comentarios de sus proyectos. El visualizador de clases se basa en javadoc, el Generador de Documentación de API de Java.
