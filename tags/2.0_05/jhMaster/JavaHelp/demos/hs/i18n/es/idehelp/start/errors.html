Tutorial Tres: Corrección de errores de código fuente

Las lecciones de este tutorial le enseñarán a editar código fuente en el proyecto Blink y a reconstruir el proyecto. Para sacar el máximo partido del tutorial, deberá realizar los ejercicios de las lecciones siguiendo el orden en que se presentan.
  • Ejercicio 1: Edición de código fuente le presenta algunas características básicas del editor de código fuente de Java Development Environment.
  • Ejercicio 2: Corrección de errores de una construcción corrige errores del compilador que se han encontrado durante el proceso de construcción.
