Ejercicio Dos: Edición de atributos de construcción

En este ejercicio, aprenderá cómo construye el programa Java Development Environment.

Este proyecto opera con el proyecto Blink; es preciso también haber realizado los ejercicios de las lecciones anteriores de los Tutoriales Uno y Dos. Si no ha editado los atributos de ejecución en el ejercicio anterior, deberá hacerlo ahora siguiendo las instrucciones que se indican en Edición de atributos de ejecución.

  1. Consulte en el administrador de proyectos el nombre del proyecto actual.
    Si el proyecto actual no es Blink, pulse dos veces en el nombre del proyecto Blink.
  2. En el administrador de proyectos, seleccione "Proyecto -> Editar".
    En la ventana "Editar proyecto" hay cuatro pestañas: "General", "Construir", "Depurar/Examinar" y "Ejecutar".
  3. Pulse en la pestaña "Construir".
    Los atributos de esta pestaña se utilizan para definir cómo desea construir el programa.
  4. Pulse en "Opciones".
    Aparece el cuadro de diálogo "Opciones del compilador de Java"
  5. Asegúrese de que la opción "Habilitar generación de tablas de depuración" está seleccionada.
  6. Pulse en "Aceptar".
  7. Pulse en "Aceptar" en la ventana "Editar proyecto".
