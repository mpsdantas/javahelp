제 1과: 실행 속성 수정하기

자습서 1에서 생성된 <applet> 태그는 웹 페이지에 애플릿을 올려 놓는데 필요한 최소의 요구 조건을 지정하는 것입니다. 이 과에서는, 애플릿을 화면에 표시하는 방식을 사용자가 변경할 수 있는 매개 변수들을 추가하게 될 것입니다.

이 과에서는 Blink 프로젝트를 사용합니다. 아직 Blink 프로젝트를 만들어서 빌드하지 않았다면, 이제 Blink 프로젝트 만들기에 나오는 지침을 사용하여 그렇게 해야 합니다.

  1. 프로젝트 관리자에서 현재 프로젝트의 이름을 확인하십시오.
    현재 프로젝트가 Blink가 아니라면, Blink라는 프로젝트 이름을 마우스로 두 번 연속 누르십시오.
  2. 프로젝트 관리자에서, [프로젝트] -> [편집]을 선택하십시오.
    [프로젝트 편집] 창에 [일반], [빌드], [디버그/찾아보기], [실행] 등의 네 개의 탭이 나타납니다.
  3. [실행] 탭을 마우스로 누르십시오.
    이 탭에 나오는 속성들은 Blink.tmp.html 파일에 <applet> 태그를 생성하는데 사용됩니다.
  4. (창의 중간 쯤에 있는) [매개 변수] 필드를 찾으십시오.
    현재로서는, 애플릿에 대해 정의된 매개 변수가 아무 것도 없습니다. 매개 변수는 애플릿이 화면에 나타나는 방식을 정의하는 비표준 속성입니다. 매개 변수에는 이름과 값이 있습니다. 이제 매개 변수를 추가하여 깜박거리는 텍스트의 단어들을 변경해 봅시다.
  5. [매개 변수] 필드의 오른쪽에 있는 [이름] 입력란에 lbl ("label"의 약어)이라고 입력하십시오.
  6. [값] 란에 다음 텍스트를 입력하십시오:
    Java Development Environment is really cool :-)
  7. [추가] 단추를 마우스로 누르십시오.
    [매개 변수] 필드에 lbl=Java Development Environment is really cool :-)라는 라인이 나타날 것입니다.
  8. [확인]을 마우스로 눌러 변경한 내용을 저장하십시오.
  9. 주 도구 모음에서 [실행] 단추를 마우스로 누르십시오.
    Blink.tmp.html 파일의 <applet> 태그 안에: 
    <param name="lbl" value="Java Development Environment is really cool :-)">
    라는 라인이 들어갈 것입니다. 이제 이 파일이 브라우저로 로딩되고, 여러 가지 색으로 깜박거리는 "Java Development Environment is really cool :-)"라는 단어들이 나타날 것입니다.

    실행 중인 애플릿을 멈추려면, [애플릿 뷰어] 창에서 [애플릿] -> [종료]를 선택하십시오.
