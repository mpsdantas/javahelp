제 3과: 코드에서 문자열 찾기

이 과에서는, 소스 코드에서 지정된 문자열이 나오는 모든 곳을 찾아내는 방법을 익힐 것입니다. 그 다음에 그 일치하는 문자열을 앞뒤의 소스 코드와 함께 볼 수 있습니다.
  1. 프로젝트 관리자에서 현재 프로젝트의 이름을 확인하십시오.
    현재 프로젝트가 Blink가 아니라면, Blink라는 프로젝트 이름을 마우스로 두 번 연속 누르십시오.
  2. Java Development Environment 주 도구 모음에서 [검색] 단추 를 마우스로 누르십시오.
  3. [문자열] 란에 green이라고 입력하십시오.
  4. [검색]을 마우스로 누르십시오.
    소스 코드에서 해당 문자열이 나오는 모든 곳을 찾습니다. 문자열 검색기는 green이라는 문자열이 나오는 두 곳을 찾아내어 소스 편집기 창의 아래에 있는 [문자열 찾기] 탭에 그 일치하는 위치를 표시합니다. 일치하는 위치는 filename: line_number의 형태로 표시됩니다. 해당 문자열이 들어 있는 소스 코드 라인도 나타날 것입니다.
  5. 첫번째 일치하는 위치의 링크인 Blink.java:32를 마우스로 누르십시오.
    소스 편집기에 해당 문자열이 들어 있는 소스 라인이 나타납니다. 이 라인은 파랑색으로 반전 표시됩니다.
  6. [찾아보기] -> [다음 찾기]를 선택하십시오.
    소스 편집기에는 그 다음 일치하는 위치의 코드가 반전 표시됩니다.
