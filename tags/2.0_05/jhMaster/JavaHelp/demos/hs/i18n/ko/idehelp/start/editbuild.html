제 2과: 빌드 속성 수정하기

이 과에서는, Java Development Environment이 프로그램을 빌드하는 방식을 수정할 것입니다.

이 프로젝트에서는 Blink 프로젝트를 사용하며 자습서 1과 2에 나오는 과들과 연결이 됩니다. 앞 과에서 실행 속성들을 수정하지 않았다면, 지금 실행 속성 수정하기에 나오는 지침을 사용하여 수정해야 합니다.

  1. 프로젝트 관리자에서 현재 프로젝트의 이름을 확인하십시오.
    현재 프로젝트가 Blink가 아니라면, Blink라는 프로젝트 이름을 마우스로 두 번 연속 누르십시오.
  2. 프로젝트 관리자에서, [프로젝트] -> [편집]을 선택하십시오.
    [프로젝트 편집] 창에 [일반], [빌드], [디버그/찾아보기], [실행] 등의 네 개의 탭이 나타납니다.
  3. [빌드] 탭을 마우스로 누르십시오.
    이 탭에 나오는 속성들은 프로그램을 빌드하는 방식을 정의하는데 사용됩니다.
  4. [옵션]을 마우스로 누르십시오.
    [Java Compiler 옵션] 대화 상자가 화면에 나타납니다.
  5. [디버깅 테이블 생성 가능]이 선택되어 있는지 확인하십시오.
  6. [확인]을 마우스로 누르십시오.
  7. [프로젝트 편집] 창에서 [확인]을 마우스로 누르십시오.
