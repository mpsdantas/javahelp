제 2과: 프로젝트 빌드하기

이 과에서는 Blink 프로젝트를 빌드하는 방법을 설명합니다. 실행할 수 있는 프로젝트를 빌드할 것입니다. 프로젝트를 빌드하면, 그 프로젝트의 Java 소스 코드가 컴파일되어 filename.class라고 하는 파일이 됩니다.

아직 Blink 프로젝트를 만들지 않았다면, 이제 Blink 프로젝트 만들기에 나오는 방법을 사용하여 그렇게 해야 합니다.

  1. [Java Development Environment 프로젝트 관리자] 창에서 현재 프로젝트의 이름을 확인하십시오.
    현재 프로젝트가 Blink가 아니라면, Blink라는 프로젝트 이름을 마우스로 두 번 연속 누르십시오.
  2. Java Development Environment 주 도구 모음에서 [빌드] 단추 를 마우스로 누르십시오.
    Java Development Environment이 Blink.java 프로그램을 컴파일하여 바이트코드로 된 Blink.class 파일로 만듭니다. Java Development Environment이 프로젝트를 빌드할 때 (소스 편집기 아래에 있는) [빌드] 탭을 관찰해 보십시오. 소스의 빌드와 컴파일을 완료되면 [빌드] 탭에 "완료"라는 상태 메시지가 나타납니다.
