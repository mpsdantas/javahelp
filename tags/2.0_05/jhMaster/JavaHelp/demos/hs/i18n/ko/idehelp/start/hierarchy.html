제 1과: 클래스 계층 구조 보기

이 과에서는, Blink 프로젝트의 클래스들을 도출하는 방법을 익힐 것입니다.
  1. 프로젝트 관리자에서 현재 프로젝트의 이름을 확인하십시오.
    현재 프로젝트가 Blink가 아니라면, Blink라는 프로젝트 이름을 마우스로 두 번 연속 누르십시오.
  2. [찾아보기] -> [클래스 보기]를 선택하십시오.
    Blink 프로젝트를 구성하는 클래스, 메서드, 주석 등의 목록이 나오는 클래스 뷰어가 열립니다. 이 프로젝트에는 Blink.java라는 하나의 소스 파일만 있습니다.
  3. Blink.java를 마우스로 누르십시오.
    클래스 뷰어에 Blink가 도출된 클래스가 로딩됩니다.
    Blink는 Applet에서 도출되고, Applet은 Panel에서 도출되는 식입니다.

    클래스 뷰어 정보는 Java API Documentation Generator(설명서 생성기)인 javadoc에서 도출됩니다. Javadoc은 프로젝트의 소스 코드에 있는 선언문과 주석을 분석하여 API를 HTML 코드 포맷으로 만듭니다.
