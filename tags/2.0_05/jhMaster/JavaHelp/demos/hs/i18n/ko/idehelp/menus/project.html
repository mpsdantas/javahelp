[프로젝트] 메뉴

[프로젝트 관리자 보기] 이 메뉴 명령을 선택하면 [Java Development Environment 프로젝트 관리자]가 열립니다.
[편집] 이 메뉴 명령을 선택하면 [프로젝트 편집] 창이 열립니다. 이 창의 탭들을 사용하여 프로젝트의 속성을 수정하십시오. 더 자세한 내용은 프로젝트 속성 수정하기를 참조하십시오.
[빌드] 이 메뉴 명령을 선택하면 현재 프로젝트를 빌드합니다.
[실행] 이 메뉴 명령을 선택하면 현재 프로젝트를 실행합니다.
[성능 분석] 이 메뉴 명령을 선택하면 성능 분석 도구를 사용하여 현재 프로젝트를 실행합니다.
[프로그램 멈춤] 이 메뉴 명령을 선택하면 현재 프로젝트 실행을 멈춥니다.
[찾아보기] 이 메뉴 명령은 소스 뷰어 제어 명령입니다. 더 자세한 내용은 소스 찾아보기를 참조하십시오.
[애플릿 태그 복사] 이 메뉴 명령을 선택하면 작업 애플릿 프로젝트에서 애플릿 태그를 복사합니다.
[파일 컴파일] 이 메뉴 명령을 선택하면 .prj 파일을 컴파일합니다.
[새로 만들기] 이 메뉴 명령을 선택하면 [프로젝트 만들기 마법사]가 열립니다.
[추가] 이 메뉴 명령을 선택하면 [프로젝트 추가] 마법사가 열립니다.
