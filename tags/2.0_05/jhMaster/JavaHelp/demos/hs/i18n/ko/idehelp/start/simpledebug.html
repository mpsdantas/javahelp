자습서 5: 프로젝트를 디버깅하기

본 자습서에서는 Java Development Environment에서 디버깅하는 방법을 소개합니다. 프로그램에 있는 스레드들을 검사하고, 중단점을 설정하고, 프로그램의 데이터를 검사하는 방법을 익힐 것입니다.

본 자습서는 일곱 개의 과로 구성되어 있습니다. 각 과를 차례대로 익혀야 합니다.

  • 중요한 개념: 스레드는 스레드와 스레드 그룹을 간단하게 정의합니다. 다음 과들로 넘어가기 전에 이러한 개념들을 이해해야 합니다.
  • 제 1과: 디버깅 세션 시작하기에서는 프로젝트 디버깅을 시작하는 방법을 알려줍니다.
  • 제 2과: 중단점 설정하기에서는 코드 내의 특정한 지점에서 프로그램 실행을 일시 정지시키는 방법을 설명합니다.
  • 제 3과: 스레드 검사하기에서는 스레드 그룹, 스레드, 그리고 프로젝트를 구성하는 스택 프레임 등을 보는 방법을 보여 줍니다.
  • 제 4과: 메서드를 프로시저 단위로 실행하기에서는 코드를 단계적으로 실행하는 방법을 보여 줍니다.
  • 제 5과: 표현식 평가하기에서는 프로그램의 데이터의 값을 점검하는 방법을 보여 줍니다.
  • 제 6과: 디버깅 세션 끝내기에서는 디버깅 세션을 끝마치는 방법을 보여 줍니다.
