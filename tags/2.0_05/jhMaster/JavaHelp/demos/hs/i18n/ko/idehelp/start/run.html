제 3과: 프로젝트 실행하기

이 과에서는, Java Development Environment이 만든 HTML 파일을 사용하여 Blink 프로젝트를 실행할 것입니다.
  1. 프로젝트 관리자에서 현재 프로젝트의 이름을 확인하십시오.
    현재 프로젝트가 Blink가 아니라면, Blink라는 프로젝트 이름을 마우스로 두 번 연속 누르십시오.
  2. Java Development Environment 주 도구 모음에서 [실행] 단추 를 마우스로 누르십시오.
    Java Development Environment은 Blink 디렉터리에 HTML 파일을 생성합니다. 이 HTML 파일에는 웹 페이지에 Blink 프로젝트를 올려 놓을 때 사용하는 옵션들을 지정하는 <애플릿> 태그가 들어 있습니다. Java Development Environment는 파일을 애플릿 뷰어로 로딩하므로, Blink 프로젝트가 실행되는 것이 보일 것입니다. "Blink"라는 단어가 여러 가지 색으로 깜박거립니다.
  3. 실행 중인 애플릿을 멈추려면, [애플릿 뷰어] 창에서 [애플릿] -> [종료]를 선택하십시오.
