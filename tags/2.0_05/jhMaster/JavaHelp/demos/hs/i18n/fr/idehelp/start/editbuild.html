Exercice Deux : Edition des attributs de génération

Dans cet exercice, vous personnaliserez la manière dont Java Development Environment génère le programme.

Ce projet fonctionne avec le projet Blink et les leçons précédentes des Didacticiels Un et Deux. Si vous n'avez pas édité les attributs d'exécution, au cours de l'exercice précédent, vous devriez le faire en utilisant les instructions de Edition des attributs d'exécution.

  1. Vérifiez dans le gestionnaire de projet le nom du projet en cours.
    Si celui-ci n'est pas Blink, double-cliquez sur le nom de projet Blink.
  2. Dans le gestionnaire de projet, choisissez Projet -> Editer.
    La fenêtre Edition de projet présente quatre onglets : Généralités, Générer, Déboguer/Parcourir et Exécuter.
  3. Cliquez sur l'onglet Générer.
    Les attributs proposés dans cet onglet servent à définir la manière dont le programme sera généré.
  4. Cliquez sur Options.
    La boîte de dialogue Options du compilateur Java apparaît.
  5. Autoriser la génération des tables de débogage doit être sélectionné.
  6. Cliquez sur OK.
  7. Cliquez sur OK dans la fenêtre Edition de projet.
