Terminologie Java Development Environment

Les termes suivants sont utilisés partout dans l'aide en ligne. Leur compréhension peut vous aider à compléter plus rapidement le didacticiel. D'autres termes spécifiques à Java Development Environment (comme appliquette ou thread) seront expliqués lors de leur première occurrence dans le didacticiel.

Lors de votre progression dans le didacticiel, observez l'icône animée dans l'angle supérieur droit du navigateur. Le cube effectue une rotation quand Java Development Environment charge une page de l'aide.

Bouton
Elément soit d'une barre d'outils, soit d'une fenêtre, sur lequel vous pouvez cliquer pour déclencher une action. Si vous cliquez, par exemple, sur le bouton Projets la fenêtre du gestionnaire de projet de Java Development Environment s'affiche.

Des infobulles accompagnent les boutons des barres d'outils. Si vous n'êtes pas sûr de l'action d'un bouton de la barre d'outils, placez le pointeur sur celui-ci pour afficher sa description. Si vous placez, par exemple, le pointeur sur le bouton « Chaîne de recherche » apparaît à l'écran.

Menu
Liste d'éléments parmi lesquels vous pouvez choisir une tâche spécifique ou un objet. Les menus sont disposés dans le navigateur entre la barre de titre et la barre d'outils principale de Java Development Environment. Vous rencontrerez fréquemment des références à une option de menu sous la forme Menu -> Option de menu. Par exemple, Projet -> Générer signifie « choisir l'option Générer dans le menu Projet ».
Onglet
Page à onglets (semblable à une boîte de dialogue) affichée lorsque vous cliquez sur un onglet d'une fenêtre. La fenêtre Edition de projet, par exemple, comporte les onglets suivants : Général, Générer, Déboguer/Parcourir et Exécuter.
