Exercice 1 : Edition du fichier HTML

Au cours de cette leçon, vous éditerez le fichier HTML que Java Development Environment a créé pour vous lorsque vous avez exécuté le projet Blink. Vous spécifierez un paramètre d'appliquette et ajouterez un lien au code source. Une fois ce fichier édité, enregistrez-le sous un nom différent, faute de quoi vos modifications seront écrasées lors de la prochaine exécution du projet.

Si vous n'avez pas encore créé et construit ce projet, faites cela maintenant en suivant les instructions de Création du projet Blink.

  1. Vérifiez dans le gestionnaire de projet le nom du projet en cours.
    Si celui-ci n'est pas Blink, double-cliquez sur le nom de projet Blink.
  2. Dans l'éditeur de source, sélectionnez Fichier -> Ouvrir.
    Java Development Environment ouvre un sélecteur de fichier.
  3. Ouvrez le fichier Blink.tmp.html dans votre répertoire Blink.
    Le fichier contient une balise d'appliquette se présentant comme suit : 
    <applet
   name="Blink"
   code="Blink.class"
   codebase="/votre_rép_d'origine/jws/Blink"
   width="500"
   height="600"
   align="Top"
   alt="Si vous aviez un explorateur compatible Java, vous verriez une appliquette ici."
>
<param name="lbl" value="Java Development Environment est vraiment super :-)">
<hr>Si votre explorateur a reconnu la balise de l'appliquette,
      vous voyez une appliquette ici.<hr>
</applet>

    Java Development Environment a créé ce fichier lorsque vous avez exécuté le projet. Les informations qu'il contient sont basées sur les informations se trouvant dans l'onglet Exécuter de la fenêtre Edition de projet. Si vous n'avez pas exécuté le projet auparavant, reportez-vous à Exécution du projet.

  4. Choisissez Fichier -> Enregistrer sous dans la barre de menu de Java Development Environment.
    La boîte de dialogue Enregistrement du fichier en cours apparaît.
  5. Sélectionnez le répertoire Blink, puis entrez un nouveau nom pour le fichier HTML et cliquez sur OK.
    Vous devez donner un nouveau nom au fichier HTML, faute de quoi vos modifications seront écrasées la prochaine fois que vous exécuterez le projet. Vous devez aussi enregistrer le fichier dans votre répertoire Blink.
  6. Dans l'éditeur de source, remplacez la valeur de hauteur par 75.
  7. Ajoutez le paramètre suivant avant le paramètre lbl afin d'accroître la vitesse de clignotement des mots : 
    <param name="speed" value="50">
  8. Ajoutez le lien suivant au code source après la balise <applet> : 
    <p>
<a href="Blink.java">Source Code</a>
    Le chemin d'accès au code source doit être relatif au répertoire contenant le fichier HTML.
  9. Cliquez sur le bouton Enregistrer de la barre d'outils Edition/Débogage.
