Exercice 1 : Edition du code source

Pour vous familiariser avec le travail effectué avec l'éditeur de source, vous entrerez dans le fichier Blink.java un code qui aura pour effet de faire rebondir les mots.

Si vous n'avez pas encore créé le projet Blink, faites-le maintenant en suivant les instructions données dans Création du projet Blink.

  1. Vérifiez dans la fenêtre Gestionnaire de projet de Java Development Environment le nom du projet en cours.
    Si celui-ci n'est pas Blink, double-cliquez sur le nom de projet Blink.
  2. Entrez les lignes de code source suivantes.
    Le code comporte intentionnellement des erreurs ; en effet, dans la leçon suivante vous apprendrez à corriger les erreurs.

    Vous pouvez atteindre un numéro de ligne spécifique en choisissant Editer -> Aller à la ligne. Le numéro de ligne où se trouve le pointeur est indiqué à droite en bas de l'écran.

  3. Cliquez sur le bouton Enregistrer de la barre d'outils Edition/Débogage.
    Le chemin d'accès du fichier enregistré apparaît au bas de l'écran de l'éditeur de source.
