Chaîne de recherche
Active la boîte de dialogue Chaîne de recherche. Vous pouvez rechercher un texte spécifié à l'intérieur d'un projet ou d'un répertoire. Cette commande est également accessible par un bouton de raccourci à partir de la barre d'outils principale.
Arrêter la recherche Arrête la recherche en cours.
Correspondance suivante Affiche l'occurrence suivante résultant de la recherche d'une chaîne. L'occurrence suivante est répertoriée après l'occurrence courante dans l'onglet Chaîne de recherche. Le cas échéant, un nouveau fichier est chargé dans l'éditeur.
Correspondance précédente Affiche l'occurrence précédente résultant de la recherche d'une chaîne. L'occurrence précédente est répertoriée avant l'occurrence courante dans l'onglet Chaîne de recherche. Le cas échéant, un nouveau fichier est chargé dans l'éditeur.
Afficher les classes Ouvre l'afficheur de classe. Celui-ci vous permet d'explorer les classes, les méthodes et les commentaires de vos projets. Cet afficheur est basé sur javadoc, le générateur de documentation d'API Java.
