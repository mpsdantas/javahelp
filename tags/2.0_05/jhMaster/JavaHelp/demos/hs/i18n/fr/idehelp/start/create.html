Exercice 1 : Création du projet Blink

Cet exercice montre comment créer un projet appliquette, c'est-à-dire un programme Java s'exécutant dans un navigateur Web. Le projet que vous produirez, Blink, sera une petite appliquette ayant pour effet de faire clignoter le texte dans différentes couleurs. Vous créerez le projet à partir d'un fichier source existant, Blink.java, lequel est installé dans votre répertoire Blink.
  1. Cliquez sur le bouton Projets de la barre d'outils principale de Java WorkShop.
    La fenêtre Gestionnaire de projet s'affiche.
  2. Vérifiez dans le gestionnaire le nom du portfolio courant (identifié par des caractères bleus).
    Si le portfolio courant n'est pas personal, double-cliquez sur le nom de votre portfolio personnel. Si vous n'en avez pas, reportez-vous à Ajout d'un portfolio personnel.
  3. Choisissez Fichier -> Nouveau -> Projet (vous devez aller dans le menu Fichier de la fenêtre Gestionnaire de projet).
    L'assistant création de projet s'ouvre.
  4. Sélectionnez untitled et entrez Blink dans la zone de nom du projet.
    N'oubliez pas d'utiliser un "B" majuscule pour Blink. En effet, Java fait la distinction entre majuscules et minuscules. Le nom de projet Blink concorde avec le nom du fichier source Blink.java, qui est conforme à la convention Java de dénomination avec majuscule en début de mot.
  5. Cliquez sur le bouton radio Appliquette pour désigner le type d'application.
  6. Cliquez sur le bouton radio Pas d'interface.
    Ce projet ne comportera pas d'interface graphique utilisateur.
  7. Cliquez sur Suivant.
  8. Enregistrez le projet Blink dans le répertoire par défaut.
  9. Cliquez sur Oui pour ajouter au projet Blink des fichiers Java existants.
  10. Cliquez sur Suivant.
  11. Cliquez sur le bouton Ajouter tout dans répertoire (sous la liste Fichiers projet).
    Le nom de fichier Blink.java apparaît dans la liste Fichiers projet.
  12. Cliquez sur Suivant.
    Dans la zone Classe appliquette, Java Development Environment entre Blink. Le fichier de la classe principale est le fichier .class qui étend la classe Applet de la bibliothèque Java.
  13. Cliquez sur Suivant.
  14. Utilisez la page HTML générée par Java Development Environment.
  15. Cliquez sur Terminer.
    La fenêtre de l'éditeur de source affiche le fichier source Blink.java, dont le nom apparaît dans la barre de titre de Java Development Environment. Dans le gestionnaire de projet, votre portfolio personnel indique Blink comme projet en cours. Le projet en cours définit le contexte de tous les outils de Java Development Environment.

    Dans cet exemple, vous avez créé le projet Blink dans le fichier Blink.prj enregistré dans votre répertoire Blink. A l'avenir, lorsque vous créerez un projet, il pourra être enregistré dans n'importe quel répertoire de votre système de fichiers.
