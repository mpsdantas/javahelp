Ajout d'un portfolio personnel

La première fois que vous exécutez le logiciel Java Workshop, il crée un portfolio nommé "personal" pour lequel vous détenez une autorisation en écriture. Si vous supprimez ce portfolio mais non notre votre fichier portfolio (.psf) de votre système de fichiers, vous pouvez le réimporter dans Java Development Environment.

Pour les besoins de ce didacticiel, vous pouvez cependant utiliser tout portfolio pour lequel vous avez une autorisation en écriture. Vous pouvez également créer un nouveau portfolio à l'aide de l'aide fournie dans Création d'un portfolio.

Les instructions suivantes décrivent comment ajouter un portfolio personnel :

  1. Dans la fenêtre Gestionnaire de projet de Java Development Environment, choisissez Fichier -> Ajouter -> Portfolio.
    La boîte de dialogue Ajout d'un portfolio apparaît.
  2. Entrez /votre_répertoire_d'installation/jws/personal.psf dans la zone Fichier portfolio.
    Tous les noms de fichier portfolio portent l'extension .psf.
  3. Cliquez sur Ajouter.
    Java Development Environment fait de votre portfolio personnel le portfolio en cours et affiche son nom en caractères bleus.
