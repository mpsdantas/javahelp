Menu Aide

Sommaire de l'aide Ouvre la table des matières de l'aide en ligne.
Aide basée sur les tâches Affiche le menu des rubriques d'aide des tâches communes.
Aide des menus Affiche la liste des rubriques d'aide sur les menus de Java Development Environment.
Didacticiel de Java Development Environment Lance le didacticiel de Java Workshop, série de sept leçons interactives qui vous accompagne étape par étape tout au long du processus de création, génération, exécution et débogage d'un projet appliquette.
Informations version Affiche les informations relatives à cette version de Java Development Environment.
Numéro de série/Licence Affiche le numéro de série de Java Development Environment ainsi que les informations de licence.
Inscription Ouvre la boîte de dialogue Inscription de Java WorkShop, qui permet d'inscrire électroniquement votre copie de Java Development Environment.
Envoyer commentaires Ouvre la boîte de dialogue Commentaires de Java Development Environment. Cette boîte vous permet d'envoyer électroniquement des commentaires et des suggestions à l'équipe produit Java Development Environment.
Documentation API Java (1.1) Affiche la documentation API du Java 1.1 dans l'afficheur d'aide. Vous pouvez également accéder à cette documentation à partir de la table des matières de l'aide en ligne.
Spécification du langage Java Affiche la table des matières du manuel « Java Language Specification » de James Gosling, Bill Joy et Guy Steele.
A propos de Java Development Environment Ouvre la boîte A propos, où vous trouverez les informations suivantes :
  • Numéro de série
  • Numéro de version
  • Photographie de l'équipe de développement de Java Development Environment
  • Notice des droits de propriété
