Exercice 1 : Démarrage d'une session de débogage

Cet exercice démontre comment lancer le débogage du projet Blink. Si vous n'avez pas encore créé et construit ce projet, faites cela maintenant en suivant les instructions de Création du projet Blink.
  1. Vérifiez dans le gestionnaire de projet le nom du projet en cours.
    Le fichier Blink.java doit s'afficher dans l'éditeur de source. Si le projet en cours n'est pas Blink, double-cliquez sur le nom de projet Blink.
  2. Cliquez sur le bouton Redémarrer de la barre d'outils Edition/Débogage.
    L'exécution de Blink.java commence et Java Development Environment exécute le projet Blink dans la fenêtre de l'afficheur d'appliquette.

Pendant que Blink.java s'exécute, vous pouvez fixer un point d'arrêt en choisissant Déboguer -> Points d'arrêt. Vous pouvez aussi suspendre le programme en choisissant Déboguer -> Interrompre le programme.
