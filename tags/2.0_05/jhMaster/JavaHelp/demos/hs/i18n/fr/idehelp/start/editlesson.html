Exercice 1 : Edition des attributs d'exécution

La balise <applet> que vous avez générée dans le Didacticiel Un définit le strict nécessaire pour positionner l'appliquette dans une page Web. Dans cet exercice, vous ajouterez des paramètres qui vous permettront de personnaliser la manière dont l'appliquette sera affichée.

Cet exercice est associé au projet Blink. Si vous n'avez pas encore créé et construit ce projet, faites cela maintenant en suivant les instructions de Création du projet Blink.

  1. Vérifiez dans le gestionnaire de projet le nom du projet en cours.
    Si celui-ci n'est pas Blink, double-cliquez sur le nom de projet Blink.
  2. Dans le gestionnaire de projet, choisissez Projet -> Editer.
    La fenêtre Edition de projet présente quatre onglets : Généralités, Générer, Déboguer/Parcourir et Exécuter.
  3. Cliquez sur l'onglet Exécuter.
    Les attributs proposés dans cet onglet servent à générer la balise <applet> dans le fichier Blink.tmp.html.
  4. Trouvez le champ Paramètres (environ à mi-fenêtre).
    Actuellement, aucun paramètre n'est défini pour l'appliquette. Un paramètre est un attribut non standard qui définit la manière dont l'appliquette sera affichée. Il comporte un nom et une valeur. Vous ajouterez un paramètre pour modifier les mots dans le texte clignotant.
  5. Entrez lbl ("label" (étiquette) en abrégé), dans la zone de texte Nom située à droite du champ Paramètres.
  6. Dans la zone Valeur, saisissez le texte suivant :
    Java Development Environment est vraiment super :-)
  7. Cliquez sur le bouton Ajouter.
    La ligne lbl=Java Development Environment est vraiment super :-) apparaît dans le champ Paramètres.
  8. Cliquez sur OK pour enregistrer vos modifications.
  9. Cliquez sur le bouton Exécuter de la barre d'outils principale de Java Development Environment.
    Java Development Environment présente maintenant cette ligne : 
    <param name="lbl" value="Java Development Environment est vraiment super :-)">
    dans la balise <applet> du fichier Blink.tmp.html. Le fichier est chargé dans l'explorateur, et vous voyez les mots "Java Development Environment est vraiment super :-)" clignotant en plusieurs couleurs.

    Pour arrêter l'appliquette en cours d'exécution, choisissez Appliquette -> Quitter dans la fenêtre de l'afficheur d'appliquette.
