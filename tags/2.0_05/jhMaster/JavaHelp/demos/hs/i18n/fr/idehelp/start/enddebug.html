Exercice 6 : Fin d'une session de débogage

Dans cet exercice, vous mettrez fin à la session de débogage.
  1. Choisissez Déboguer -> Arrêter le débogage.
    Java Development Environment termine la session, et l'afficheur d'appliquette se ferme.
