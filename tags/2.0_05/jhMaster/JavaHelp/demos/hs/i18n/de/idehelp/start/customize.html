Trainingshandbuch, Kapitel 6: Anpassen einer Run-Page

In diesem Kapitel des Trainingshandbuchs lernen Sie, wie die HTML-Datei bearbeitet wird, mit der ein Projekt ausgeführt werden kann. Damit Sie dieses Trainingshandbuch optimal nutzen können, sollten Sie die Übungen in der Reihenfolge durchführen, in der sie angezeigt werden.
  • In Übung 1: Anpassen einer HTML-Datei wird das Bearbeiten einer HTML-Datei beschrieben, die beim Ausführen eines Projekts durch Java WorkShop erstellt wird.
  • In Übung 2: Aktualisieren einer Run-Page wird das Ausführen der bearbeiteten HTML-Seite anstelle der Standarddatei Blink.tmp.html beschrieben.
