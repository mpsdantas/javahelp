Übung 2: Beheben von Fehlern während des Aufbauprozesses

Diese Übung zeigt, wie Sie beim Auftreten von Fehlern während des Aufbauprozesses die verantwortliche Quellzeile schnell finden und korrigieren können. Die häufigsten Fehler beim Aufbauen sind fehlerhafte Sprachsyntax, nicht deklarierte Variablen und falsch geschriebene Schlüsselwörter.

Falls noch nicht geschehen, sollten Sie jetzt den Quelltext aus dem Blink-Projekt hinzufügen wie im Abschnitt Bearbeiten von Quellcode beschrieben.

Auffinden von Fehlern

  1. Prüfen Sie, welches aktuelle Projekt im Projekt-Manager angezeigt wird.
    Handelt es sich bei dem aktuellen Projekt nicht um das Blink-Projekt, doppelklicken Sie auf den Projektnamen "Blink".
  2. Klicken Sie in der Hauptsymbolleiste von Java Development Environment auf die Schaltfläche "Erstellen" .
    Java Development Environment kompiliert das Blink-Projekt standardmäßig mit dem Java Development Environment Java-Compiler; dieser Compiler ist schneller als der standardmäßige JDK Java-Compiler.

    Die Registerkarte "Erstellen" führt zwei Fehler auf. Im Quelleditor ist die für den ersten Fehler verantwortliche Quellzeile gelb gekennzeichnet.

Beheben von Fehlern

  1. Um den ersten Fehler zu beheben, setzen Sie die fehlenden Klammern um offset.
    Die Zeile sollte jetzt folgendermaßen aussehen:
    if (offset)
  2. Wählen Sie jetzt in der Registerkarte "Erstellen" den zweiten Fehler Blink.java: 61
    Im Quelleditor ist die für den Fehler verantwortliche Quellzeile gekennzeichnet.
  3. Beheben Sie den Fehler durch Hinzufügen von ; nach y = y / 2 in Zeile 60.
    Die Zeile sollte jetzt folgendermaßen aussehen:
    y = y / 2;
  4. Klicken Sie in der Symbolleiste "Bearbeiten/Debug" auf die Schaltfläche "Datei speichern" .
    Damit sind die Fehler im Programm behoben, und Sie können es neu aufbauen.

Erneutes Aufbauen des Projekts

  1. Klicken Sie in der Hauptsymbolleiste auf die Schaltfläche "Erstellen" .
    Jetzt sollten beim Aufbau keine Fehler mehr auftreten. Andernfalls wechseln Sie wieder in den Quelleditor und prüfen die eingegebenen Änderungen.
  2. Klicken Sie in der Hauptsymbolleiste von Java Development Environment auf die Schaltfläche "Ausführen" .
    Java Development Environment führt das Projekt mit dem sich auf und ab bewegenden Blinktext aus.
