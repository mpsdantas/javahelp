Übung 3: Suchen von Zeichenfolgen im Code

In dieser Übung suchen Sie alle Vorkommen einer angegebenen Zeichenfolge im Quellcode. Anschließend lassen Sie die Übereinstimmungen zusammen mit dem umgebenden Quellcode anzeigen.
  1. Prüfen Sie, welches aktuelle Projekt im Projekt-Manager angezeigt wird.
    Handelt es sich bei dem aktuellen Projekt nicht um das Blink-Projekt, doppelklicken Sie auf den Namen des Blink-Projekts.
  2. Klicken Sie in der Hauptsymbolleiste von Java Development Environment auf die Schaltfläche "Zeichenfolge suchen".
  3. Geben Sie im Feld "Zeichenfolge" green ein.
  4. Klicken Sie auf "Suchen".
    Jedes Vorkommen der Zeichenkette im Quellcode wird als Übereinstimmung angezeigt. Die Funktion zum Suchen von Zeichenfolgen findet zwei Vorkommen der Zeichenfolge green. Die Übereinstimmungen werden in der Registerkarte "Zeichenfolge suchen" unterhalb des Quelleditorfensters angezeigt. Die Anzeige der Übereinstimmungen erfolgt in dieser Form: Dateiname: Zeilennummer. Darüber hinaus werden die Zeilen des Quellcodes angezeigt, in denen die Zeichenfolge enthalten ist.
  5. Klicken Sie für die erste Übereinstimmung auf die Verknüpfung Blink.java:32.
    Im Quelleditor wird die Quellzeile angezeigt, die den Code enthält. Die Zeile wird blau gekennzeichnet.
  6. Wählen Sie "Durchsuchen" -> "Nächste Entsprechung" aus.
    Der Code der nächsten Übereinstimmung wird im Quelleditor gekennzeichnet.
