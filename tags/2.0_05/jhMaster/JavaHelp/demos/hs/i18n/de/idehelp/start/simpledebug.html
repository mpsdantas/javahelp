Trainingshandbuch, Kapitel 5: Fehlersuche in einem Projekt

In diesem Trainingshandbuch wird Ihnen die Fehlersuche in Java WorkShop vorgestellt. Sie lernen, wie die Threads in einem Programm zu untersuchen sind, Haltepunkte festgelegt werden und wie die Daten eines Programms untersucht werden.

Das Trainingshandbuch besteht aus sieben Lektionen. Sie sollten die Aufgaben in der angegebenen Reihenfolge ausführen.

  • Wichtige Konzepte: Threads beschreibt kurz Threads und Threadgruppen. Sie sollten diese Konzepte verstehen, bevor Sie mit dem Trainingshandbuch fortfahren.
  • Übung 1: Starten einer Fehlersuche-Sitzung zeigt, wie Sie mit der Fehlersuche in einem Projekt beginnen.
  • Übung 2: Festlegen eines Haltepunkts erklärt, wie die Ausführung eines Programms an einem bestimmten Punkt des Codes angehalten werden kann.
  • Übung 3: Überprüfen von Threads zeigt, wie Sie Thread-Gruppen, Threads und Stapelrahmen, die ein Projekt ausmachen, anzeigen können.
  • Übung 4: Schrittweise eine Methode testen zeigt, wie Sie schrittweise einen Code testen können.
  • Übung 5: Auswertung von Ausdücken zeigt, wie Sie den Wert der Programmdaten prüfen können.
  • Übung 6: Beenden einer Fehlersuche-Sitzung zeigt, wie Sie eine Fehlersuche-Sitzung beenden können.
