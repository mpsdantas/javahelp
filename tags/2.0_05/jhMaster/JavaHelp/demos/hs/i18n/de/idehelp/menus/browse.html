Menü "Durchsuchen"

Zeichenfolge suchen
Zeigt das Dialogfeld Zeichenfolge suchen an. Sie können innerhalb eines Projekts bzw. eines Verzeichnisses nach einer bestimmten Zeichenfolge suchen. Für diesen Befehl steht auf der Hauptsymbolleiste auch eine Schaltfläche zur Verfügung.
Suche stoppen Beendet die aktuelle Suche.
Weitersuchen Zeigt die nächste Übereinstimmung einer Zeichenkettensuche an. Dabei handelt es sich bei der nächsten Übereinstimmung um die auf die aktuelle Übereinstimmung auf der Registerkarte "Zeichenfolge suchen" folgende Übereinstimmung. Falls erforderlich, wird eine neue Datei in den Editor geladen.
Rückwärts suchen Zeigt die vorherige Übereinstimmung einer Zeichenkettensuche an. Dabei handelt es sich bei der vorherigen Übereinstimmung um die der aktuellen Übereinstimmung auf der Registerkarte "Zeichenfolge suchen" vorangehende Übereinstimmung. Falls erforderlich, wird eine neue Datei in den Editor geladen.
Klassen anzeigen Öffnet die Klassenanzeige. Mit der Klassenanzeige können Sie die in Ihrem Projekt enthaltenen Klassen, Methoden und Kommentare anzeigen. Die Klassenanzeige basiert auf javadoc, dem Java API-Dokumentationsgenerator.
