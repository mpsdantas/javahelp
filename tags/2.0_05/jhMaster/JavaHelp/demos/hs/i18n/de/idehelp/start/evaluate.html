Übung 3: Auswerten von Ausdrücken

Neben dem Fenster "Threads/Stapel" können Sie auch mit dem Fenster "Ausdruckauswertung" Ausdrücke überprüfen. In dieser Übung werden Sie einen Ausdruck des Blink-Applets überprüfen und die Ausführung danach fortsetzen.
  1. Wählen Sie "Fehlersuche" -> "Auswerten".
    Das Fenster "Ausdruckauswertung" wird angezeigt.
  2. Geben Sie in das Feld "Ausdruck" speed ein.
  3. Klicken Sie auf "Auswerten".
    Der Ausdruck speed sowie dessen entsprechende Werte werden auf der Registerkarte "Fehlersuche" angezeigt.
  4. Klicken Sie auf die Schaltfläche "Wiederaufnehmen" auf der Symbolleiste "Bearbeiten/Debug".
    Das Blink-Projekt wird weiter im Applet-Viewer ausgeführt.
