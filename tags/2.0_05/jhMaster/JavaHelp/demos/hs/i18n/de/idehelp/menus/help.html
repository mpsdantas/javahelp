Menü "Hilfe"

Hilfethemen Öffnet das Inhaltsverzeichnis der Online-Hilfe.
Aufgabenbasierte Hilfe Zeigt ein Menü mit Hilfethemen zu häufig benötigten Aufgaben an.
Hilfe zum Menü Zeigt eine Liste der Hilfethemen für Java-WorkShop-Menüs an.
Java Development Environment-Trainingshandbuch Ruft das Java Development Environment-Trainingshandbuch auf. Hier finden Sie acht interaktive Lernprogramme, in denen Schritt für Schritt gezeigt wird, wie Sie ein Applet-Projekt anlegen, aufbauen, ausführen und Fehler darin beheben.
Versionshinweise Enthält aktuelle Informationen über die vorliegende Version von Java Development Environment.
Seriennummer/Lizenz Zeigt die Seriennummer und Lizenzdaten Ihrer Java WorkShop-Installation an.
Registrierung Öffnet das Dialogfeld "Java WorkShop-Registrierung". Über dieses Dialogfeld können Sie Ihre Kopie von Java WorkShop direkt online registrieren lassen.
Kommentare senden Öffnet das Java Development Environment-Kommentarfeld. Darin können Sie Ihre Kommentare und Vorschläge auf elektronischem Weg an das Java Development Environment-Entwicklungsteam weiterleiten.
Java API-Dokumentation (1.1) Öffnet die Dokumentation der Java API 1.1 in der Hilfe-Anzeige. Diese Dokumentation ist auch über das Inhaltsverzeichnis der Online-Hilfe erreichbar.
Java-Sprachspezifikation Zeigt das Inhaltsverzeichnis der "Java-Sprachspezifikation" von James Gosling, Bill Joy und Guy Steele an.
Info über Java Development Environment Öffnet das Feld "Info". Darin werden die folgenden Informationen angezeigt:
  • Seriennummer
  • Versionsnummer
  • Foto des Java Development Environment-Entwicklungsteams
  • Copyright-Hinweise
