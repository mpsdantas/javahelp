Menü "Fehlersuche"

Starten / Neu starten Startet die Fehlersuche beim aktuellen Projekt bzw. startet sie erneut.
Fortsetzen Setzt das Ausführen des aktuellen Projekts fort.
Ausführen bis hier Führt das aktuelle Projekt bis zur angegebenen Position aus.
Springen über Springt über eine Methode.
Springen in Springt in eine Methode.
Auswerten
 Wertet einen Datenwert aus.
Aufrufende Methode anzeigen Springt in die nächst niedrigere Aufrufebene.
Aufgerufene Methode anzeigen Springt in die nächst höhere Aufrufebene.
Fäden/Stapel Öffnet das Fenster "Fäden/Stapel".
Breakpoint-Schalter
 Setzt bzw. entfernt einen Breakpoint in der Zeile der aktuellen Cursorposition.
Breakpoint Öffnet das Breakpoint-Fenster.
Ausnahmen Zeigt eine Liste vorhandener Bedingungen an, die mit dem Debugger verarbeitet werden.
Fehlersuche beenden Beendet die aktuelle Fehlersuche-Sitzung.
