Trainingshandbuch, Kapitel 3: Beheben von Fehlern im Quellcode

In diesem Kapitel lernen Sie, wie Sie den Quellcode des Blink-Projekts bearbeiten und das Projekt erneut aufbauen. Damit Sie dieses Trainingshandbuch optimal nutzen können, sollten Sie die Übungen in der Reihenfolge durchführen, in der sie angezeigt werden.
  • In Übung 1: Bearbeiten von Quellcode erfahren Sie, wie Sie die grundsätzlichen Funktionen des Java Development Environment-Quelleditors verwenden.
  • In Übung 2: Beheben von Fehlern in einem aufgebauten Programm werden die im Verlauf des Aufbauprozesses festgestellten Fehler behoben.
