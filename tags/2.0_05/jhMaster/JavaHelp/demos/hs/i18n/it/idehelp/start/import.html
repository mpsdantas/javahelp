Aggiunta del portafoglio personale

Alla prima esecuzione di Java Workshop viene creato un portafoglio denominato "personale" per il quale l'utente dispone dell'autorizzazione di scrittura. Se si rimuove il portafoglio personale senza eliminare il file di portafoglio (.psf) dal proprio file system, sarà possibile importarlo nuovamente all'interno di Java Development Environment.

Durante l'esercitazione sarà tuttavia possibile utilizzare qualsiasi portafoglio per il quale si dispone dell'autorizzazione di scrittura. In alternativa, è possibile creare un nuovo portafoglio seguendo le istruzioni fornite nella sezione Creazione di un portafoglio.

Per aggiungere il portafoglio personale, attenersi alla seguente procedura.

  1. Nella finestra del gestore progetti di Java Development Environment scegliere File -> Aggiungi -> Portafoglio.
    Verrà visualizzata la finestra di dialogo Aggiungi portafoglio.
  2. Digitare /directory_di_installazione/jws/personal.psf nella casella File di portafoglio.
    I nomi dei file di portafoglio hanno tutti estensione .psf.
  3. Scegliere il pulsante Aggiungi.
    Il portafoglio personale verrà impostato come portafoglio corrente e il relativo nome verrà visualizzato in blu.
