Cerca stringhe
Visualizza la finestra di dialogo Cerca stringhe in cui è possibile ricercare il testo specificato all'interno di un progetto o di una directory. Per questo comando è disponibile anche un pulsante di scelta rapida sulla barra degli strumenti principale.
Ferma ricerca Interrompe la ricerca corrente.
Trova successivo Visualizza la voce successiva trovata come risultato di una ricerca. La voce successiva è la voce elencata dopo quella corrente nella scheda Cerca stringhe. Se necessario, nella finestra dell'editor viene caricato un nuovo file.
Trova precedente Visualizza la voce precedente trovata come risultato di una ricerca. La voce precedente è la corrispondenza elencata prima di quella corrente nella scheda Cerca stringhe. Se necessario, nella finestra dell'editor viene caricato un nuovo file.
Visualizza classi Apre il visualizzatore delle classi che può essere utilizzato per sfogliare le classi, i metodi e i commenti presenti nei progetti. Il visualizzatore delle classi è basato su javadoc, il generatore di documentazione API di Java.
