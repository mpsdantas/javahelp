Esercizio 1: Modifica degli attributi di esecuzione

Il tag dell'<applet> generato nell'Esercitazione 1 consente di specificare i requisiti minimi per inserire l'applet in una pagina web. In questo esercizio verranno aggiunti i parametri che consentono di personalizzare il modo in cui l'applet viene visualizzato.

In questo esercizio verrà utilizzato il progetto Blink. Se il progetto Blink non è stato ancora creato, crearlo seguendo le istruzioni riportate in Creazione del progetto Blink.

  1. Individuare il nome del progetto corrente nella finestra del gestore progetti di Java Development Environment.
    Se il progetto corrente non è Blink, fare doppio clic sul nome del progetto Blink.
  2. Nel gestore progetti scegliere Progetto -> Modifica.
    Nella finestra Modifica progetto sono visualizzate: Generale, Compila, Debug/cerca ed Esegui.
  3. Fare clic sulla scheda Esegui.
    Gli attributi riportati in questa scheda vengono utilizzati per generare il tag <applet> nel file Blink.tmp.html.
  4. Individuare il campo Parametri (posto nella parte inferiore della finestra).
    Attualmente, non è stato definito alcun parametro per l'applet. Un parametro è un attributo non standard che definisce il modo in cui l'applet viene visualizzato. Un parametro presenta un nome e un valore. Nell'esercizio verrà aggiunto un parametro per modificare le parole nel testo lampeggiante.
  5. Digitare lbl (abbreviazione di "label"), nella casella di testo Nome a destra del campo Parametri.
  6. Digitare il seguente testo nella casella Valore:
    Java Development Environment è davvero incredibile :-)
  7. Fare clic sul pulsante Aggiungi.
    La riga lbl=Java Development Environment è davvero incredibile :-) viene visualizzata nel campo Parametri.
  8. Fare clic su OK per salvare le modifiche apportate.
  9. Fare clic sul pulsante Esegui sulla barra degli strumenti principale di Java Development Environment.
    La riga 
    <param name="lbl" value="Java Development Environment è davvero incredibile :-)">
    viene inclusa nel tag dell'<applet> nel file Blink.tmp.html. Il file viene caricato nel browser e le parole "Java Development Environment è davvero incredibile :-)" lampeggeranno in diversi colori.

    Per interrompere l'esecuzione del progetto, scegliere Applet -> Esci nella finestra del visualizzatore applet.

Lezione successiva
