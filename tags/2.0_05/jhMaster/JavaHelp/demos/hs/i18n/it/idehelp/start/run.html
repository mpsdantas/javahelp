Esercizio 3: Esecuzione di un progetto

In questo esercizio verrà eseguito il progetto Blink utilizzando il file HTML creato da Java Development Environment.
  1. Individuare nel gestore progetti il nome del progetto corrente.
    Se il progetto corrente non è Blink, fare doppio clic sul nome di progetto Blink.
  2. Fare clic sul pulsante Esegui sulla barra degli strumenti principale di Java Development Environment.
    Verrà generato il file HTML nella directory Blink. Il file HTML contiene il tag <applet>, che determina le opzioni da utilizzare per inserire il progetto Blink in una pagina web. Il file verrà caricato nel visualizzatore applet e il progetto Blink verrà eseguito. La parola "Blink" lampeggerà e verrà visualizzata con colori diversi.
  3. Per interrompere l'esecuzione dell'applet, scegliere Applet -> Esci nella finestra del visualizzatore applet.
