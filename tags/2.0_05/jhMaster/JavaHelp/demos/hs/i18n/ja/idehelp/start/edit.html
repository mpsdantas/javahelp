チュートリアル 2: プロジェクト属性の編集

このチュートリアルでは、プロジェクト属性とは何か、およびプロジェクト属性を編集する方法について説明します。チュートリアルの内容を効果的に習得することができるように、以下に示す順序で学習を進めてください。
  • 重要な概念: プロジェクト属性 プロジェクト属性とは何か、およびプロジェクト属性が Java WorkShop でどのように使用されるかについて説明します。これらの概念を理解してから実習に進んでください。
  • 実習 1: 実行属性を編集する アプレットをどのように実行するかを指定する方法について説明します。
  • 実習 2: 構築属性を編集する 構築時のオプションなどを指定する方法について説明します。
