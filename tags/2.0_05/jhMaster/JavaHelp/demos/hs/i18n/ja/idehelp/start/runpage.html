実習 2: 実行ページを更新する

この実習では、デフォルトの HTML ファイル Blink.tmp.html の代わりに、新しく作成した HTML ファイルを実行します。
  1. 「プロジェクトマネージャ」ウィンドウで、現在のプロジェクトの名前を確認します。
    現在のプロジェクトが Blink でない場合は、プロジェクト名「Blink」をダブルクリックします。
  2. 「プロジェクト」->「編集」を選択します。
    「プロジェクト編集」ウィンドウが表示されます。
  3. 「実行」タブをクリックします。
  4. 「既存の実行ページ (HTML ファイル) を使用」を「はい」に切り替えます。
    「既存の実行ページ (HTML ファイル) を使用」以外のすべてのテキストフィールドがグレー表示されます。「既存の実行ページ (HTML ファイル) を使用」で「はい」が選択されている時は、プロジェクトの実行時に「実行」タブ付きページ中での設定内容は無視されます。
  5. 編集した新しい HTML ファイル名をテキストフィールドに入力します。
    パス名全体ではなくファイル名だけを入力します。ファイルはソースディレクトリから取得されます。
  6. 「了解」をクリックします。
  7. メインツールバー上の「実行」ボタン をクリックします。
    新しい HTML ファイルが読み込まれます。実習 1 で HTML ファイルを編集する前と比較すると、アプレットビューアの高さが小さくなり、Blink プロジェクトの実行速度が速くなったことが確認できます。
