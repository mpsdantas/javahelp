実習 1: デバッグセッションを開始する

この実習では、Blink プロジェクトのデバッグを開始する方法を説明します。まだ Blink プロジェクトを作成していない場合は、ここで Blink プロジェクトを作成するの手順に沿って Blink プロジェクトを作成してから、以下の実習に進んでください。
  1. 「プロジェクトマネージャ」ウィンドウで、現在のプロジェクトの名前を確認します。
    Blink.java ファイルがソースエディタに表示されている必要があります。現在のプロジェクトが Blink でない場合は、プロジェクト名「Blink」をダブルクリックします。
  2. 編集デバッグツールバー上の「開始」ボタン をクリックします。
    Blink.java が実行され、アプレットビューアで Blink プロジェクトが実行されます。

Blink.java の実行中に、「デバッグ」->「ブレークポイント」を選択して、ブレークポイントを設定することができます。また、「デバッグ」->「プログラム割り込み」を選択して、プログラムの実行を中断することもできます。
