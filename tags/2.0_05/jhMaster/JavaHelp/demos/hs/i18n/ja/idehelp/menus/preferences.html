「設定の変更」メニュー

プロキシ 「プロキシ」ダイアログを使って、プロキシサーバー (プロキシ) を変更します。
バージョン管理 Java WorkShop で使用するバージョン管理システムを選択します。
ソースエディタ ソースエディタに関する、以下の内容を設定します。
  • インデントとタブ
  • 構文の強調表示と色
  • フォント選択
キーボード 「キーボード」ダイアログで、Java WorkShop 中で有効なキー割り当てを表示、追加、削除できます。詳細については、キーボード設定を参照してください。
コンソール入力 コマンド行からのプログラム入力をコンソールウィンドウで解釈する方法を指定します。詳細については、「コンソール入力」ダイアログを参照してください。
構築 プロジェクトのコンパイルに使用する Java コンパイラを指定します。詳細については、「構築」ダイアログを参照してください。
ツールバーでラベルを表示 ツールバーの各ボタンの下にボタン名を示すラベルを表示します。
