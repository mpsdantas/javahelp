Java WorkShop のオンラインヘルプ

Java WorkShop のヘルプファイルは、以下の 6 つのトピックで構成されています。これらのトピックには、メインウィンドウの「ヘルプ」メニューから直接アクセスすることができます。
  • プロジェクトを管理する -- プロジェクトやポートフォリオの作成と利用の方法について
  • ソースコードを編集する -- ソースファイルの編集方法について
  • プロジェクトを構築する -- Java プログラムの構築方法について
  • Java WorkShop GUI ビルダー -- GUI の配置や生成の方法について
  • ソースコードを表示する -- ソースファイルの表示方法について
  • プロジェクトをデバッグする -- Java プログラムのデバッグ方法について

「目次」ボタン をクリックすると、目次ページに階層化されたヘルプトピックが表示されます。閉じた本のアイコンをクリックすると、そのトピック下の階層が表示され、開いた本のアイコンを再度クリックすると元に戻ります。また、目次はナビゲーションの助けとして利用することもできます。あるページを表示中に「目次」ボタンをクリックすると、そのトピックが目次ページで強調表示され、そのページの場所を知ることができます。

Java WorkShop のオンラインヘルプの詳細については、ヘルプビューアについて をご覧ください。
