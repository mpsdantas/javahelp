浏览菜单

字符串查找
激活字符串查找 对话框。您可以在项目或目录中 查找指定的文本。该命令在主工具栏上 还有一个快捷按钮。
停止查找 停止当前的查找操作。
下一次匹配 显示下一次字符串查找所找到的匹配结果。 下一次匹配就是在“字符串查找”制表项中 列在当前匹配之后的匹配项。需要的时候， 会在编辑器中装入一个新文件。
上一次匹配 显示上一次字符串查找所找到的匹配结果。 上一次匹配就是在“字符串查找”制表项中 列在当前匹配之前的匹配项。需要的时候， 会在编辑器中装入一个新文件。
查看类 打开类查看器 。您可以使用类查看器来浏览项目中的 类、方法和注释。类查看器 主要基于 javadoc，即Java API 文档生成程序。
