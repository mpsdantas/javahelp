练习2：生成项目

本练习将讲述如何生成Blink项目。在您生成一个项目时，Java Development Environment会将该项目的Java源代码编译到 一个名为filename.class的文件中。

如果您还没有创建Blink项目，就应先使用创建Blink项目中的步骤完成这项工作。

  1. 在Java Development Environment项目管理器窗口中检查当前项目的名字。
    如果当前项目不是Blink，请双击Blink项目名。
  2. 单击Java Development Environment主工具栏中的“生成”按钮。
    Java Development Environment将Blink.java程序编译成字节码文件Blink.class。请在Java Development Environment生成项目时， 留意“生成”制表项（在源代码编辑器下边）。当源代码完成生成和编译工作时，会在 “生成”制表项中显示状态消息“完成”。
