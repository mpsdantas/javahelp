Java Development Environment术语

在线帮助的通篇都会用到下列术语。理解这些术语可以帮助您更快地学完教程。其它的Java Development Environment专用术语（比如 applet小程序 和 线程）会当其首次出现在教程中时加以解释。

在您按教程进行练习的过程中，请时刻留意浏览器右上角的动画图标。当中的立方体会在Java Development Environment装入帮助页面时旋转。

按钮
是一个位于工具栏或窗口的单元，在其上单击即可引发一个动作。例如，如果您单击“项目”按钮 ，就会显示Java Development Environment项目管理器窗口。

工具栏中的按钮都有工具提示。如果您对一个工具栏按钮所执行的动作不太了解，您可以把指针 指向该按钮来显示描述信息。例如，如果您把指针指向按钮，就会显示“字符串查找”。

菜单
是一个项目列表，您可以从中选择一个特定的任务或对象。浏览器中的菜单出现在标题栏和 Java Development Environment主工具栏之间。菜单中的项目通常以格式菜单->菜单项来表示。例如，项目->生成指的是 “在项目菜单中选择生成菜单项。”
制表项
是一个制表页面（类似于对话框），单击窗口中的某个制表项时即可进行显示。例如， “编辑项目”窗口就包含了下列制表项：通用、生成、调试/浏览和运行。
