窗口菜单

项目管理器
打开 项目管理器窗口。利用“项目管理器”窗口， 您可以选择项目、文件以及能更改项目 的操作。
简要表结果 启动Java Development Environment性能简要表。有关详情 请参阅 Java Development Environment简要表。
Web浏览器 启动Java Development Environment web浏览器。有关详情 请参阅 web浏览器。
启动 打开“启动帮助”，其中为新的 Java Development Environment用户提供了几种用来 启动该产品的方式选项。
Java控制台 打开“控制台”窗口。有关详情请参阅 控制台窗口。
