练习2：查看项目中的方法

您将在本练习中列出Blink类所定义的方法。您可以利用这张表来查看方法的源代码。 要想成功完成本练习，您必须先完成前面的练习查看类层次。
  1. 在类查看器中，向下滚动到“构造函数索引表”。
    构造函数是一个特殊的方法，用于在创建对象时对它进行初始化。类查看器列出了一个在Blink类中 定义的构造函数Blink。
  2. 单击Blink。
    源代码编辑器将显示构造函数Blink的源代码。构造函数的第一行用蓝色高亮显示。
  3. 在类查看器中，向下滚动到“方法索引表”。
    Blink类中包含了五个方法： init、 paint、 start、 stop、和run。
  4. 单击paint。
    源代码编辑器将显示paint方法的源代码。第一行用蓝色高亮显示。
