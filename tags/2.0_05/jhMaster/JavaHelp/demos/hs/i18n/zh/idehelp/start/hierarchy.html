练习1：查看类层次

您将在本练习中学习Blink项目中的类是如何派生的。
  1. 在项目管理器中检查当前项目的名字。
    如果当前项目不是Blink，就双击Blink项目名。
  2. 选择浏览->查看类。
    Java Development Environment将打开类查看器，其中列出了组成Blink项目的类、方法和注释。 该项目有一个源文件， Blink.java。
  3. 单击Blink.java。
    类查看器将装入派生Blink的类。
    Blink从Applet派生出来， 而后者又从Panel派生出来，依此类推。

    类查看器信息是从javadoc，即Java API文档生成程序派生出来的。 Javadoc会对项目源代码中的声明和注释进行语法分析，并将API格式化成HTML代码。
