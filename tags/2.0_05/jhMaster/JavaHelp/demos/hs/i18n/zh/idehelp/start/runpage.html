练习2：更新运行页面

您将在本练习中运行新建的HTML文件，以取代缺省的Blink.tmp.html文件。
  1. 在项目管理器中检查当前项目的名字。
    如果当前项目不是Blink，就双击Blink项目名。
  2. 选择项目->编辑。
    将显示“编辑项目”窗口。
  3. 单击“运行”制表项。
  4. 对“现有运行页面的URL”单击“是”。
    窗口中除“现有运行页面的URL”以外的所有域都变成暗淡的。当把“现有运行页面的URL” 设置成“是”时，Java Development Environment会在运行项目时忽略“运行”制表项中的信息。
  5. 在“现有运行页面的URL”框中，输入刚才给更新过的HTML文件赋予的新名字。
    您不必输入完整的路径名。Java Development Environment会转去源代码目录读取这个文件。
  6. 单击“确定”。
  7. 单击主工具栏中的“运行”按钮。
    Java Development Environment会将新的HTML文件装入浏览器。可以看出，Blink项目在以更快的速度执行。
