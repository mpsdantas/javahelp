练习3：在代码中查找字符串

您将在本练习中学习如何在源代码中查找所有出现指定字符串的地方。 然后您可以查看这些匹配字符串及其周围的源代码。
  1. 在项目管理器中检查当前项目的名字。
    如果当前项目不是Blink，就双击Blink项目名。
  2. 单击Java Development Environment主工具栏中的“查找”按钮。
  3. 在“字符串”框中输入green。
  4. 单击“查找”。
    匹配项指的是源代码中任何与该字符串相匹配的字符串。字符串搜索器会找到两个与字符串green 相匹配的内容，并在源代码编辑器窗口底部的“字符串查找”制表项中显示这些匹配项。匹配项的 显示格式是： filename: line_number。您还会看到包含该字符串的源代码行。
  5. 单击指向第一个匹配的链接Blink.java:32。
    源代码编辑器将显示包含该字符串的源代码行。该行用蓝色高亮显示。
  6. 选择浏览->下一次匹配。
    源代码编辑器将高亮显示下一个匹配项的代码。
