「ウィンドウ」メニュー

プロジェクトマネージャ
プロジェクトマネージャ ウィンドウを開きます。プロジェクトマネージャウィンドウでは、プロジェクトやファイルを選択して内容を変更したり、新たにプロジェクトを作成することができます。
プロファイル結果 Java WorkShop パフォーマンスプロファイラを起動します。詳細については、Java WorkShop プロファイラを参照してください。
Web ブラウザ Java WorkShop の Web ブラウザを起動します。詳細については、Web ブラウザの概要を参照してください。
かんたんスタート 「Java WorkShop かんたんスタート」ウィンドウが表示されます。このウィンドウでは、初めて Java WorkShop を使用される方も迷わずに機能を使えるように、機能別のショートカットキーが用意されています。
Java コンソール コンソールウィンドウを開きます。詳細については、Java コンソールを参照してください。
