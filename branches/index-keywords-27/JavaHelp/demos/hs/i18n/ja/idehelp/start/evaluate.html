実習 5: 式を評価する

「スレッド/スタック」ウィンドウの他に、「式の評価」ウィンドウを使用して式を調べることもできます。この実習では、Blink アプレットの式を評価し、実行を再開します。
  1. 「デバッグ」->「評価」を選択します。
    「式の評価」ウィンドウが表示されます。
  2. 「式」テキストフィールドに、speed と入力します。
  3. 「評価」をクリックします。
    式 speed とその値が、「デバッグ」タブ付きページに表示されます。
  4. 編集デバッグツールバー上の「継続」ボタン をクリックします。
    アプレットビューアで Blink プロジェクトの実行が再開されます。
