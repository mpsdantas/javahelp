「プロジェクト」メニュー

プロジェクトマネージャ表示 Java WorkShop プロジェクトマネージャを起動します。
編集 「プロジェクト編集」ウィンドウを開きます。このウィンドウのタブ付きページを使用して、プロジェクトの属性を編集することができます。詳細については、プロジェクトの属性を編集するを参照してください。
構築 現在のプロジェクトを構築します。
実行 現在のプロジェクトを実行します。
プロファイル プロファイラを使用して現在のプロジェクトを実行します。
プログラム停止 現在のプロジェクトの実行を停止します。
ブラウズ ソースビューアの制御コマンドです。詳細については、ソースコードを表示するを参照してください。
アプレットタグをコピー 現在のアプレットプロジェクトから <applet> タグをコピーします。
ファイルをコンパイル .prj ファイルをコンパイルします。
新規 「プロジェクトの作成」ウィザードを開きます。
追加 「プロジェクトの追加」ウィザードを開きます。
