Trainingshandbuch, Kapitel 4:Durchsuchen eines Projekts

In diesem Kapitel lernen Sie das Durchsuchen eines Projekts. Zur optimalen Nutzung des Trainingshandbuchs ist zu empfehlen, die Übungen in der hier vorgegebenen Reihenfolge auszuführen.
  • Übung 1: Anzeige der Klassenhierarchie erlätert, wie Sie die Klassenhierarchie in dem Projekt durchlaufen können.
  • Übung 2: Anzeigen der Methoden in Ihrem Projekt zeigt Ihnen, wie Sie die im Projekt definierten Methoden auflisten können. In dieser Liste können Sie den Quellcode der einzelnen Methoden anzeigen.
  • Übung 3: Ermitteln von Strings in Ihrem Code zeigt Ihnen, wie Sie im Quellcode auftretende festgelegte Zeichenfolgenkonstanten ermitteln können.
