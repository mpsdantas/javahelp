Übung 2: Aufbau von Projekten

In dieser Übung wird der Aufbau des Blinkprojekts beschrieben. Sie müssen ein Projekt aufbauen, um es ausführen zu können. Wenn Sie ein Projekt aufbauen, wird der Java-Quellcode des Projekts in Java Development Environment in eine Datei mit dem Namen filename.class kompiliert.

Wenn Sie das Blinkprojekt noch nicht erstellt haben, so holen Sie dies jetzt nach. Verfahren Sie dabei nach den Anweisungen in Erstellen des Blinkprojekts.

  1. Prüfen Sie den Namen des aktuellen Projekts im Fenster Java Development Environment Projekt-Managers.
    Wenn das aktuelle Projekt nicht "Blink" ist, doppelklicken Sie auf den Projektnamen "Blink".
  2. Klicken Sie auf die Schaltfläche "Erstellen" auf der Hauptsymbolleiste von Java Development Environment.
    Java Development Environment kompiliert das Programm Blink.java in die Bytecode-Datei Blink.class. Beachten Sie die Registerkarte "Aufbau" (unter dem Quelleditor), während Java Development Environment das Projekt aufbaut. Die Statusmeldung "Abgeschlossen" wird angezeigt, wenn der Aufbau und die Kompilierung der Quellen beendet sind.
