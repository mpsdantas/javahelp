Übung 2: Aktualisieren der Run Page

In dieser Übung werden Sie eine neu erstellte HTML-Datei ausführen und nicht die Standarddatei Blink.tmp.html.
  1. Überprüfen Sie den Projekt-Manager für den Namen des aktuellen Projekts.
    Wenn das aktuelle Projekt nicht "Blink" ist, doppelklicken Sie auf den Projektnamen "Blink".
  2. Wählen Sie "Projekt" -> "Bearbeiten".
    Das Fenster "Projekt bearbeiten" wird angezeigt.
  3. Klicken Sie auf die Registerkarte "Ausführen".
  4. Klicken Sie bei "Bestehende Run Page URL" auf "Ja".
    Alle Felder im Fenster werden deaktiviert, mit Ausnahme von "Bestehende Run Page URL". Sobald "Bestehende Run Page URL" mit "Ja" angegeben ist, ignoriert Java Development Environment die Informationen in der Registerkarte "Ausführen", wenn das Projekt ausgeführt wird.
  5. Geben Sie im Feld für "Bestehende Run Page URL" den neuen Namen ein, den Sie gerade der aktualisierten HTML-Datei gegeben haben.
    Sie brauchen nicht den ganzen Pfadnamen einzugeben. Java Development Environment wechselt in den Quelleditor, um die Datei zu finden.
  6. Klicken Sie auf "OK".
  7. Klicken Sie auf die Schaltfläche "Ausführen" auf der Hauptsymbolleiste.
    Java Development Environment lädt die HTML-Datei in den Browser. Sie werden sehen, daß das Blink-Projekt mit schnellerer Geschwindigkeit ausgeführt wird.
