Menü "Fenster"

Projekt-Manager
Öffnet das Fenster "Projekt-Manager". In diesem Fenster können Projekte, Dateien und Operationen, mit denen Projekte geändert werden, ausgewählt werden.
Profil Startet den Java Development Environment Profiler. Weitere Informationen finden Sie unter Java Development Environment Profiler.
Web-Browser Startet den Java Development Environment Browser. Weitere Informationen finden Sie unter Web-Browser.
Start Öffnet den Start-Assistenten, in dem neuen Benutzern von Java WorkShop eine Auswahl von Startmöglichkeiten zur Verfügung steht.
Java-Konsole Öffnet das Konsolenfenster. Weitere Informationen finden Sie unter Fenster "Java-Konsole".
