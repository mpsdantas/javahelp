Übung 4: Sammeln von Profildaten

In dieser Übung lernen Sie, wie ein Blink-Projekt mit dem Java WorkShop-Profiler ausgeführt wird. Der Profiler hilft Ihnen, die Leistung Ihres Programms zu analysieren, zu bestimmen, wieviel Zeit in jeder Programmethode verbracht wurde und wie oft jede Methode ausgeführt wurde.
  1. Überprüfen Sie den Projekt-Manager für den Namen des aktuellen Projekts.
    Wenn das aktuelle Projekt nicht Blink ist, doppelklicken Sie auf den Projektnamen Blink.
  2. Starten Sie die Profilerstellung des Blink-Projekts, indem Sie auf die Schaltfläche "Profil" auf der Hauptsymbolleiste von Java Development Environment klicken.
    Das Fenster "Applet Viewer" wird angezeigt.
  3. Warten Sie mindestens 45 Sekunden, bis Sie den nächsten Schritt ausführen.

    4. So können genügend Daten bereitgestellt werden, um eine verwendbare Profilerstellungsanzeige zu produzieren.
  4. Beenden Sie die Profilerstellungssitzung, indem Sie im Applet Viewer wählen "Applet" -> "Beenden", .
    Nach der Profilerstellung des Programms werden im Fenster "Profil" die in die Datei Blink.prof des Projektquellverzeichnisses geschriebenen Daten angezeigt.
  5. Wählen Sie im Fenster "Profiler" "Filter" -> "Nicht filtern", um die Daten der Systemklasse anzuzeigen.
  6. Wählen Sie "Anzeigen" -> "Nach Namen sortieren", um die Methodendaten in alphabetischer Reihenfolge anzuzeigen.
  7. Wählen Sie die Methode Blink.paint (Ljava/awt/Graphics;).
  8. Sehen Sie sich nochmals die Profildaten mit ähnlichen Ergebnissen, wie in der unten gezeigten Abbildung an:

Die Ergebnisse zeigen folgendes an:

  • Java Development Environment verbrachte 93 Millisekunden damit, die Methode Blink.paint (Ljava/awt/Graphics;) aufzurufen.
  • Blink.paint (Ljava/awt/Graphics;) ruft drei Methoden auf: java.awt.Color.<init> (III), java.awt.Component.size () und java.awt.Font.getSize ().
  • Die zwei Methoden, die die Blink.paint (Ljava/awt/Graphics;) aufrufen sind: java.awt.Component.dispatchEventImpl (Ljava/awt/AWTEvent;) und java.awt.Component.update (Ljava/awt/Graphics;).
