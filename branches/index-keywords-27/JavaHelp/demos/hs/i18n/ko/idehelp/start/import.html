personal 포트폴리오 추가하기

Java Development Environment을 처음 실행하면, "personal"이라고 하는 포트폴리오가 자동으로 만들어집니다. 사용자는 이 포트폴리오에 대해 쓰기 권한을 가지고 있습니다. personal 포트폴리오를 삭제했다 해도 파일 시스템에서 해당 포트폴리오 (.psf) 파일을 지우지 않았다면, Java Development Environment으로 다시 불러올 수 있습니다.

파일 시스템에서도 지웠다면, 본 자습서에서 사용자에게 쓰기 권한이 있는 임의의 포트폴리오를 사용할 수 있습니다. 또는 포트폴리오 만들기의 도움말을 참조하여 새 포트폴리오를 만들 수도 있습니다.

다음 지침은 personal 포트폴리오를 추가하는 방법에 대한 해설입니다:

  1. [Java Development Environment 프로젝트 관리자] 창에서, [파일] -> [추가] -> [포트폴리오]를 선택하십시오.
    [포트폴리오 추가] 대화 상자가 화면에 나타납니다.
  2. [포트폴리오 파일] 란에 /your_installation_directory/jws/personal.psf라고 입력하십시오.
    모든 포트폴리오 파일 이름은 .psf라는 확장자로 끝납니다.
  3. [추가]를 마우스로 누르십시오.
    personal 포트폴리오가 현재 포트폴리오가 되고 그 이름이 파랑색 텍스트로 화면에 표시됩니다.
