[창] 메뉴

[프로젝트 관리자]
이 메뉴 명령을 선택하면 [프로젝트 관리자] 창이 열립니다. [프로젝트 관리자] 창을 사용하면, 프로젝트나 파일을 선택하고 프로젝트를 변경하는 동작을 할 수 있습니다.
[성능 분석 결과] 이 메뉴 명령을 선택하면 Java Development Environment 성능 분석 프로그램이 실행됩니다. 더 자세한 내용은 Java Development Environment 성능분석기를 참조하십시오.
[웹 브라우저] 이 메뉴 명령을 선택하면 [Java Development Environment 웹 브라우저]가 열립니다. 더 자세한 내용은 웹 브라우저를 참조하십시오.
[시작] 이 메뉴 명령을 선택하면 Java Development Environment을 처음 사용하는 사람이 이 제품을 실행하는 여러 가지 방법을 선택할 수 있도록 [시작 어시스턴트]가 열립니다.
[Java 콘솔] 이 메뉴 명령을 선택하면 [콘솔] 창이 열립니다. 더 자세한 내용은 [콘솔] 창을 참조하십시오.
