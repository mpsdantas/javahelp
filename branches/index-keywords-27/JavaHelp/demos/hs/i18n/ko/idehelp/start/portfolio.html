과정 1: 공용 포트폴리오 만들기

이 과정에서는 공용 포트폴리오(인터넷에 연결된 다른 사용자가 프로젝트를 액세스할 수 있는 포트폴리오)를 만드는 방법을 보여줍니다.
  1. [포트폴리오] => [만들기]를 선택합니다.
    이 동작은 [포트폴리오 만들기] 대화 상자를 엽니다.

  2. 텍스트 필드에서 HTTP 서버에서 액세스 가능한 포트폴리오 경로 이름을 입력합니다.
    포트폴리오 이름은 .psf 확장명으로 끝나야 합니다. 예를 들어, Solaris 환경에서는graphics.psf 포트폴리오를 /smith/public_html/folios 디렉토리에 만들고자 할 때 /smith/public_html/folios/graphics.psf와 같이 입력해야 합니다.

  3. [만들기] 버튼을 클릭합니다.
    Java Development Environment은 이 포트폴리오를 현재 포트폴리오로 설정합니다. 인터넷에 연결된 다른 WorkShop 사용자들은 이 포트폴리오의 URL을 브라우저에 입력하여 액세스할 수 있습니다. 다음 과정에서는 프로젝트를 포트폴리오 복사하는 방법에 대해 설명합니다.
