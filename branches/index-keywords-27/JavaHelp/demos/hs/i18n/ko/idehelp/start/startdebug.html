제 1과: 디버깅 세션 시작하기

이 과에서는 Blink 프로젝트 디버깅을 시작하는 방법을 설명합니다. 아직 Blink 프로젝트를 만들어 빌드하지 않았다면, 이제 Blink 프로젝트 만들기에 나오는 지침을 사용하여 해 봅시다.
  1. 프로젝트 관리자에서 현재 프로젝트의 이름을 확인하십시오.
    Blink.java 파일이 소스 편집기에 나타나 있어야 합니다. 현재 프로젝트가 Blink가 아니라면, Blink라는 프로젝트 이름을 마우스로 두 번 연속 누르십시오.
  2. 편집/디버그 도구 모음에서 [다시 시작] 단추 를 마우스로 누르십시오.
    Blink.java 실행이 시작됩니다. Java Development Environment는 Blink 프로젝트를 [애플릿 뷰어] 창에서 실행합니다.

Blink.java가 실행되는 동안, [디버그] -> [중단점]을 선택하여 중단점을 설정할 수 있습니다. 또한, [디버그] -> [프로그램 중단]을 선택하여 프로그램을 일시 정지시킬 수도 있습니다.
