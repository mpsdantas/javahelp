[도움말] 메뉴

[도움말 목차] 이 메뉴 명령을 선택하면 온라인 도움말의 목차가 열립니다.
[작업별 도움말] 이 메뉴 명령을 선택하면 일반적인 작업에 대한 도움말 항목 메뉴가 화면에 나타납니다.
[메뉴 도움말] 이 메뉴 명령을 선택하면 Java Development Environment의 도움말 항목 목록이 화면에 나타납니다.
[Java Development Environment 자습서] 이 메뉴 명령을 선택하면 Java Development Environment 자습서가 실행됩니다. 이 자습서는 애플릿 프로젝트를 만들고, 빌드하고, 디버깅하는 전체 과정을 단계적으로 안내해 주는 일련의 여덟 가지 대화식 자습서입니다.
[릴리스 참고 사항] 이 메뉴 명령을 선택하면 이 Java Development Environment 릴리스에 관한 최신 정보가 나옵니다.
[일련 번호/라이센스] 이 메뉴 명령을 선택하면 Java Development Environment 일련 번호와 라이센스 정보가 화면에 나타납니다.
[등록] 이 메뉴 명령을 선택하면 [Java Development Environment 등록] 대화 상자가 열립니다. 사용하는 Java Development Environment 프로그램을 전자 방식으로 등록하려면 이 대화 상자를 사용하십시오.
[의견 보내기] 이 메뉴 명령을 선택하면 Java Development Environment [의견란]이 열립니다. Java Development Environment 제품 개발팀으로 의견이나 제안을 전자 방식으로 보내려면 이 [의견란]을 사용하십시오.
[Java API 문서 (1.1)] 이 메뉴 명령을 선택하면 도움말 뷰어에 대한 1.1 Java API 문서가 화면에 나타납니다. 온라인 도움말의 목차를 통하여 이 설명서로 들어갈 수도 있습니다.
[Java 언어 규정] 이 메뉴 명령을 선택하면 James Gosling, Bill Joy, 그리고 Guy Steele이 제작한 "Java 언어 규정"의 목차가 화면에 나타납니다.
[Java Development Environment 정보] 이 메뉴 명령을 선택하면 [제품 정보] 란이 열립니다. [제품 정보] 란에는 다음 정보가 들어 있습니다:
  • 일련 번호
  • 버전 번호
  • Java Development Environment 개발 팀 사진
  • 독점 소유권 표시
