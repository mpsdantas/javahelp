Java Development Environment 자습서

본 자습서의 목표는 Java Development Environment의 주요 기능을 사용하여 프로젝트를 만들고, 수정하고, 빌드하고, 디버깅하고, 실행하는 방법을 보여 주는 것입니다.

첫번째 항목은 Java Development Environment에서 사용된 개념, 도구, 및 개발 과정에 익숙해지게 하기 위해 마련된 것입니다. Java Development Environment의 기본적인 특징들을 익히려면, 자습서 1에서 5까지 차례로 익히십시오. 자습서 6과 7은 Java Development Environment의 고급 기능들을 설명합니다.

  • Java Development Environment 첫걸음에서는 Java Development Environment을 사용하여 인터넷 응용 프로그램과 웹 페이지를 만드는 과정을 설명합니다.
  • 자습서 1: Blink 프로젝트 만들기에서는 Java Development Environment에서 애플릿을 만들고 빌드하고 실행하는 기본적인 방법을 차례로 설명합니다.
  • 자습서 2: 프로젝트 속성 수정하기에서는 프로젝트의 주요 특징을 수정하는 과정을 단계적으로 설명합니다.
  • 자습서 3: 소스 코드에서 오류 바로잡기는 빌드 과정 중에 오류를 바로잡는 법과 프로젝트를 다시 빌드하는 법을 설명합니다.
  • 자습서 4: 프로젝트 찾아보기는 프로젝트를 구성하는 클래스와 메서드를 찾아보는 방법을 보여 줍니다. 소스 코드 내에서 문자열을 검색하는 방법도 익힐 것입니다.
  • 자습서 5: 프로젝트를 디버깅하기에서는 프로젝트가 제대로 실행되지 못하게 하는 버그를 찾아서 바로잡는 방법을 설명합니다. 디버거를 사용하여 스레드를 검사하고, 중단점을 설정하고, 표현식을 검사하게 될 것입니다.
  • 자습서 6: 실행 페이지 수정하기에서는 프로젝트를 실행할 때 자동으로 만들어지는 HTML을 수정하는 방법을 설명합니다.
  • 자습서 7: GUI 만들기에서는 프로젝트에 맞는 사용자 인터페이스를 구성하는 방법을 설명합니다.
  • 자습서 8: Clock 프로젝트 만들기에서는 Bean 프로젝트를 만드는 방법과 JavaBeans을 GUI 빌더로 임포트하는 방법을 설명합니다.
