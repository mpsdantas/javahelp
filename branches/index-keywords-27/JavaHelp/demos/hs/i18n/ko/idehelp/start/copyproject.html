과정 2: 프로젝트 복사 및 붙여넣기

이 연습은 개인 포트폴리오에 있는 프로젝트를 복사하여 공용 포트폴리오에 붙여 넣는 방법을 보여줍니다. 공용 포트폴리오가 없다면 지금 만드십시오. 포트폴리오를 만드는 방법에 대한 단계별 절차는 포트폴리오 만들기를 참고하십시오.

프로젝트 복사

  1. [포트폴리오] => [선택] =>개인을 선택하십시오.
    그러면, Java Development Environment에서 개인 포트폴리오를 엽니다. 개인 포트폴리오가 없다면 개인 포트폴리오 임포트를 참고하십시오.

  2. WorkShop 도구 모음에서 [포트폴리오 관리자] 컨트롤을 클릭합니다.

  3. Blink 프로젝트를 선택합니다.
    Blink 프로젝트가 없다면 프로젝트 만들기 에서 만드는 방법을 참고하십시오.

  4. [포트폴리오 관리자] 도구 모음에서 [복사] 컨트롤을 클릭합니다.
    그러면, WorkShop에서 Blink 프로젝트를 클립보드로 복사합니다.

프로젝트 붙여넣기

  1. [포트폴리오]=>[선택]을 선택하고 공용 포트폴리오 이름을 선택합니다.
    그러면, WorkShop이 공용 포트폴리오를 엽니다.

  2. [포트폴리오 관리자] 도구 모음에서 [붙여넣기] 컨트롤을 클릭합니다.
    그러면, WorkShop에서 [프로젝트 붙여넣기] 대화 상자를 엽니다. [프로젝트 파일] 필드에 Blink 프로젝트의 경로 이름이 나타납니다.

  3. [작성자] 필드의 오른쪽에 있는 복사 라디오 버튼이 선택되어야 합니다.
    이 버튼은 WorkShop에서 모든 프로젝트 파일을 (Blink.prj, Blink.java, Blink.class) 대상 디렉토리에 복사하도록 지정하는 것입니다. WorkShop에서 기본 HTML 파일인 Blink.tmp.html 파일은 복사하지 않습니다.

  4. 대상 디렉토리에서 파일을 저장할 디렉토리 경로 이름을 입력합니다.
    이 디렉토리는 HTTP 서버에서 액세스 가능한 디렉토리여야 합니다.

  5. [포함 소스] 필드의 오른쪽에 있는 [예] 라디오 버튼을 클릭합니다.

  6. Blink가 [새 프로젝트 이름] 필드에 나타나야 합니다.

  7. [붙여넣기] 버튼을 클릭합니다.
    프로젝트 이름과 이미지가 공용 포트폴리오에 나타납니다. 이 프로젝트 파일은 Blink.jws.src 및 Blink.jws 두 파일과 함께 대상 디렉토리에 추가됩니다. 첫째 파일은 프로젝트 소스 파일을 포함하고 있으며, 둘째 파일은 .prj 파일과 같은 기타 모든 파일을 포함하고 있습니다. 다른 사용자가 이 프로젝트를 복사할 때 WorkShop은 개별 프로젝트 파일 대신 .jws.src 및 .jws 파일을 복사합니다.
