제 1과: Blink 프로젝트 만들기

이 과에서는 애플릿 프로젝트--웹 브라우저 내에서 실행하도록 만들어진 Java 프로그램--을 만드는 방법을 보여줍니다. 이제 만들게 될 프로젝트인 Blink는 텍스트를 여러 가지 색으로 깜박거리게 하는 자그마한 애플릿입니다. Blink 디렉터리에 설치되어 있는 이미 존재하는 소스 파일인 Blink.java를 사용하여 프로젝트를 만들어 봅시다.
  1. Java Development Environment 주 도구 모음에서 [프로젝트] 단추 를 마우스로 누르십시오.
    [Java Development Environment 프로젝트 관리자] 창이 화면에 나타납니다.
  2. 프로젝트 관리자에서 현재 포트폴리오의 이름을 확인하십시오--현재 포트폴리오의 이름은 파랑색 텍스트로 표시됩니다.
    현재 포트폴리오가 personal이 아니라면, personal 포트폴리오 이름을 마우스로 두 번 연속 누르십시오. personal 포트폴리오가 없으면, personal 포트폴리오 추가하기를 참조하십시오.
  3. [파일] -> [새로 만들기] -> [프로젝트]를 선택하십시오 (반드시 [프로젝트 관리자] 창의 [파일] 메뉴로 가야 합니다).
    [프로젝트 만들기 마법사] 창이 열립니다.
  4. [프로젝트 이름] 란에서 untitled를 선택하고 Blink라고 입력하십시오.
    Blink에서 "B"를 반드시 대문자로 표시해야 합니다. Java는 대소문자를 구분합니다. 프로젝트 이름 Blink는 소스 파일 이름 Blink.java과 일관성이 있어야 합니다. 소스 파일 이름도 첫 글자를 대문자로 표시하는 Java의 관례를 따릅니다.
  5. 응용 프로그램 유형으로 [애플릿] 선택형 단추를 마우스로 누르십시오.
  6. [GUI 없음] 선택형 단추를 누르십시오.
    이 프로젝트에서는 그래픽 방식 사용자 인터페이스를 사용하지 않을 것입니다.
  7. [다음]을 마우스로 누르십시오.
  8. Blink 프로젝트를 기본 설정된 디렉터리에 저장해야 합니다.
  9. [예]를 마우스로 눌러 이미 존재하는 Java 파일을 Blink 프로젝트에 추가하십시오.
  10. [다음]을 마우스로 누르십시오.
  11. ([프로젝트 파일] 목록 아래의) [디렉터리내 모든 항목 추가] 단추를 마우스로 누르십시오.
    파일 이름 Blink.java가 [프로젝트 파일] 목록에 나타납니다.
  12. [다음]을 마우스로 누르십시오.
    [애플릿 클래스] 란에, Blink가 자동으로 입력됩니다. main 클래스 파일은 Java 라이브러리의 Applet 클래스를 확장하는 .class 파일입니다.
  13. [다음]을 마우스로 누르십시오.
  14. Java Development Environment이 생성한 html 페이지를 사용해야 합니다.
  15. [마침]을 마우스로 누르십시오.
    소스 편집기 창에 Blink.java 소스 파일이 나타나고 그 파일의 이름이 Java Development Environment 제목 표시줄에 나타납니다. 프로젝트 관리자에서, personal 포트폴리오에 Blink 프로젝트가 현재 프로젝트로 나타납니다. 현재 프로젝트는 java WorkShop의 모든 도구를 사용하는 환경이 됩니다.

    이 예제에서는, Blink 디렉터리에 있는 Blink.prj 파일에 Blink 프로젝트를 만들었습니다. 앞으로 프로젝트를 만들 때는 사용하는 파일 시스템의 임의의 디렉터리에서 프로젝트를 만들 수 있습니다.
