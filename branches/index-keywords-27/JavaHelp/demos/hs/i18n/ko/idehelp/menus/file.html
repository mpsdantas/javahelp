[파일] 메뉴

[새로 만들기] 이 메뉴 명령을 선택하면 소스 편집기가 빈 화면으로 바뀝니다. 텍스트를 입력한 다음 새 파일로 저장할 수 있습니다. 소스 편집기에 관한 더 자세한 내용은 소스 코드 수정하기를 참조하십시오.
[열기] 이 메뉴 명령을 선택하면 소스 편집기에 이미 존재하는 파일이 열립니다.
[새 편집기 창] 이 메뉴 명령을 선택하면 파일이 로딩되지 않은 상태로 새 소스 편집기 창이 나타납니다.
[복제 편집기 창] 이 메뉴 명령을 선택하면 새 편집기 창이 열리고 편집기 창에 있는 동일한 파일이 복제되어 로딩됩니다. 이 과정은 여러 번 반복할 수 있습니다. 더 자세한 내용은 기본적인 수정 방법을 참조하십시오.
[소스 파일 다시 로드] 이 메뉴 명령을 선택하면 소스 편집기에 현재 로딩되어 있는 파일이 다시 로딩됩니다.
[저장] 이 메뉴 명령을 선택하면 소스 편집기에 현재 로딩되어 있는 파일을 현재의 파일 이름으로 저장합니다.
[다른 이름으로 저장] 이 메뉴 명령을 선택하면 소스 편집기에 현재 로딩되어 있는 파일을 [파일 선택] 대화 상자를 사용하여 지정하는 이름으로 저장합니다.
[모든 소스 파일 저장] 이 메뉴 명령을 선택하면 활성 소스 편집기 창에 현재 로딩되어 있는 모든 파일을 저장합니다.
[버전] 파일 버전 제어 명령입니다. 더 자세한 내용은 버전 제어를 참조하십시오
파일 선택 목록 Java Development Environment은 소스 편집기에 로딩된 파일 목록을 관리합니다. 편집기에 파일을 로딩할 때마다, 그 이름이 목록의 제일 위에 추가됩니다. 소스 편집기에 파일을 로딩하려면 이 목록에서 파일을 선택하십시오.

이 목록에는 가장 최근에 편집한 12 개의 파일이 편집기에 로딩된 순서로 들어 있습니다. 이 목록은 Java Development Environment 세션을 통하여 관리합니다. 파일 시스템에서 파일을 지우면, 목록에서도 지워집니다.

[종료] 이 메뉴 명령을 선택하면 Java Development Environment이 종료합니다.
