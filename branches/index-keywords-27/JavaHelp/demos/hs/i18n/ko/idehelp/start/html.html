제 1과: HTML 파일 수정하기

이 과에서는, Blink 프로젝트를 실행했을 때 자동으로 만들어진 HTML 파일을 수정할 것입니다. 애플릿 매개 변수를 지정하고 소스 코드로 연결되는 링크를 추가할 것입니다. 파일을 수정한 후에 파일을 다른 파일 이름으로 저장하여 다음에 해당 프로젝트를 실행해도 변경 내용이 지워지지 않게 할 것입니다.

아직 Blink 프로젝트를 만들어서 빌드하지 않았다면, Blink 프로젝트 만들기의 방법대로 지금 만들어서 빌드하십시오.

  1. 프로젝트 관리자에서 현재 프로젝트의 이름을 확인하십시오.
    현재 프로젝트가 Blink가 아니라면, Blink라는 프로젝트 이름을 마우스로 두 번 연속 누르십시오.
  2. 소스 편집기에서, [파일] -> [열기]를 선택하십시오.
    파일 선택창이 열립니다.
  3. Blink 디렉터리에 있는 Blink.tmp.html 파일을 여십시오.
    이 파일에는 다음과 비슷한 애플릿 태그가 들어 있습니다: 
    <applet
   name="Blink"
   code="Blink.class"
   codebase="/your_home_directory/jws/Blink"
   width="500"
   height="600"
   align="Top"
   alt="If you had a java-enabled browser, 
   you would see an applet here."
> 
<param name="lbl" value="Java Development Environment is really cool :-)">
<hr>If your browser recognized the applet tag,
      you would see an applet here.<hr>
</applet>

    이 파일은 프로젝트를 실행했을 때 자동으로 만들어졌습니다. 이 파일에 들어 있는 정보는 [프로젝트 편집] 창의 [실행] 탭에 있는 정보에 기초한 것입니다. 프로젝트를 실행한 적이 없다면, 프로젝트 실행하기를 참조하십시오.

  4. Java Development Environment 도구 모음에서 [파일] -> [다른 이름으로 저장]을 선택하십시오.
    [현재 파일 저장] 대화 상자가 화면에 나타납니다.
  5. Blink 디렉터리를 선택한 다음, HTML 파일의 새 이름을 입력하고 [확인]을 마우스로 누르십시오.
    반드시 HTML 파일의 이름을 변경해야 합니다. 그렇지 않으면, 다음에 해당 프로젝트를 실행할 때 변경 내용이 지워질 것입니다. 또한, 파일을 반드시 Blink 디렉터리에 저장해야 합니다.
  6. 소스 편집기에서, 높이의 값을 75로 변경하십시오.
  7. lbl 매개 변수 앞에 다음 매개 변수를 추가하여 단어가 깜박거리는 속도를 증가시키십시오: 
    <param name="speed" value="50">
  8. <applet> 태그 다음의 소스 코드에 다음 링크를 추가하십시오: 
    <p>
<a href="Blink.java">Source Code</a>
    소스 코드의 경로는 HTML 파일이 들어 있는 디렉터리를 기준으로 한 것이어야 합니다.
  9. 편집/디버그 도구 모음에서 [저장] 단추 를 마우스로 누르십시오.
