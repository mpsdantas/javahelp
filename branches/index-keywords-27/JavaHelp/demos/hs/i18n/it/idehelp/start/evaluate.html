Esercizio 5: Valutazione delle espressioni

Oltre alla finestra di dialogo Thread/stack, per valutare le espressioni è possibile utilizzare la finestra Valutazione di espressione. In questo esercizio verrà prima valutata un'espressione, quindi ripresa l'esecuzione dell'applet Blink.
  1. Scegliere Debug -> Valuta.
    Verrà visualizzata la finestra Valutazione di espressione.
  2. Digitare speed nella casella Espressione.
  3. Scegliere il pulsante Valuta.
    L'espressione speed e il relativo valore verranno visualizzati nella scheda Debug.
  4. Fare clic sul pulsante Vai sulla barra degli strumenti dell'editor/debug.
    Verrà ripresa l'esecuzione del progetto Blink.

Lezione successiva
