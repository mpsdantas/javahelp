Esercitazione 1: Creazione di un progetto

Questa esercitazione è rivolta ai nuovi utenti di Java Development Environment. In questa esercitazione verrà spiegato come creare, compilare ed eseguire un progetto con Java Development Environment.

L'esercitazione 1 è suddivisa in cinque lezioni. Per ottenere i risultati migliori, è opportuno eseguire le lezioni nell'ordine di presentazione.

  • Nella sezione Concetti importanti: progetti e portafogli è contenuta una panoramica completa sul funzionamento di progetti e portafogli. Prima di proseguire nell'esercitazione, è opportuno avere dimestichezza con questi concetti.
  • Nell'Esercizio 1: Creazione del progetto Blink viene spiegato come creare un progetto contenente un applet. Un applet è un piccolo programma che viene eseguito in un browser che supporta il linguaggio Java.
  • Nell'Esercizio 2: Creazione di un progetto viene spiegato come creare un applet.
  • Nell'Esercizio 3: Esecuzione di un progetto viene spiegato come eseguire l'applet e visualizzarlo utilizzando il relativo visualizzatore.
  • Nell'Esercizio 4: Raccolta dei dati del profilo viene spiegato come ottenere le informazioni sul profilo.
