Esercitazione di Java Development Environment

Lo scopo delle esercitazioni è quello di mostrare l'uso delle funzioni principali di Java Development Environment al fine di consentire la creazione, la modifica, la compilazione, il debug e l'esecuzione di un progetto.

Nella prima sezione sono riportate informazioni che consentono di acquisire una certa familiarità con i concetti, gli strumenti e il processo di sviluppo utilizzati in Java Development Environment. Per imparare a utilizzare le funzioni di base di Java Development Environment, seguire le esercitazioni da 1 a 5. Nelle esercitazioni 6 e 7 sono invece illustrate le funzioni avanzate di Java Development Environment.

  • Introduzione a Java Development Environment: spiega come utilizzare Java Development Environment per creare applicazioni Internet e pagine web.
  • Esercitazione 1: Creazione di un progetto Blink: illustra i principi fondamentali relativi alla creazione, compilazione ed esecuzione di un applet con Java Development Environment.
  • Esercitazione 2: Modifica degli attributi di un progetto: guida l'utente nelle operazioni di modifica delle caratteristiche di un progetto.
  • Esercitazione 3: Correzione degli errori nel codice sorgente: spiega come correggere gli errori di compilazione e come ricompilare il progetto.
  • Esercitazione 4: Sfogliare un progetto: illustra come effettuare ricerche nelle classi e nei metodi che costituiscono il progetto. In questa esercitazione viene anche spiegato come ricercare delle stringhe nel codice sorgente.
  • Esercitazione 5: Debug di un progetto: illustra come individuare e correggere errori che non consentono la corretta esecuzione del progetto. Verrà utilizzato il debugger per controllare i thread, impostare dei breakpoint ed esaminare delle espressioni.
  • Esercitazione 6: Personalizzazione di una pagina di esecuzione: illustra come modificare il file HTML creato da Java Development Environment quando si esegue un progetto.
  • Esercitazione 7: Creazione di un'interfaccia grafica (GUI): illustra come assemblare un'interfaccia grafica per un progetto.
  • Esercitazione 8: Creazione di un progetto Clock: illustra come creare un progetto Bean e come importare JavaBeans nel GUI builder.
