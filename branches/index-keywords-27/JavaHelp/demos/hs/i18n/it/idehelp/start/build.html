Esercizio 2: Creazione del progetto

In questo esercizio verrà illustrato come creare il progetto Blink. Al termine della creazione, il progetto verrà eseguito. A tale scopo, il codice sorgente Java del progetto verrà compilato in un file denominato nomefile.class.

Se non si è ancora creato il progetto Blink, sarà necessario farlo seguendo i passaggi illustrati nella sezione Creazione del progetto Blink.

  1. Verificare il nome del progetto corrente nella finestra del gestore progetti di Java Development Environment.
    Se il progetto corrente non è Blink, fare doppio clic sul nome di progetto Blink.
  2. Scegliere il pulsante Compila sulla barra degli strumenti principale di Java Development Environment.
    Il programma Blink.java verrà compilato nel file di bytecode Blink.class. È possibile assistere ai vari passaggi della creazione del progetto all'interno della scheda Compila situata sotto l'editor del codice sorgente. Al termine della creazione e della compilazione dei file sorgente verrà visualizzato un messaggio di stato relativo al completamento dell'operazione.
