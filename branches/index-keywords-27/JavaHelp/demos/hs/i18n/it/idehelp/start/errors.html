Esercitazione 3: Correzione degli errori nel codice sorgente

Nelle lezioni di questa esercitazione verrà illustrato come modificare il codice sorgente del progetto Blink e come ricreare il progetto. Per ottenere i massimi risultati, è necessario svolgere le lezioni nell'ordine nel quale sono presentate.
  • Nell'Esercizio 1: Modifica del codice sorgente vengono introdotte alcune delle funzioni fondamentali dell'editor del codice sorgente di Java Development Environment.
  • Nell'Esercizio 2: Risoluzione degli errori di creazione vengono corretti gli errori di compilazione individuati nel corso del processo di creazione.
