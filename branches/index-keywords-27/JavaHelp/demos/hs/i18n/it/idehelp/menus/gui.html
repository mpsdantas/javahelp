Menu GUI

Mostra GUI builder Aggiunge i controlli del GUI builder alla finestra di Java Development Environment. Questa voce è attiva solo per i progetti creati utilizzando Visual Java GUI Builder.
Unisci file GUI Apre la finestra di selezione dei file e importa il file selezionato, cioè un file contenente una finestra del GUI builder che si desidera includere nell'applicazione. Il file deve avere l'estensione .gui.
Salva e genera file GUI Salva il formato sul disco nel file nome-progetto.gui e genera il codice sorgente Java dal formato. Per ulteriori informazioni, vedere Generazione del codice sorgente Java.
Taglia componente Taglia il componente o la finestra selezionati e li inserisce negli appunti. Il componente o la finestra tagliati possono essere incollati in un'altra cella nel formato.
Copia componente Copia il componente o la finestra selezionati negli appunti. Il componente o la finestra copiati possono essere incollati dagli appunti in un'altra cella nel formato.
Incolla componente Incolla un componente o una finestra tagliati o copiati in un'altra cella nel formato.
Elimina componente Elimina il componente o la finestra selezionati. Non inserisce una copia del componente negli appunti, pertanto non è possibile annullare l'effetto prodotto dal comando.
Modifica attributi Apre l'editor degli attributi in cui vengono visualizzati gli attributi del componente o della finestra selezionati. Per ulteriori informazioni sulla modifica degli attributi di un componente vedere la Guida dell'editor degli attributi. Per questo comando è disponibile anche un pulsante sulla barra degli strumenti principale.
Aggiungi colonna Aggiunge una colonna sulla sinistra o sulla destra della cella o del componente selezionati.
Aggiungi riga Aggiunge una riga sopra o sotto la cella o il componente selezionati.
Aggiungi nuova finestra Visualizza la finestra di dialogo Nuova finestra. Selezionare un contenitore specifico dall'elenco e fare clic su Crea per aprire un nuovo formato griglia per il contenitore specifico. Per questo comando è disponibile anche un pulsante visualizzato alla sinistra della tavolozza dei componenti.
Imposta tipo di finestra Visualizza la finestra di dialogo Contenitore principale. Utilizzare questa finestra per modificare il tipo di contenitore principale in una cornice o in un pannello.
Annida pannello Annida il componente selezionato del pannello del formato in un componente Pannello formato griglia.
Personalizza tavolozza Visualizza l'editor della tavolozza. Utilizzare questo editor per personalizzare, eliminare e riorganizzare i componenti nelle cartelle della tavolozza.
Importa componenti nella tavolozza Visualizza la finestra di dialogo Importa componenti nella tavolozza. Utilizzare questa finestra per importare JavaBeans o componenti con classi shadow. Per ulteriori informazioni, vedere Importazione di componenti.
Ripristina tavolozza predefinita Ripristina il contenuto originale della tavolozza standard del GUI builder (schede AWT, Pannelli e Aggiuntive).
BeanInfo per componente selezionato Visualizza la finestra di dialogo Informazioni Bean in cui sono riportate informazioni specifiche sui Bean, che includono le proprietà, gli eventi e i metodi.
Aggiorna classi componenti nel disegno Visualizza la finestra di dialogo Aggiorna classi componenti nel disegno in cui è riportato un elenco di classi Bean. Selezionare una classe o più classi da aggiornare. Viene aggiornato il componente nella tavolozza, ma non quello nella finestra del formato. Se è necessario rieseguire il caricamento, verrà visualizzata una finestra di dialogo Ricarica design.
Rapporto utilizzo classi Visualizza un elenco delle classi di componenti utilizzate nel progetto corrente.
