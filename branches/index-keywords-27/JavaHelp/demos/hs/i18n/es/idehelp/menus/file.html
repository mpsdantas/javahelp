Menú "Archivo"

Nuevo Vacía el contenido del editor de código fuente. Puede introducir texto y guardarlo como un archivo nuevo. VéaseEdición de código fuente para más información acerca del editor de código fuente.
Abrir Abre un archivo existente en el editor de código fuente.
Ventana de editor nueva Inicia una nueva ventana de código fuente del editor sin ningún archivo cargado.
Ventana del editor de clones Abre un nueva nueva ventana de edición cargada con el mismo archivo que había en la ventana de edición desde donde se clonó. Este proceso puede repetirse muchas veces. Véase Edición básica para más información.
Recargar archivo fuente Recarga el archivo actualmente cargado en el editor de código fuente.
Guardar Guarda el archivo actualmente cargado en el editor de código fuente utilizando el nombre actual del archivo.
Guardar como Guarda el archivo actualmente cargado en el editor de código fuente con un nombre que se proporciona mediante un cuadro de diálogo de selección de archivos.
Guardar todos los archivos fuente Guarda todos los archivos actualmente cargados en las ventanas de edición de código fuente activas.
Versión Comandos de control de versiones de archivos. Véase Control de versiones para más información.
Lista de selección de archivos Java Development Environment conserva una lista de los archivos que se cargan en el editor de código fuente. Cada vez que carga un archivo en el editor, su nombre se añade al principio de la lista. Elija un archivo de esta lista para cargarlo en el editor de código fuente.

Esta lista contiene los 12 archivos más recientemente modificados en el orden en que se cargaron en el editor. Esta lista se conserva de una sesión a otra de Java Development Environment. Si se suprime un archivo del sistema de archivos, también se elimina de la lista.
