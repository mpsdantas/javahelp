Ejercicio 1: Visualización de la jerarquía de clases

En este ejercicio, aprenderá cómo se derivan las clases del proyecto Blink.
  1. Consulte en el administrador de proyectos el nombre del proyecto actual.
    Si el nombre del proyecto actual no es Blink, pulse dos veces en el nombre de proyecto Blink.
  2. Seleccione "Examinar -> Ver clases".
    Java Development Environment abre el visualizador de clases, en donde se listan las clases, métodos y comentarios que constituyen el proyecto Blink. El proyecto sólo tiene un archivo fuente, Blink.java.
  3. Pulse en Blink.java.
    El visualizador de clases carga las clases de las que se deriva Blink.
    Blink se deriva de Applet, que a su vez se deriva de Panel, y así sucesivamente.

    La información del visualizador de clases se deriva de javadoc, generado por la Documentación de la API de Java. Javadoc analiza la declaración y comentarios del código fuente del proyecto y da formato a la API con código HTML.
