Terminología de Java Development Environment

Los siguientes términos se utilizan a lo largo de toda la Ayuda en línea. La comprensión de estos términos le puede ayudar a realizar los ejercicios del tutorial más rápidamente. Otros términos específicos de Java Development Environment (como applet y subproceso) se explican la primera vez que aparecen en el tutorial.

A medida que avance en el tutorial, preste atención al icono animado de la esquina superior derecha del examinador. El cubo gira cuando Java Development Environment está cargando una página de ayuda.

Botón
Es un elemento de la barra de herramientas o de la ventana que se puede pulsar para que se produzca una acción. Por ejemplo, si pulsa el botón "Proyectos" , aparece la ventana del Administrador de proyectos de Java Development Environment.

Los botones de la barra de herramientas tienen pistas para las herramientas. Si no está seguro de qué acción realiza un botón de la barra de herramientas, coloque el puntero encima del botón para mostrar una descripción de éste. Por ejemplo, si coloca el puntero encima del botón , se puede leer "Cadena de búsqueda".

Menú
Es una lista de elementos donde puede elegir una tarea o un objeto en particular. Los menús aparecen en el examinador, entre la barra de título y la barra de herramientas principal de Java WorkShop. Las referencias a opciones de menús suelen tener el formato "Menú -> Opción de menú". Por ejemplo, "Proyecto -> Construir" significa la elección de la opción "Construir" del menú "Proyecto."
Pestaña
Página con pestañas (similar a un cuadro de diálogo) que aparece cuando pulsa en una ventana. Por ejemplo, la ventana "Editar proyecto" contiene las siguientes pestañas: "General", "Construir", "Depurar/Examinar" y "Ejecutar".
