Adición de su cartera personal

La primera vez que se ejecuta Java Development Environment, se crea una cartera llamada "personal," donde podrá escribir. Si elimina su cartera personal, pero no elimina el archivo (.psf) de la cartera de su sistema de archivos, podrá importarla de nuevo a Java Development Environment.

Si no es así, podrá utilizar para este tutorial cualquier cartera en la que tenga permiso para escribir. O bien, podrá crear una nueva cartera utilizando la ayuda que se ofrece en Creación de una cartera.

En las siguientes instrucciones se describe la forma de añadir su cartera personal:

  1. En la ventana "Administrador de proyectos" de Java Development Environment, seleccione "Archivo -> Añadir -> Cartera".
    Aparecerá el cuadro de diálogo "Añadir cartera".
  2. Escriba /directorio_instalación/jws/personal.psf en el cuadro "Archivo de cartera".
    Todos los nombres de archivo de cartera tienen la extensión .psf.
  3. Pulse en "Añadir".
    Java Development Environment convertirá su cartera personal en la cartera actual y mostrará su nombre en texto azul.
