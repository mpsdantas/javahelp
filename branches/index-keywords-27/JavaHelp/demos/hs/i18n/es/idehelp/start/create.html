Ejercicio 1: Creación del proyecto Blink

Este ejercicio le muestra cómo crear un proyecto applet: un programa Java diseñado para su ejecución dentro de un examinador de Web. El proyecto que creará, Blink, es una pequeña aplicación que muestra intermitentemente un texto, en diferentes colores. El proyecto se crea utilizando un archivo fuente existente, Blink.java, que se instaló en el directorio Blink.
  1. Pulse en el botón "Proyectos" en la barra de herramientas principal de Java Development Environment.
    Aparece la ventana del Administrador de proyectos de Java WorkShop.
  2. Compruebe en el administrador de proyectos el nombre de la cartera actual, que aparece en texto azul.
    Si la cartera actual no es personal, pulse dos veces en en el nombre de la cartera. Si no tiene una cartera personal, véase Adición de una cartera personal.
  3. Seleccione "Archivo -> Nuevo -> Proyecto" (asegúrese de ir al menú "Archivo" en la ventana del administrador del proyecto).
    Se abre la ventana del asistente de creación de proyectos.
  4. Seleccione sin título y escriba Blink en el cuadro de nombre del proyecto.
    Asegúrese de escribir la primera letra de Blink en mayúscula. Java distingue entre mayúsculas y minúsculas. El nombre del proyecto Blink es coherente con el nombre del archivo fuente Blink.java, que sigue la convención de Java en cuanto a las mayúsculas iniciales.
  5. Pulse en el botón de radio "Applet" correspondiente al tipo de aplicación.
  6. Pulse en el botón de radio "Ninguna GUI".
    Este proyecto no tendrá ninguna interfaz gráfica de usuario.
  7. Pulse en "Siguiente".
  8. Debe guardar el proyecto Blink en el directorio predeterminado.
  9. Pulse en "Sí" y añada los archivos Java existentes al proyecto Blink.
  10. Pulse en "Siguiente".
  11. Pulse en el botón "Añadir todo en directorio" (debajo de la lista de archivos del proyecto).
    El nombre de archivo Blink.java aparece en la lista "Archivos del proyecto".
  12. Pulse en "Siguiente".
    En el cuadro "Clase de applet", Java Development Environment introduce Blink. El archivo de clase main es el archivo .class que amplía la clase Applet de la biblioteca Java.
  13. Pulse en "Siguiente".
  14. Debe utilizar la página html generada por Java Development Environment.
  15. Pulse en "Finalizar".
    La ventana del editor de código fuente muestra el archivo fuente Blink.java con su nombre en la barra de título de Java Development Environment. En el administrador de proyectos, su cartera personal muestra el proyecto Blink como proyecto actual. El proyecto actual define el contexto de todas las herramientas de Java Development Environment.

    En este ejemplo, ha creado el proyecto Blink en el archivo Blink.prj en el directorio Blink. Cuando cree un proyecto en el futuro, puede crearlo en cualquier directorio del sistema de archivos.
