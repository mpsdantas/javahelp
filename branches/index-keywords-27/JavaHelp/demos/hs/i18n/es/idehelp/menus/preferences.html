Menú "Preferencias"

Agentes delegados Utilice el cuadro de diálogo "Agentes delegados" para cambiar los servidores delegados (proxies).
Control de versiones Selecciona un sistema de control de versión para utilizarlo con Java Development Environment.
Editor de código fuente Define preferencias para los siguientes comportamientos del editor de código fuente:
  • Sangría/Tabuladores
  • Resaltado de sintaxis y colores
  • Selección de fuentes
Teclado Permite ver, añadir, o eliminar asignaciones de teclas (vinculaciones) para grupos funcionales dentro de Java Development Environment. Véase Preferencias del teclado para obtener más información al respecto.
Entrada de consola Determina la forma en que se interpretará la entrada del programa basada en línea de comandos en la ventana de la consola. Véase el cuadro de diálogo "Entrada de consola" para obtener más información al respecto.
Constructor Determina el compilador Java que utilizará Java Development Environment para compilar sus proyectos. Véase el cuadro de diálogo "Preferencias del constructor" para obtener más información al respecto.
Mostrar etiquetas en barra de herramientas Añade etiquetas debajo de los botones de la barra de herramientas.
