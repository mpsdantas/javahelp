Menú "Ayuda"

Índice de la Ayuda Abre la tabla de contenido de la Ayuda en línea.
Ayuda basada en tareas Muestra un menú de temas de ayuda para las tareas comunes.
Ayuda de menús Muestra una lista de temas de ayuda para los menús de Java Development Environment.
Tutorial de Java Development Environment Inicia el tutorial de Java Development Environment, una serie de ocho tutoriales interactivos que le guían paso a paso por el proceso de creación, construcción y depuración de un proyecto applet.
Notas sobre la versión Proporciona información de última hora sobre esta versión de Java Development Environment.
Número de serie/Licencia Muestra la información del número de serie y la licencia de Java Development Environment.
Registro Abre el cuadro de diálogo "Registro de Java Development Environment". Utilice este cuadro de diálogo para registrar electrónicamente su copia de Java Development Environment.
Enviar comentarios Abre el cuadro "Comentarios de Java WorkShop". Utilice el cuadro "Comentarios" para enviar electrónicamente comentarios y sugerencias al equipo del producto de Java Development Environment.
Documentación de Java API (1.1) Muestra la documentación de Java API 1.1 en el visualizador de la Ayuda. También puede acceder a esta documentación a través de la tabla de contenidos de la Ayuda en línea.
Especificación del lenguaje Java Muestra la tabla de contenidos del documento "Java Language Specification", de James Gosling, Bill Joy y Guy Steele
Acerca de Java Development Environment Abre el cuadro "Acerca de". El cuadro "Acerca de" contiene la siguiente información:
  • Número de serie
  • Número de versión
  • Fotografía del equipo de desarrollo de Java Development Environment
  • Aviso de derechos de propiedad intelectual
