Menú "GUI"

Mostrar constructor de GUI Añade controles del constructor de GUI a la ventana de Java Development Environment. Este elemento está activo sólo para los proyectos creados con el constructor de GUI de Visual Java.
Fusionar archivo de GUI Abre un selector de archivos e importa el archivo seleccionado (un archivo que contiene una ventana del constructor de GUI que desea incluir en la aplicación). El archivo debe tener una extensión .gui.
Guardar y generar archivo de GUI Guarda el diseño en disco en el archivo nombre-proyecto.gui y genera código fuente Java a partir del diseño. Véase Generación de código fuente Java para obtener más detalles.
Cortar componente Corta la ventana o el componente seleccionado y lo coloca en el portapapeles. Puede pegar el componente o la ventana que haya cortado en otra celda del diseño.
Copiar componente Copia la ventana o el componente seleccionado en el portapapeles. Puede pegar la ventana o el componente que haya seleccionado del portapapeles a otra celda del diseño.
Pegar componente Pega una ventana o un componente cortado o copiado a otra celda del diseño.
Suprimir componente Suprime la ventana o el componente seleccionado. No coloca ninguna copia en el portapapeles y no se puede deshacer.
Editar atributos Abre el editor de atributos mostrando los atributos del componente o la ventana que se ha seleccionado. Véase la ayuda del editor de atributos para obtener más detalles sobre la forma de modificar los atributos de un componente. Este comando también tiene un botón en la barra de herramientas principal.
Añadir columna Añade una columna a la izquierda o a la derecha de la celda o componente seleccionado.
Añadir fila Añade una fila encima o debajo de la celda o componente seleccionado.
Añadir ventana nueva Abra el cuadro de diálogo Ventana nueva. Seleccione un contenedor específico de la lista y pulse en "Crear" para abrir un nuevo diseño de gridbag para el contenedor dado. Este comando tiene un botón correspondiente a la izquierda de la paleta de componentes.
Definir tipo de ventana Abre el cuadro de diálogo Contenedor principal. Utilice este cuadro de diálogo para cambiar el tipo de contenedor principal de un marco o panel.
Anidar panel Anida (en su lugar) el componente del panel del diseño seleccionado dentro de un componente de panel de diseño GridBag.
Personalizar paleta Abre el editor de paleta. Utilice el editor de paleta para personalizar, suprimir y cambiar el diseño de los componentes en las carpetas de paleta.
Importar componentes a paleta Abre el cuadro de diálogo "Importar componentes a paleta". Utilice este cuadro de diálogo para importar JavaBeans o componentes con clases shadow. Véase Importación de componentes para obtener más detalles.
Reiniciar paleta de fábrica Devuelve el contenido de la paleta del constructor de GUI estándar (las pestañas "AWT", "Paneles" y "Adicional") a su estado original.
Información de Bean en componente seleccionado Abre el cuadro de diálogo "Información de Bean" que muestra información específica del bean, incluidos propiedades, eventos y métodos.
Actualizar clases de componentes en diseño Abre el cuadro de diálogo Actualizar clases de componentes en diseño, que muestra una lista de clases de beans. Seleccione una o más clase para su actualización. El componente de la paleta se actualiza, pero no el componente de la ventana de diseño. Si es necesario, aparece el cuadro de diálogo "Recargar componente en diseño".
Informe de uso de clases Muestra una lista de clases de componentes utilizadas en el proyecto actual.
