Ejercicio 2: Actualización de la página de ejecución

En este ejercicio, ejecutará el archivo HTML que acaba de crear, en lugar del archivo predeterminado Blink.tmp.html.
  1. Consulte en el administrador de proyectos el nombre del proyecto actual.
    Si el proyecto actual no es Blink, pulse dos veces en el nombre del proyecto Blink.
  2. Seleccione "Proyecto -> Editar".
    Aparecerá la ventana "Editar proyecto".
  3. Pulse en la pestaña "Ejecutar".
  4. Pulse en "Sí" en "URL existente de página de ejecución".
    Se atenuarán todos los campos de la ventana excepto el campo "URL existente de página de ejecución". Cuando "URL existente de página de ejecución" esté definido como "Sí", WorkShop ignorará la información de la pestaña "Ejecutar" cuando se ejecute el proyecto.
  5. En el cuadro "URL existente de página de ejecución", escriba el nombre que acaba de dar a su archivo HTML actualizado.
    No será necesario escribir el nombre de ruta completo. Java Development Environment irá al directorio de origen para obtener el archivo.
  6. Pulse en "Aceptar".
  7. Pulse en el botón "Ejecutar" en la barra de herramientas principal.
    Java Development Environment cargará el nuevo archivo HTML en el examinador. Verá el proyecto Blink ejecutándose a mayor velocidad.
