Ejercicio 3: Búsqueda de cadenas en el código

En este ejercicio aprenderá a buscar todas las ocurrencias de una cadena especificada en el código fuente. Luego visualizará las coincidencias con el código fuente de alrededor.
  1. Compruebe el nombre del proyecto actual en la ventana del administrador de proyectos.
    Si el proyecto actual no es Blink, pulse dos veces en el nombre del proyecto Blink.
  2. Pulse en el botón "Buscar" en la barra de herramientas principal de Java Development Environment.
  3. Escriba green en el cuadro "Cadena".
  4. Pulse en "Buscar".
    Una coincidencia es cualquier ocurrencia de la cadena en el código fuente. El buscador de cadenas encuentra dos coincidencias con la cadena green y las muestra en la pestaña "Cadena de búsqueda" debajo de la ventana del editor de código fuente. Las coincidencias aparecen con el formato: nombre_archivo: número_línea. También verá la línea de código fuente que contiene la cadena.
  5. Pulse en el enlace de la primera coincidencia, Blink.java:32.
    El editor de código fuente muestra la línea de código fuente que contiene la cadena. La línea aparece resaltada en azul.
  6. Seleccione "Examinar -> Coincidencia siguiente".
    El editor de código fuente resalta el código de la siguiente coincidencia.
