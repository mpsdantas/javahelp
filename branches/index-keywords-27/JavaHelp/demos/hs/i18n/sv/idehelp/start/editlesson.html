Övning 1: Redigera körattribut

Märket för <miniprogrammet> som du genererade i Självstudie 1 anger minimikraven för miniprogrammets placering på en webbsida. I den här övningen kommer du att lägga till parametrar för anpassning av visningen av miniprogrammet.

Använd Blink-projektet för den här övningen. Om du ännu inte har skapat ett Blink-projekt kan du göra det nu, utifrån anvisningarna i Skapa Blink-projektet.

  1. Kontrollera namnet på det aktuella projektet (det finns i projekthanteraren).
    Om det aktuella projektet inte är ett Blink-projekt dubbelklickar du på Blink-projektets namn.
  2. Välj Projekt -> Redigera i projekthanteraren.
    Fönstret Redigera projekt innehåller fyra flikar: Allmän, Bygg, Felsök/Bläddra och Kör.
  3. Klicka på fliken Kör.
    Attributen på den här fliken används för att generera märket för <miniprogrammet> i filen Blink.tmp.html.
  4. Sök efter fältet Parametrar (ungefär i mitten av fönstret).
    För närvarande är inga parametrar definierade för miniprogrammet. En parameter är ett attribut som inte är standard och som anger hur miniprogrammet visas. En parameter har ett namn och ett värde. Nästa uppgift blir att lägga till en parameter som ändrar orden i den blinkande texten.
  5. Skriv lbl (en förkortning för "label", etikett), i textrutan Namn till höger om fältet Parametrar.
  6. Skriv följande text i rutan Värde:
    Java Development Environment är toppen :-)
  7. Klicka på knappen Lägg till.
    Raden lbl=Java Development Environment är toppen :-) visas i fältet Parametrar.
  8. Klicka på OK så sparas ändringen.
  9. Klicka på knappen Kör i verktygsfältet i huvudfönstret.
    Java Development Environment inkluderar raden: 
    <param name="lbl" value="Java Development Environment är toppen :-)">
    i märket för <miniprogrammet> i filen Blink.tmp.html. Webbläsaren öppnar filen och du kan nu se att orden "Java Development Environment är toppen :-)" blinkar i olika färger.

    Du avslutar miniprogrammet genom att välja Miniprogram -> Avsluta i fönstret Miniprogramgranskaren.
