Övning 1: Redigera källkod

För att du ska få en uppfattning om hur det går till att arbeta med en källredigerare, kommer du nu att ange kod i filen Blink.java, som gör så att orden studsar.

Om du ännu inte har skapat ett Blink-projekt kan du göra det nu utifrån anvisningarna i Skapa Blink-projektet.

  1. Kontrollera namnet på det aktuella projektet (det finns i fönstret Java Development Environment Projekthanteraren.
    Om ett annat projekt än Blink-projektet är markerat dubbelklickar du på Blink-projektets namn.
  2. Skriv nedanstående rader med källkod.
    Koden är avsiktligt felaktig så att du i nästa lektion kan lära dig hur du åtgärdar felen.

    Du går till en viss rad genom att välja Redigera -> Gå till rad. Radnumret visas till höger i fottexten.

    Efter rad 17 
    int speed;

    boolean offset;

    Efter rad 25 
    lbl = (att == null) ? "Blink" : att;
    skriv: 
    offset = false;

    g.setColor(Color.lightGray);
}

  3. Klicka på knappen Spara i verktygsfältet för redigering och felsökning.
    Den sparade filens sökväg visas i källredigerarens fottext.
