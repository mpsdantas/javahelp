Övning 2: Uppdatera Körsida

I den här övningen kommer du att köra den HTML-fil du skapade nyss, istället för default-filen Blink.tmp.html.
  1. Kontrollera i Projekthanteraren efter namnet på aktuellt projekt.
    Om aktuellt projekt inte är Blink dubbelklickar du på projektnamnet Blink.
  2. Välj Projekt -> Redigera.
    Fönstret Redigera projekt öppnas.
  3. Klicka på fliken Kör.
  4. Klicka på Ja för befintlig URL-adress för körsida.
    Alla fälten i fönstret är nedtonade, förutom befintlig URL-adress för körsida. När inställningen är Ja i fönstret URL-adress för körsida så ignorerar Java Development Environment informationen i fliken Kör när den kör projektet.
  5. I rutan befintlig URL-adress för körsida skriver du in det nya namnet som du nyss gav din uppdaterade HTML- fil.
    Du behöver inte skriva hela sökvägsnamnet. Java Development Environment går till källkatalogen för att hämta filen.
  6. Klicka på OK.
  7. Klicka på knappen Kör i verktygsfältet.
    Java Development Environment laddar den nya HTML-filen till läsaren. Nu kan du se exekveringen av projektet Blink i en snabbare hastighet.
