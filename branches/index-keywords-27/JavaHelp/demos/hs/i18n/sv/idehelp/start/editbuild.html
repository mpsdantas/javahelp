Övning 2: Redigera Byggattribut

I den här övningen kommer du att anpassa sättet som Java Development Environment bygger upp programmet på.

Detta projekt arbetar med projektet Blink och de tidigare lektionerna i Självstudie 1 och 2. Om du inte har redigerat körningsattributen i tidigare övning måste du göra detta nu via instruktionerna i Redigera körattribut.

  1. Kontrollera i Projekthanteraren efter namnet på aktuellt projekt.
    Om aktuellt projekt inte är Blink dubbelklickar du på projektnamnet Blink.
  2. I Projekthanteraren väljer du Projekt -> Redigera.
    I fönstret Redigera projekt visas fyra flikar: Allmänt, Bygga, Felsök/Bläddra och Kör.
  3. Klicka på fliken Bygga.
    Attributen i denna flik används för att definiera hur du vill bygga upp programmet.
  4. Klicka på Alternativ.
    Dialogrutan Alternativ i Java Compiler visas.
  5. Se till att du väljer Aktivera generering av felsökningstabeller.
  6. Klicka på OK.
  7. Klicka på OK i fönstret Redigera projekt.

Nästa lektion:

Nu är du klar med lektionerna i den här självstudien. Nästa självstudie är Självstudie tre: Reparera fel i källkod.
