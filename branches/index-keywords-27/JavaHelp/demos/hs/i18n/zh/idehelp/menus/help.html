帮助菜单

帮助目录 打开在线帮助的目录表。
基于任务的帮助 显示内含常用任务的帮助标题菜单。
菜单帮助 显示Java Development Environment菜单的帮助主题 列表。
Java Development Environment教程 启动Java Development Environment教程，内含的八个 交互式系列教程可以指导您逐步完成 对一个applet小程序项目的创建、 生成、运行和调试过程。
版本注释 提供关于这个版本的Java Development Environment的 最新信息。
序列号/许可证 显示Java Development Environment的序列号和许可证信息。
登记 打开“Java Development Environment登记”对话框。您可以 利用该对话框，通过电子邮件来登记您的 Java Development Environment复本。
发送意见 打开“Java Development Environment意见箱”。您可以利用“意见箱”， 通过电子邮件把您的意见和建议发送给 Java Development Environment产品组。
Java API文档（1.1） 在帮助查看器中显示1.1 Java API文档。 另外，您还可以通过在线帮助目录表来 访问这些文档。
Java语言规范 显示由James Gosling、Bill Joy和 Guy Steele编写的“Java语言规范” 目录。
关于Java Development Environment 打开“关于框”。“关于框”中包含下列 信息：
  • 序列号
  • 版本号
  • Java Development Environment开发小组的照片
  • 专利权警告
