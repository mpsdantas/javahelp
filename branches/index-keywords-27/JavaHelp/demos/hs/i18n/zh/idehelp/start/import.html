添加个人公文夹

在您第一次运行Java Workshop时，它会创建一个名为“personal”的公文夹，您对该文件夹拥有写权限。 如果您删除了您的个人公文夹，却没有从文件系统中删除公文夹文件（.psf），那么您以后还可以 将其重新引入到Java Development Environment中。

要不然，您可以象在本教程中一样，使用任何拥有写权限的公文夹。或者，利用创建公文夹 中的帮助信息来创建一个新的公文夹。

以下说明将描述如何添加您的个人公文夹：

  1. 在Java Development Environment项目管理器窗口中，选择文件->添加->公文夹。
    将显示“添加公文夹”对话框。
  2. 在“公文夹”文件框中输入/your_installation_directory/jws/personal.psf。
    所有公文夹的文件名都以扩展名.psf来结尾。
  3. 单击“添加”。
    Java Development Environment会让您的个人公文夹成为当前公文夹，其文件名显示为蓝色文本。
