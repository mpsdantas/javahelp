练习1：编辑HTML文件

您将在本练习中学习编辑Java Development Environment在运行Blink项目时所创建的一个HTML文件。您将要指定一个 applet小程序参数，并添加一个到源代码的链接。在编辑好这个文件之后，要用一个不同的名字对其进行保存， 以便使这些更改内容不会在下一次运行该项目时被覆盖掉。

如果您还没有创建和生成Blink项目，就应先按创建Blink项目中的说明去完成这项工作。

  1. 在项目管理器中检查当前项目的名字。
    如果当前项目不是Blink，就双击Blink项目名。
  2. 在源代码编辑器中，选择文件->打开。
    Java Development Environment将打开一个文件选择器。
  3. 打开Blink目录中的Blink.tmp.html文件。
    该文件包含一个applet标签，内容如下： 
    <applet
   name="Blink"
   code="Blink.class"
   codebase="/your_home_directory/jws/Blink"
   width="500"
   height="600"
   align="Top"
   alt="If you had a java-enabled browser, 
   you would see an applet here."
> 
<param name="lbl" value="Java Development Environment is really cool :-)">
<hr>If your browser recognized the applet tag,
      you would see an applet here.<hr>
</applet>

    这个文件是在运行该项目时由Java Development Environment创建的。文件中的内容主要根据“编辑项目”窗口的 “运行”制表项中的信息。如果您以前没运行过这个项目，请参阅运行项目。

  4. 从Java Development Environment菜单栏中选择文件->另存为。
    将显示“保存当前文件”对话框。
  5. 选择Blink目录；然后输入该HTML文件的新文件名并单击“确定”。
    确保已经重命名HTML文件；否则这些更改内容会在下一次运行该项目时被覆盖掉。 另外，一定要将文件保存在Blink目录中。
  6. 在源代码编辑器中，把高度值更改成75。
  7. 在lbl参数之前添加下列参数，以加快单词闪烁的速度： 
    <param name="speed" value="50">
  8. 在源代码的<applet>标签之后添加下列链接： 
    <p>
<a href="Blink.java">Source Code</a>
    源代码的路径必须相对包含HTML文件的目录。
  9. 单击“编辑/调试”工具栏中的“保存”按钮。
