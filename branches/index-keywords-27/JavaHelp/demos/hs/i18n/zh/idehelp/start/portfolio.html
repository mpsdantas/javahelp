练习1：创建公用公文夹

本练习将指导您学习如何创建一个公用公文夹；Internet上的其他用户将可以存取这个公文夹中的项目。
  1. 选择“公文夹” => “创建”。
    这个动作会打开“创建公文夹”对话框。

  2. 在文本域中输入一个HTTP服务器能存取到的公文夹路径名。
    公文夹的名字必须使用扩展名.psf作为结尾。例如，在Solaris计算环境下，要想在目录/smith/public_html/folios中创建公文夹graphics.psf，就要输入/smith/public_html/folios/graphics.psf。

  3. 单击“创建”按钮。
    Java Development Environment会把这个公文夹作为当前公文夹。Internet上的其他WorkShop用户将可以通过在浏览器中输入此公文夹的URL，来查看由您创建的这个公文夹。在下一课中，您将学习如何把项目复制到公文夹中。
