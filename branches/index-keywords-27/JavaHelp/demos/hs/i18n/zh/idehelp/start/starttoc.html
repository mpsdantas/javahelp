Java Development Environment教程

本教程的目的是讲解如何利用Java Development Environment的基本特性来创建、编辑、生成、调试和运行一个项目。

第一部分教程可以帮助您熟悉Java Development Environment所使用的概念、工具和开发过程。要想了解Java Development Environment 的基本特性，请完成教程一至教程五。教程六和七则描述了Java Development Environment的高级特性。

  • Java Development Environment简介 讨论利用Java Development Environment来创建Internet应用程序和web页面的过程。
  • 教程一：创建Blink项目 详细讲解在Java Development Environment中创建、生成和运行一个applet小程序的基本步骤。
  • 教程二：编辑项目属性 引领您完成编辑项目特性的各个步骤。
  • 教程三：为源代码排错 描述如何在生成过程中进行排错，以及如何重新生成项目。
  • 教程四：浏览项目 讲解如何对组成项目的类和方法进行浏览。另外，您还会学到如何在源代码中 查找字符串。
  • 教程五：调试项目 描述如何定位和改正那些造成项目不能正常运行的各种错误。您将学习 用调试程序来检查线程、设置断点和验证表达式。
  • 教程六：定制运行页面 描述如何对Java Development Environment在运行项目时所创建的HTML文件进行编辑。
  • 教程七：创建GUI 描述如何装配项目的用户界面。
  • 教程八：创建时钟项目 描述如何创建程序节项目，以及如何将JavaBeans引入到GUI生成器中。
