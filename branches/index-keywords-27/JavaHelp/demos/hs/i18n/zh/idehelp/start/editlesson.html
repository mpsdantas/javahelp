练习1：编辑运行属性

“教程一”中所生成的<applet>标签指定了在web页面上放置applet小程序所需的最低要求。 在本练习中，您将添加一些能定制applet小程序显示方式的参数。

本练习要使用Blink项目。如果您还没有创建和生成Blink项目，就应先按创建Blink项目中的说明 去完成这项工作。

  1. 在项目管理器中检查当前项目的名字。
    如果当前项目不是Blink，就双击Blink项目名。
  2. 在项目管理器中，选择项目->编辑。
    “编辑项目”窗口中共有四个制表项：通用、生成、调试/浏览和运行。
  3. 单击“运行”制表项。
    该制表项中的属性可用来在Blink.tmp.html文件中产生<applet>标签。
  4. 查找“参数”域（大约在窗口的中下部分）。
    当前，还没有为applet小程序定义任何参数。参数是 一个非标准的属性，用来定义显示applet小程序的方式。每个参数 都有一个名字和值。下面您将添加一个参数来更改出现在闪烁文本 中的单词。
  5. 在“参数”域右边的“名字”文本框中输入lbl（“label”的缩写）。
  6. 在“值”框中输入下列文本：
    Java Development Environment is really cool :-)
  7. 单击“添加”按钮。
    lbl=Java Development Environment is really cool :-)一行就出现在“参数”域中。
  8. 单击“确定”保存所做的更改。
  9. 单击Java Development Environment主工具栏中的“运行”按钮。
    Java Development Environment将下列行： 
    <param name="lbl" value="Java Development Environment is really cool :-)">
    包括在Blink.tmp.html文件的<applet>标签中。该文件将被装入浏览器，您现在可以看到 “Java Development Environment is really cool :-)”一行正以不同的颜色交替闪烁。

    要想终止正在运行的applet小程序，可在“Applet小程序查看器”窗口中选择 Applet小程序->退出。
