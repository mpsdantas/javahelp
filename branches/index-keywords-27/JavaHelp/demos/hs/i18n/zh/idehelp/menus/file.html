文件菜单

新建 将源代码编辑器的内容置空。您可以在其中输入 文本并将其保存成新文件。有关源代码编辑器的详情， 请参阅编辑源代码。
打开 在源代码编辑器中打开一个现有文件。
新的编辑器窗口 启动一个不装入任何文件的新的源代码编辑器窗口。
复制编辑器窗口 打开一个新的编辑器窗口，并装入 被复制编辑器窗口中的同一个文件。 这个过程可以重复进行多次。 有关详情请参阅 基本编辑。
重新装入源文件 在源代码编辑器中重新装入当前已装入的文件。
保存 保存源代码编辑器中当前装入的文件， 并使用当前的文件名。
另存为 保存源代码编辑器中当前装入的文件， 并使用由文件选择器对话框提供的名字。
保存所有源文件 保存在激活源代码编辑器窗口中当前装入的 所有文件。
版本 文件版本控制命令。有关详情请参阅 版本控制。
文件选取表 Java Development Environment会维护一个已装入源代码编辑器 的文件列表。每当在编辑器中装入一个文件 时，其文件名就被添加到列表的最上面。从该列表 中选择一个文件即可将其装入源代码编辑器中。

该列表包括最近编辑的20个文件，并按它们 装入编辑器的顺序进行排列。整个Java Development Environment 会话过程都会维护这个列表。如果一个文件 已从文件系统中删除，它也会从这个列表中删除。

退出 退出Java Development Environment。
