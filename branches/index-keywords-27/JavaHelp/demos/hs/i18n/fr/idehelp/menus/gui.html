Menu Interface

Afficher le générateur d'interface Ajoute les commandes du générateur d'interface à la fenêtre Java Development Environment. Cette option est active seulement pour les projets créés à l'aide du générateur d'interface Visual Java.
Fusionner le fichier d'interface Ouvre un sélecteur de fichier et importe le fichier sélectionné (celui-ci contient une fenêtre du générateur d'interface que vous souhaitez inclure à votre application). Ce fichier doit avoir l'extension .gui.
Enregistrer & Générer le fichier d'interface Enregistre votre interface dans un fichier nomprojet.gui et génère le code source Java à partir de votre composition. Voir Générer le code source Java pour plus de détails.
Couper un composant Supprime la fenêtre ou le composant sélectionné et le place dans le presse-papiers. Vous pouvez coller l'élément coupé dans une autre cellule de votre composition.
Copier un composant Copie la fenêtre ou le composant sélectionné dans le presse-papiers. Vous pouvez coller l'élément copié dans une autre cellule de votre composition.
Coller un composant Colle la fenêtre ou le composant coupé ou copié dans une autre cellule de la composition.
Supprimer un composant Supprime la fenêtre ou le composant sélectionné. Ne place pas une copie dans le presse-papiers et ne peut donc pas être défait.
Editer les attributs Ouvre l'éditeur d'attributs associé à la fenêtre ou au composant sélectionné. Reportez-vous à l'aide sur l'éditeur d'attributs pour plus d'informations sur la modification des attributs d'un composant. Cette commande est également accessible par un bouton de la barre d'outils principale.
Ajouter une colonne Ajoute une colonne à gauche ou à droite de la cellule ou du composant sélectionné.
Ajouter une ligne Ajoute une ligne en dessous ou en dessus de la cellule ou du composant sélectionné.
Ajouter une nouvelle fenêtre Ouvre la boîte de dialogue Nouvelle fenêtre. Sélectionnez un conteneur dans la liste et cliquez sur Créer pour ouvrir un nouveau gridbag pour le conteneur donné. Cette commande est également accessible par un bouton situé à gauche de la palette de composants.
Définir le type de fenêtre Ouvre la boîte de dialogue Conteneur principal, qui permet de changer le type d'un conteneur principal en cadre ou panneau.
Imbriquer panneau Imbrique (en place) le composant panneau sélectionné dans un composant panneau d'affichage GridBag.
Personnaliser la palette Ouvre l'éditeur de palette. Utilisez l'éditeur de palette pour personnaliser, supprimer et réorganiser les composants dans des dossiers palette.
Importer des composants dans la palette Ouvre la boîte de dialogue Importation des composants dans la palette. Cette boîte permet d'importer des JavaBeans ou des composants des classes shadow. Voir Importation de composants pour plus de détails.
Réinitialiser à la palette d'origine Rétablit le contenu de la palette du générateur d'interface standard (onglets AWT, Panneaux et Supplémentaire) dans son état original.
Information Bean sur le composant sélectionné Ouvre la boîte de dialogue Informations sur les beans, qui affiche les spécificités des beans telles que propriétés, événements et méthodes.
Mettre à jour les classes de composants en conception Ouvre la boîte de dialogue Mise à jour des classes de composants en conception qui affiche la liste des classes bean. Sélectionnez une ou plusieurs classes à mettre à jour. Le composant est mis à jour dans la palette mais pas dans la fenêtre de composition. Une boîte de dialogue Rechargement des composants en conception apparaît si le rechargement est nécessaire.
Rapport d'utilisation des classes Affiche la liste des classes composant utilisées dans le projet en cours.
