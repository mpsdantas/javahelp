Exercice 2 : Génération du projet

Cette exercice explique comment générer le projet Blink. Vous générez un projet de manière à pouvoir l'exécuter. Quand vous générez un projet, Java Development Environment compile le code source Java de celui-ci dans un fichier nommé nomdefichier.class.

Si vous n'avez pas encore créé le projet Blink, faites-le maintenant en suivant les instructions données dans Création du projet Blink.

  1. Vérifiez dans la fenêtre Gestionnaire de projet de Java Development Environment le nom du projet en cours.
    Si celui-ci n'est pas Blink, double-cliquez sur le nom de projet Blink.
  2. Cliquez sur le bouton Générer de la barre d'outils principale de Java WorkShop.
    Java Development Environment compile le programme Blink.javadans le fichier de code octet Blink.class. Observez l'onglet Générer (sous l'éditeur de source) tandis que Java Development Environment génère votre projet. Dès que vos sources seront générés et compilés, il affichera le message d'état "Effectué".
