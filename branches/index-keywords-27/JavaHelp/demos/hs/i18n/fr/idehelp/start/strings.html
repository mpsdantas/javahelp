Exercice 3 : Recherche de chaînes dans votre code

Dans cet exercice, vous apprendrez à trouver toutes les occurrences d'une chaîne spécifiée dans votre code source. Vous pourrez alors visualiser ces occurrences dans le contexte de leur code source.
  1. Vérifiez dans le gestionnaire de projet le nom du projet en cours.
    Si celui-ci n'est pas Blink, double-cliquez sur le nom de projet Blink.
  2. Cliquez sur le bouton Chercher de la barre d'outils principale de Java WorkShop.
  3. Saisissez green dans la zone Chaîne.
  4. Cliquez sur Chercher.
    Les correspondances seront toutes les occurrences de cette chaîne dans votre code source. Le chercheur de chaîne décèle deux correspondances de la chaîne green et les affiche dans l'onglet Chaîne de recherche en dessous de la fenêtre Editeur de source. Elles sont restituées sous la forme nomfichier: numéro_ligne. Le code source qui contient la chaîne est lui aussi affiché.
  5. Cliquez sur le lien de la première correspondance, Blink.java:32.
    L'éditeur de source affiche la ligne source où figure la chaîne. La ligne est surlignée en bleu.
  6. Sélectionnez Parcourir -> Correspondance suivante.
    L'éditeur de source surligne le code contenant l'occurrence suivante.
