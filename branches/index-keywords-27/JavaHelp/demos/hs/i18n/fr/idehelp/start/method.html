Exercice 2 : Visualisation des méthodes de votre projet

Dans cet exercice, vous listerez les méthodes définies dans la classe Blink. A partir de cette liste, vous pourrez afficher le code source de la méthode. Pour réussir cet exercice, vous devez avoir réalisé le précédent, Affichage de la hiérarchie des classes.
  1. Dans l'afficheur de classe, faites défiler jusqu'à l'étiquette Constructor Index (index des constructeurs).
    Un constructeur est une méthode spéciale initialisant un objet lors de sa création. L'afficheur de classe liste un constructeur, Blink, défini dans la classe Blink.
  2. Cliquez sur Blink.
    L'éditeur de source affiche le code source du constructeur Blink. La première ligne de celui-ci apparaît en surbrillance bleue.
  3. Dans l'afficheur de classe, faites défiler jusqu'à l'étiquette Method Index (index des méthodes).
    La classe Blink contient cinq méthodes : init, paint, start, stop et run.
  4. Cliquez sur paint.
    L'éditeur de source affiche le code source de la méthode paint. La première ligne apparaît en surbrillance bleue.
