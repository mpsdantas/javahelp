Outils de Java Development Environment

Les outils de Java Development Environment sont représentés par des boutons sur la barre d'outils principale de Java Development Environment (située sous la barre de menu).

Les boutons de la barre d'outils principale sont les suivants :

Générer
Génère le projet en cours. Si votre projet comporte des erreurs, l'onglet Générer les signale ainsi que les liens avec le code source.
Exécuter
Exécute le projet en cours. S'il s'agit d'une appliquette, un fichier HTML est créé et exécuté dans l'afficheur d'appliquette. S'il s'agit d'un programme autonome, ses résultats s'affichent dans la console. Vous ne pouvez pas exécuter un module Java ou un bean.
Analyse de performance
Lance l'analyseur de performance. Emet un rapport indiquant combien de temps a été consacré à chacune des méthodes du programme et combien de fois chaque méthode a été exécutée.
Chercher
Trouve les références à des chaînes de texte dans les fichiers source spécifiés.
Projets
Génère, importe, édite et supprime des projets.

Les projets sont la pierre angulaire de Java Development Environment. Un projet assure le suivi des divers fichiers requis pour générer, parcourir, exécuter et déboguer un projet dans Java WorkShop. Les projets sont regroupés en portfolios. Java Development Environment gère les types de projet suivants :

  • Appliquette. Programme écrit dans le langage Java et s'exécutant dans un explorateur compatible Java, tel que HotJava.
  • Programme autonome. Programme écrit dans le langage Java et s'exécutant à l'extérieur d'un navigateur Web, généralement à partir d'une fenêtre de commande MS-DOS ou d'un utilitaire Shell sous l'environnement d'exploitation UNIX.
  • Module Java. Groupe de classes Java déclarées avec le même nom de module.
  • Java beans. Composants logiciels réutilisables pouvant être manipulés dans le générateur GUI.
Attributs
Permet d'éditer les propriétés des composants de l'interface des projets du générateur GUI.
Aide
Fournit des informations sur Java Development Environment. Vous pouvez obtenir des informations de différentes manières :
  • en cliquant sur le bouton Aide (représenté par le point d'interrogation) de la barre d'outils principale de Java Development Environment, pour obtenir des informations sur les divers boutons de cette barre d'outils ;
  • en cliquant sur le bouton Aide (représenté par le point d'interrogation) de la barre d'outils Editer/Déboguer, pour obtenir des informations sur les divers boutons de celle-ci ;
  • dans le menu Aide, en sélectionnant la rubrique voulue ;
  • en cliquant sur le bouton Aide dans une boîte de dialogue ou un onglet.

Si vous voulez vérifier l'action réalisée par un bouton, vous pouvez afficher sa description en plaçant le pointeur dessus. Elle apparaîtra au bas de la page du navigateur.
