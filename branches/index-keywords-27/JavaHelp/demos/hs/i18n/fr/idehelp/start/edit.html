Didacticiel Deux : Edition des attributs du projet

Ce didacticiel vous apprendra ce que sont les attributs d'un projet, et la manière d'éditer les attributs d'exécution et de génération. Ce didacticiel est divisé en trois leçons. Pour tirer le meilleur parti de ce didacticiel, lisez tout d'abord la description des attributs du projet, puis effectuez les deux exercices.
  • Concepts importants : Attributs de projet explique ce que sont les attributs d'un projet et comment ils sont utilisés dans Java Development Environment. Vous devez bien comprendre ces concepts avant de passer à l'exercice 1.
  • Exercice 1 : Edition des attributs d'exécution démontre comment spécifier la manière dont l'appliquette sera exécutée.
  • Exercice 2 : Edition des attributs de génération montre comment spécifier des différences de génération.
