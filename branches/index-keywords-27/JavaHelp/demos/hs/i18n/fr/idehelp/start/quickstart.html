Didacticiel Un : Création d'un projet

Ce didacticiel est conçu pour les nouveaux utilisateurs de Java WorkShop. Il vous apprendra ce qu'est un projet et la manière d'en créer, construire et exécuter un dans Java Development Environment.

Le didacticiel Un est divisé en cinq leçons. Pour tirer le meilleur parti de ce didacticiel, suivez les leçons dans l'ordre de leur présentation.

  • Concepts importants : Projets et portfolios donne une présentation complète du fonctionnement des projets et des portfolios. Vous devez bien comprendre ces concepts avant de continuer à consulter ce didacticiel.
  • Exercice 1 : Création du projet Blink vous montre comment créer un projet contenant une appliquette. Une appliquette (ou applet) est un petit programme s'exécutant dans un explorateur gérant le langage Java.
  • Exercice 2 : Génération de projets vous explique commment construire le projet d'appliquette.
  • Exercice 3 : Exécution du projet vous apprend à exécuter et à visualiser votre projet d'appliquette dans un afficheur.
  • Exercice 4 : Recueil des données de profilage décrit la procédure de recueil des informations relatives au profilage.
