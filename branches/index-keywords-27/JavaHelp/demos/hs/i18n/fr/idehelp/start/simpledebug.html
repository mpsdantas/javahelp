Didacticiel Cinq : Débogage d'un projet

Ce didacticiel vous présente le processus de débogage de Java WorkShop. Vous apprendrez comment examiner les threads d'un programme, fixer les points d'arrêt et étudier les données de votre programme.

Le didacticiel comporte sept leçons. Effectuez les leçons dans l'ordre dans lequel elles sont présentées.

  • Concepts importants : Threads définit brièvement les threads et groupes de threads. Vous devez bien comprendre ces concepts avant d'effectuer les exercices.
  • Exercice 1 : Démarrage d'une session de débogage vous enseigne comment lancer le débogage d'un projet.
  • Exercice 2 : Définition d'un point d'arrêt vous explique comment suspendre l'exécution d'un programme à un endroit précis du code.
  • Exercice 3 : Examen des threads vous indique comment visualiser les groupes de threads, les threads et les cadres de pile constituant un projet.
  • Exercice 4 : Progression pas à pas dans une méthode vous montre comment progresser pas à pas dans le code.
  • Exercice 5 : Evaluation des expressions décrit comment vérifier la valeur des données de votre programme.
  • Exercice 6 : Fin d'une session de débogage vous explique comment terminer une session de débogage.
