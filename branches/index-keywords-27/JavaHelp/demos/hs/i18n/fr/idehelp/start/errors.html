Didacticiel Trois : Correction d'erreurs dans le code source

Les leçons de ce didacticiel vous apprendront comment éditer le code source du projet Blink et regénérer celui-ci. Pour tirer le meilleur parti de ce didacticiel, suivez les leçons dans l'ordre de leur présentation.
  • Exercice 1 : Edition du code source vous présente certaines fonctions de base de l'éditeur de source Java Development Environment.
  • Exercice 2 : Correction d'erreurs dans un programme généré corrige les erreurs de compilation apparues lors d'une génération.
