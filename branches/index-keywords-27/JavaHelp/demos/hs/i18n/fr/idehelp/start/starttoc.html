Didacticiel de Java Workshop

L'objectif des didacticiels est de vous présenter les fonctionnalités essentielles de Java Development Environment qui vous permettront de créer, éditer, construire, déboguer et exécuter un projet.

La première section vous aidera à vous familiariser avec les concepts, outils, et processus de développement utilisés dans Java Development Environment. Vous découvrirez les fonctionnalités élémentaires de Java Development Environment au cours des cinq premiers didacticiels. Les didacticiels sept et huit sont consacrés aux fonctions avancées de Java Development Environment.

  • Présentation de Java Development Environment traite du processus d'utilisation de Java Development Environment pour créer des applications Internet et des pages web.
  • Didacticiel Un : Création du projet Blink vous présente les fonctions élémentaires de création, génération et exécution d'une appliquette dans Java Development Environment.
  • Didacticiel Deux : Edition des attributs du projet vous guide dans l'édition des caractéristiques d'un projet.
  • Didacticiel Trois : Correction des erreurs dans le code source décrit comment corriger les erreurs au cours de la génération et comment régénérer le projet.
  • Didacticiel Quatre : Exploration d'un projet vous explique comment parcourir les classes et les méthodes qui composent votre projet. Vous apprendrez également comment rechercher des chaînes dans votre code source.
  • Didacticiel Cinq : Débogage d'un projet vous explique comment localiser et corriger les bogues qui empêchent votre programme de fonctionner correctement. Vous utiliserez le débogueur pour inspecter vos threads, déterminer des points d'arrêt et examiner des expressions.
  • Didacticiel Six: Personnalisation d'une page d'exécution décrit comment éditer le fichier HTML créé pour vous par Java Development Environment quand vous exécutez un projet.
  • Didacticiel Sept : Création d'une interface graphique utilisateur explique l'assemblage d'une interface utilisateur d'un projet.
  • Didacticiel Huit : Création du projet Clock (Horloge) décrit la création d'un projet bean et l'importation de JavaBeans dans le générateur d'interface.
