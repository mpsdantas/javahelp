Übung 1: Bearbeiten von Ausführ-Attributen

In der <applet>-Markierung, die Sie in Kapitel 1 des Trainingshandbuchs erstellt haben, sind nur die absolut notwendigen Informationen angegeben, die erforderlich sind, um ein Applet auf einer Web-Seite einzubinden. In dieser Übung werden Sie weitere Parameter hinzufügen, mit denen gesteuert werden kann, auf welche Weise das Applet angezeigt wird.

In dieser Übung werden Sie mit dem Blink-Projekt arbeiten. Wenn Sie das Blink-Projekt noch nicht erstellt haben, so holen Sie dies jetzt nach. Verfahren Sie dabei nach den Anweisungen in Erstellen des Blink-Projekts.

  1. Überprüfen Sie den Projekt-Manager für den Namen des aktuellen Projekts.
    Wenn das aktuelle Projekt nicht "Blink" ist, doppelklicken Sie auf den Projektnamen "Blink".
  2. Klicken Sie im Projekt-Manager auf "Projekt" -> "Bearbeiten".
    Im Fenster "Projekt bearbeiten" werden vier Registerkarten angezeigt: "Allgemein", "Erstellen", "Fehlersuche/Durchsuchen" und "Ausführen".
  3. Klicken Sie auf die Registerkarte "Ausführen".
    Mit den auf dieser Registerkarte enthaltenen Attributen wird die <applet>-Markierung in der Datei Blink.tmp.html generiert.
  4. Suchen Sie das Feld "Parameter" (etwa in der Mitte des Fensters).
    Derzeit sind für dieses Applet keine Parameter festgelegt. Bei einem Parameter handelt es sich um ein nicht standardisiertes Attribut, mit dem die Art und Weise festgelegt wird, wie das Applet angezeigt wird. Ein Parameter hat einen Namen und einen Wert. Durch das Hinzufügen eines Parameters werden Sie den Wortlaut des blinkend angezeigten Textes ändern.
  5. Geben Sie lbl (Abkürzung für "label" ("Bezeichnung")) in das Feld "Name" rechts vom Feld "Parameter" ein.
  6. Geben Sie den folgenden Text in das Feld "Wert" ein:
    Java Development Environment ist echt spitze :-)
  7. Klicken Sie auf "Hinzufügen".
    Im Feld "Parameter" wird die Zeile lbl=Java Development Environment ist echt spitze :-) angezeigt.
  8. Klicken Sie auf "OK", um die von Ihnen vorgenommenen Änderungen zu speichern.
  9. Klicken Sie auf die Schaltfläche "Ausführen" auf der Hauptsymbolleiste von Java Development Environment.
    Java Development Environment fügt die Zeile 
    <param name="lbl" value="Java Development Environment ist echt spitze :-)">
    in die <applet>-Markierung in der Datei Blink.tmp.html ein. Die Datei wird in den Browser geladen, und Sie sehen, wie die Worte "Java Development Environment ist echt spitze :-)" in verschiedenen Farben aufblinken.

    Um das laufende Projekt anzuhalten, wählen Sie "Applet" -> "Beenden" im Fenster "Applet Viewer".
