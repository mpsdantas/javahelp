Erstellen
Erstellt das aktuelle Projekt auf der Grundlage der in der Projektdefinition angegebenen Informationen. Für diesen Befehl steht auf der Hauptsymbolleiste auch eine Schaltfläche zur Verfügung.
Alle erstellen Erzwingt das Erstellen eines Projekts, auch wenn die Dateien aktuell sind.
Datei kompilieren Kompiliert die aktuell im Hauptfenster des Quelleditors geladene Datei. Weitere Informationen finden Sie unter Kompilieren einer einzelnen Datei.
Erstellen stoppen Hält die aktuell laufende Erstellung an.
Nächster Fehler Rollt das Bearbeitungsfenster zu der Zeile mit dem nächsten Erstellungsfehler.
Vorheriger Fehler Rollt das Bearbeitungsfenster zu der Zeile mit dem vorherigen Erstellungsfehler.
