Trainingshandbuch, Kapitel 1: Erstellen eines Projekts

Dieses Trainingshandbuch wurde für Einsteiger in Java Development Environment entworfen. In diesem Trainingshandbuch werden Sie lernen, was ein Projekt ist und wie ein Projekt in Java Development Environment erstellt, aufgebaut und ausgeführt wird.

Das Trainingshandbuch, Kapitel 1, ist in 5 Lektionen unterteilt. Um möglichst viel anhand dieses Trainingshandbuchs zu lernen, sollten Sie die Lektionen in der aufgeführten Reihenfolge durchführen.

  • Wichtige Konzepte: Projekte und Portfolios gibt einen vollständigen Überblick darüber, wie Projekte und Portfolios arbeiten. Sie sollten diese Konzepte verstehen, bevor Sie mit dem Trainingshandbuch fortfahren.
  • Übung 1: Erstellen eines Blink-Projekts zeigt Ihnen, wie ein Projekt, das ein Applet enthält, erstellt wird. Ein Applet ist ein kleines Programm, das in einem Browser, der eine Java-Sprache unterstützt, ausgeführt wird.
  • Übung 2: Aufbauen des Projekts zeigt Ihnen, wie ein Applet-Projekt aufgebaut wird.
  • Übung 3: Ausführen des Projekts zeigt Ihnen, wie ein Applet-Projekt ausgeführt wird, und es wird im Applet Viewer angezeigt.
  • Übung 4: Sammeln von Profildaten zeigt Ihnen, wie Sie Informationen zur Profilerstellung sammeln.
