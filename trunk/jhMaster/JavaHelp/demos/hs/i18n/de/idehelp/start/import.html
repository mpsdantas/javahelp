Hinzufügen eines persönlichen Portfolios

Wenn Sie zum ersten Mal Java Development Environment ausführen, wird ein Portfolio mit der Bezeichnung "persönlich" erstellt, für welches Sie Schreibberechtigung erhalten. Wenn Sie Ihr persönliches Portfolio löschen, können Sie dieses Portfolio wieder in Java Development Environment importieren, solange Sie die Portfolio (.psf)-Datei nicht aus Ihrem Dateisystem löschen.

Ansonsten können Sie im Zusammenhang mit den Übungen dieses Trainingshandbuchs jedes Portfolio verwenden, für welches Sie Schreibrechte haben. Mit Hilfe der im Abschnitt Erstellen eines Portfolios enthaltenen Informationen können Sie aber auch ein zusätzliches Portfolio erstellen.

Anhand der folgenden Anweisungen können Sie Ihr persönliches Portfolio hinzufügen:

  1. Wählen Sie im Fenster Java Development Environment Projekt-Manager "Datei" -> "Hinzufügen" -> "Portfolio".
    Das Dialogfeld "Portfolio hinzufügen" wird angezeigt.
  2. Geben Sie in das Feld "Portfolio-Datei" /InstallationsVerzeichnis/jws/personal.psf ein.
    Sämtliche Portfolio-Dateien haben die Erweiterung .psf.
  3. Klicken Sie auf "Hinzufügen".
    Java Development Environment trägt Ihr persönliches Portfolio als aktuelles Portfolio ein, und stellt den Namen in blauer Schrift dar.
