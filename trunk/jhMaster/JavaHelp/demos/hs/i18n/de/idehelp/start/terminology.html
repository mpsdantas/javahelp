Java Development Environment-Terminologie

Die nachfolgend erklärten Ausdrücke werden in der gesamten Online-Hilfe verwendet. Die Kenntnis dieser Ausdrücke erleichtert Ihnen die Bearbeitung der Übungen des Trainingshandbuchs. Weitere für Java WorkShop charakteristische Ausdrücke (wie beispielsweise Applet und Thread) werden bei ihrem ersten Vorkommen im Trainingshandbuch erläutert.

Achten Sie beim Durcharbeiten des Trainingshandbuchs auf das animierte Sun-Symbol in der oberen rechten Ecke des Browsers. Der Würfel dreht sich, wenn eine Hilfeseite durch Java Development Environment geladen wird.

Schaltfläche
Ein Element in einer Symbolleiste oder einem Fenster. Durch Klicken auf dieses Element wird eine Aktion ausgeführt. Wenn Sie beispielsweise auf die Schaltfläche "Projekte" klicken, wird das Fenster "Java Development Environment Projekt-Manager" angezeigt.

Für die Schaltflächen der Symbolleiste werden Quickinfos bereitgestellt. Wenn Sie nicht genau wissen, welche Aktion durch Klicken auf eine Schaltfläche ausgeführt wird, positionieren Sie den Cursor über der Schaltfläche, um eine Beschreibung der Schaltfläche anzuzeigen. Wenn Sie den Cursor beispielsweise über der Schaltfläche positionieren, wird "Zeichenfolge suchen" angezeigt.

Menü
Eine Liste von Elementen, aus denen eine bestimmte Aufgabe oder ein bestimmtes Objekt ausgewählt werden kann. Die Menüs werden im Browser zwischen der Titelleiste und der Hauptsymbolleiste von Java Development Environment angezeigt. Auf ein Element in einem Menü wird häufig in folgender Form verwiesen: "Menü" -> "Menüelement". Beispielsweise muß bei Angabe von "Projekt" -> "Erstellen" im Menü "Projekt" der Befehl "Erstellen" ausgewählt werden.
Registerkarte
Eine Seite eines Registers (ähnlich einem Dialogfeld), die durch Klicken auf eine Registerkarte in einem Fenster angezeigt wird. Das Fenster "Projekt bearbeiten" enthält beispielsweise folgende Registerkarten: "Allgemein", "Erstellen", "Fehlersuche/Durchsuchen" und "Ausführen".
