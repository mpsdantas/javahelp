Übung 1: Starten einer Fehlersuche-Sitzung

In dieser Übung wird Ihnen gezeigt, wie Sie die Fehlersuche im Blink-Projekt starten. Wenn Sie das Blink-Projekt noch nicht erstellt und aufgebaut haben, sollten Sie es zuerst nach den Anweisungen unter Erstellen des Blink-Projekts erstellen.
  1. Überprüfen Sie den Projekt-Manager für den Namen des aktuellen Projekts.
    Die Datei Blink.java sollte im Quelleditor angezeigt werden. Wenn das aktuelle Projekt nicht "Blink" ist, doppelklicken Sie auf den Projektnamen "Blink".
  2. Klicken Sie auf die Schaltfläche "Start/Neustart" auf der Symbolleiste "Bearbeiten/Debug".
    Die Ausführung von Blink.java beginnt. Java Development Environment führt das Blink-Projekt im Fenster "Applet Viewer" aus.

Während Blink.java ausgeführt wird, können Sie einen Haltepunkt festlegen, indem Sie wählen "Fehlersuche" -> "Haltepunkte". Sie können das Programm auch anhalten, indem Sie wählen "Fehlersuche" ->"Programm unterbrechen".
