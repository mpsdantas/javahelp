Übung 6: Beenden einer Fehlersuche-Sitzung

In dieser Übung lernen Sie, wie Sie eine Fehlersuche-Sitzung beenden.
  1. Wählen Sie "Fehlersuche" -> "Fehlersuche beenden".
    Java Development Environment beendet die Fehlersuche, und das Fenster des Applet-Viewers wird geschlossen.
