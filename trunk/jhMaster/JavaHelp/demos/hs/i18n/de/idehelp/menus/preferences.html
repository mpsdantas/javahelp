Menü Voreinstellungen

Proxy Verwenden Sie das Dialogfeld Proxy, um Proxy-Server (Proxies) zu ändern.
Versionskontrolle Wählt ein Versionskontrollsystem, um es mit Java Development Environment zu verwenden.
Quelleditor Legt Voreinstellungen für folgende Quelleditorverhalten fest:
  • Einrücken/Tabulatoren
  • Syntax-Hervorhebung und Farben
  • Auswahl der Schriftart
Tastatur Anzeigen, Hinzufügen oder Löschen von Tastaturzuordnungen (Verknüpfungen) für Funktionsgruppen in Java WorkShop. Weitere Informationen finden Sie unter Voreinstellungen-Tastatur.
Konsoleneingabe Bestimmt, wie befehlszeilenbasierte Programmeingaben im Fenster "Konsole" interpretiert werden. Weitere Informationen dazu finden Sie unter Dialogfeld Konsoleneingabe.
Builder Bestimmt, welcher Java-Compiler von Java Development Environment normalerweise das Projekt kompiliert. Weitere Informationen dazu finden Sie unter Dialogfeld Voreinstellungen-Builder.
Bezeichnungen in Symbolleiste anzeigen Fügt die Bezeichnungen unter den Schaltflächen der Symbolleisten hinzu.
