Übung 3: Ausführen eines Projekts

In dieser Übung werden Sie das Blink-Projekt mit Hilfe der von Java Development Environment erstellten HTML-Datei ausführen.
  1. Überprüfen Sie den Projekt-Manager für den Namen des aktuellen Projekts.
    Wenn das aktuelle Projekt nicht "Blink" ist, doppelklicken Sie auf den Projektnamen "Blink".
  2. Klicken Sie auf die Schaltfläche "Ausführen" auf der Hauptsymbolleiste von Java Development Environment.
    Java Development Environment generiert die HTML-Datei in Ihrem Blink-Verzeichnis. Diese HTML-Datei enthält die Markierung <applet>, durch welche die beim Einbinden des Projekts "Blink" in eine Web-Seite zu verwendenden Optionen angegeben werden. Java Development Environment lädt die Datei in den Applet Viewer, und Sie können sehen, wie das Blink-Projekt ausgeführt wird. Das Wort "Blink" blinkt in verschiedenen Farben.
  3. Um das laufende Projekt anzuhalten, wählen Sie "Applet" -> "Beenden". im Fenster "Applet Viewer".
