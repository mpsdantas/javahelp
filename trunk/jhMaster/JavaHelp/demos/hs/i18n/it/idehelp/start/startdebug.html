Esercizio 1: Avvio di una sessione di debug

In questo esercizio verrà illustrato come avviare il debug del progetto Blink. Se il progetto Blink non è stato ancora creato, effettuare tale operazione seguendo le istruzioni contenute in Creazione del progetto Blink.
  1. Individuare nel gestore progetti il nome del progetto corrente.
    Il file Blink.java dovrebbe essere visualizzato nell'editor del codice sorgente. Se il progetto corrente non è Blink, fare doppio clic sul nome del progetto Blink.
  2. Fare clic sul pulsante Riavvia sulla barra degli strumenti dell'editor/debug.
    Verrà avviata l'esecuzione di Blink.java e Java Development Environment eseguirà il progetto Blink nella finestra del visualizzatore applet.

Mentre il file Blink.java è in esecuzione, è possibile impostare un breakpoint scegliendo Debug -> Breakpoint. È inoltre possibile sospendere il programma scegliendo Debug -> Interrompi programma.

Lezione successiva
