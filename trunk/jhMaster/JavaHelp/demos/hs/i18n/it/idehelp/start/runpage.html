Esercizio 2: Aggiornamento della pagina di esecuzione

In questo esercizio verrà eseguito il nuovo file HTML invece del file predefinito Blink.tmp.html.
  1. Individuare nel gestore progetti il nome del progetto corrente.
    Se il progetto corrente non è Blink, fare doppio clic sul nome di progetto Blink.
  2. Scegliere Progetto -> Modifica.
    Verrà visualizzata la finestra Modifica progetto.
  3. Fare clic sulla scheda Esegui.
  4. Impostare su Sì l'opzione URL per pagina di esecuzione esistente.
    Tutti i campi della finestra saranno ombreggiati, tranne URL per pagina di esecuzione esistente. Se l'opzione URL per pagina di esecuzione esistente è impostata su Sì, durante l'esecuzione del progetto verranno ignorate le impostazioni della scheda Esegui.
  5. Nella casella URL per pagina di esecuzione esistente immettere il nuovo nome assegnato al file HTML aggiornato.
    Non è necessario immettere l'intero percorso. Il file verrà ricercato nella directory principale.
  6. Fare clic su OK.
  7. Fare clic sul pulsante Esegui sulla barra degli strumenti principale.
    Il nuovo file HTML verrà caricato nel browser. Il progetto Blink verrà eseguito più rapidamente.
