Esercitazione 5: Debug di un progetto

Questa esercitazione rappresenta un'introduzione alle operazioni di debug effettuabili con Java Development Environment. Nell'esercitazione verranno esaminati i thread di un programma, saranno impostati i breakpoint e verranno esaminati i dati del programma.

L'esercitazione è composta da sette lezioni. Per ottenere migliori risultati, è opportuno eseguire le lezioni nell'ordine di presentazione.

  • In Concetti importanti: Thread sono contenute informazioni utili sui thread e sui gruppi di thread. Prima di proseguire nell'esercitazione, è opportuno avere dimestichezza con questi concetti.
  • Nell'Esercizio 1: Avvio di una sessione di debug viene spiegato come eseguire il debug di un progetto.
  • Nell'Esercizio 2: Impostazione di un breakpoint viene spiegato come interrompere l'esecuzione di un programma in un punto specifico del codice.
  • Nell'Esercizio 3: Controllo dei thread viene spiegato come visualizzare i gruppi di thread, i thread e i frame dello stack che compongono il progetto.
  • Nell'Esercizio 4: Esecuzione di un metodo riga per riga viene spiegato come eseguire il codice riga per riga.
  • Nell'Esercizio 5: Valutazione delle espressioni viene spiegato come controllare il valore dei dati del proprio programma.
  • Nell'Esercizio 6: Chiusura della sessione di debug viene spiegato come terminare la sessione di debug.
