Esercitazione 6: Personalizzazione di una pagina di esecuzione

In questa esercitazione verrà illustrato come modificare il file HTML per l'esecuzione di un progetto. Per ottenere i massimi risultati, è necessario eseguire gli esercizi nell'ordine nel quale sono presentati.
  • Nell'Esercizio 1: Modifica del file HTML viene illustrato come modificare il file HTML creato automaticamente al momento dell'esecuzione di un progetto.
  • Nell'Esercizio 2: Aggiornamento della pagina di esecuzione viene illustrato come eseguire il file HTML modificato anziché quello predefinito Blink.tmp.html.
