Esercizio 3: Ricerca di stringhe nel codice

In questo esercizio verrà spiegato come ricercare nel codice sorgente tutte le occorrenze di una stringa specificata. Sarà quindi possibile visualizzare le corrispondenze inserite nel contesto del codice sorgente.
  1. Individuare nel gestore progetti il nome del progetto corrente.
    Se il progetto corrente non è Blink, fare doppio clic sul nome di progetto Blink.
  2. Fare clic sul pulsante Cerca della barra degli strumenti principale di Java Development Environment.
  3. Immettere verde nella casella Stringa.
  4. Fare clic su Cerca.
    Una corrispondenza è una qualsiasi occorrenza della stringa presente nel codice sorgente. Le due corrispondenze della stringa verde individuate dalla funzione di ricerca delle stringhe vengono visualizzate nella scheda Cerca stringhe sotto la finestra dell'editor del codice sorgente. Le corrispondenze vengono visualizzate nel seguente formato: nomefile: numero_riga. Viene visualizzata anche la riga del codice sorgente in cui è riportata la stringa.
  5. Fare clic sul collegamento relativo alla prima corrispondenza Blink.java:32.
    La riga del codice in cui è riportata la stringa viene visualizzata nella finestra dell'editor del codice sorgente. La riga viene evidenziata in blu.
  6. Scegliere Cerca -> Trova successiva.
    Il codice relativo alla corrispondenza successiva risulta evidenziato nell'editor del codice sorgente.

Lezione successiva

Le lezioni di questa esercitazione sono state completate. La prossima esercitazione è l'Esercitazione 5: Debug di un progetto.
