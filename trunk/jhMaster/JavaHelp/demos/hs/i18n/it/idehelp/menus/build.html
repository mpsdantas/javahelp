Compila
Compila il progetto corrente utilizzando le informazioni specificate nella definizione del progetto. Per questo comando è disponibile anche un pulsante di scelta rapida sulla barra degli strumenti principale.
Compila tutto Forza una compilazione del progetto anche se i file sono aggiornati.
Compila file Compila il file attualmente caricato nella finestra principale dell'editor del codice sorgente. Per ulteriori informazioni, vedere Compilazione di un singolo file.
Ferma compilazione Interrompe la compilazione in corso.
Errore successivo Scorre la finestra dell'editor fino alla riga che contiene il successivo errore di compilazione.
Errore precedente Scorre la finestra dell'editor fino alla riga che contiene il precedente errore di compilazione.
