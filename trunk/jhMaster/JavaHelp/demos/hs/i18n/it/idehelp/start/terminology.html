Terminologia di Java Development Environment

Nella Guida vengono utilizzati i termini elencati di seguito. Una volta compreso il significato di questi termini, l'esercitazione risulterà più facile. Gli altri termini specifici di Java Development Environment, quali applet e thread, vengono spiegati la prima volta che compaiono all'interno dell'esercitazione.

A mano a mano che si procede con l'esercitazione, controllare l'icona animata nell'angolo in alto a destra del browser. Il cubo ruota su se stesso quando viene caricata una pagina della Guida.

Pulsante
Un elemento di una barra degli strumenti o di una finestra su cui è possibile fare clic per eseguire un'operazione. Se, ad esempio, si sceglie il pulsante Progetti, verrà visualizzata la finestra del gestore progetti di Java Development Environment.

Per ciascun pulsante della barra degli strumenti è disponibile una descrizione. Se non si è certi dell'operazione che viene eseguita quando si sceglie un pulsante della barra degli strumenti, posizionare il puntatore sul pulsante per visualizzarne una descrizione. Se, ad esempio, si posiziona il puntatore sul pulsante , verrà visualizzata la finestra di dialogo "Cerca stringhe".

Menu
Un elenco di voci da cui è possibile selezionare una determinata procedura o oggetto. I menu vengono visualizzati all'interno del browser tra la barra del titolo e la barra degli strumenti principale di Java Development Environment. Una voce di menu viene in genere riportata nella forma Menu -> Voce. Ad esempio, Progetto -> Compila indica che è necessario scegliere Compila dal menu Progetto.
Scheda
Una pagina simile a una finestra di dialogo che viene visualizzata quando si fa clic sulla linguetta corrispondente. La finestra Modifica progetto contiene, ad esempio, le seguenti schede: Generali, Compila, Debug/cerca ed Esegui.
