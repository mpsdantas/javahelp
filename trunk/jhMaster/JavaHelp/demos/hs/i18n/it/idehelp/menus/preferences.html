Menu Preferenze

Proxy Utilizzare la finestra di dialogo Proxy per modificare i server proxy (proxy).
Controllo versione Selezionare un sistema di controllo della versione da utilizzare con Java Development Environment.
Editor del codice sorgente Consente di impostare le preferenze per le seguenti funzioni dell'editor del codice sorgente:
  • Rientri/tabulazioni
  • Colori ed evidenziazione della sintassi
  • Selezione dei font
Tastiera Consente di visualizzare, aggiungere o eliminare le assegnazioni ai tasti (associazioni) relative ai gruppi funzionali di Java Development Environment. Per ulteriori informazioni, consultare Preferenze per la tastiera.
Input di console Determina la modalità di interpretazione nella finestra della console dell'input dei programmi basati sulla riga di comando. Per ulteriori informazioni, consultare Finestra di dialogo Input di console.
Builder Determina il compilatore Java da utilizzare per compilare i progetti. Per ulteriori informazioni, consultare Finestra di dialogo delle preferenze del builder.
