実習 3: プロジェクトを実行する

この実習では、Java WorkShop によって作成された HTML ファイルを使用して Blink プロジェクトを実行します。
  1. 「プロジェクトマネージャ」ウィンドウで、現在のプロジェクト名を確認します。
    現在のプロジェクトが Blink でない場合は、プロジェクト名「Blink」をダブルクリックします。
  2. メインツールバー上の「実行」ボタン をクリックします。
    HTML ファイルが Blink ディレクトリに生成されます。この HTML ファイルには、Blink プロジェクトを Web ページ上に配置する際に使用するオプションを指定するための <applet> タグが含まれています。Java WorkShop がこの HTML ファイルをアプレットビューアに読み込むと Blink プロジェクトが実行され、Blink という語がさまざまな色で点滅します。
  3. 実行中のアプレットを停止するには、アプレットビューアで「アプレット」->「停止」を選択します。
