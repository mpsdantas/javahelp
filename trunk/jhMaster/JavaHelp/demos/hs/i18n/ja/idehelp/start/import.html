個人用のポートフォリオを追加する

Java Workshop を初めて実行した時に、ユーザーの書き込み権がある personal という名前のポートフォリオが作成されます。personal ポートフォリオを削除した場合でも、ポートフォリオのファイル (.psf) をファイルシステムから削除しないで残しておけば、その personal ポートフォリオを Java WorkShop に取り込んで元に戻すことができます。

このチュートリアルの実習では、書き込み権のあるポートフォリオならどれでも使うことができます。また、ポートフォリオを作成するの手順に沿って、ポートフォリオを新しく作成してもかまいません。

personal ポートフォリオを追加する手順を以下に説明します。

  1. 「プロジェクトマネージャ」ウィンドウで、「ファイル」->「追加」->「ポートフォリオ」を選択します。
    「ポートフォリオ追加」ダイアログが表示されます。
  2. 「ポートフォリオファイル」テキストフィールドに、
    /ホームディレクトリ名/jws/personal.psf と入力します。
    ポートフォリオファイル名にはすべて、拡張子 .psf が付きます。
  3. 「追加」ボタンをクリックします。
    personal ポートフォリオが現在のポートフォリオになり、名前が青色で表示されます。
