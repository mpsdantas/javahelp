「ブラウズ」メニュー

文字列検索
「文字列検索」ダイアログを開きます。指定した文字列を、プロジェクトまたはディレクトリの中で検索することができます。メインツールバーには、このメニュー項目へのショートカットボタンも用意されています。
検索中止 現在の検索を中止します。
次の一致 文字列検索で見つかった、次の一致を表示します。次の一致とは、「文字列検索」タブ付きページにリストされている項目で、現在ソースエディタに内容が表示されている項目のすぐ下にリストされているものです。必要に応じて、新しいファイルがエディタに読み込まれます。
前の一致 文字列検索で見つかった、前の一致を表示します。前の一致とは、「文字列検索」タブ 付きページにリストされている項目で、現在ソースエディタに内容が表示されている項目 のすぐ上にリストされているものです。必要に応じて、新しいファイルがエディタに読み込まれます。
クラス表示 「クラス表示」ウィンドウを開きます。このダイアログで、プロジェクトの中のクラス、メソッド、コメントを表示できます。「クラス表示」ウィンドウには、javadoc で生成された Java API ドキュメントにもとづいた内容が表示されます。
