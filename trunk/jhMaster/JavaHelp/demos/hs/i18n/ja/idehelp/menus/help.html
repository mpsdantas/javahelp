「ヘルプ」メニュー

ヘルプの目次 オンラインヘルプの目次を開きます。
機能別ヘルプ 一般的な機能に関するヘルプ項目を表示します。
メニューのヘルプ Java WorkShop のメニューに関するヘルプ項目のリストを表示します。
Java WorkShop チュートリアル Java WorkShop のチュートリアルを起動します。このチュートリアルは 8 つの対話形式のチュートリアルで構成され、アプレットプロジェクトの作成、構築、実行、デバッグを手順を追って学習することができます。
リリースノート このリリースの Java WorkShop に関する、最新の情報を表示します。
シリアル番号 (ライセンス) Java WorkShop のシリアル番号とライセンスの内容を表示します。
Java API ドキュメント (1.1) ヘルプビューアに、Java API ドキュメント (1.1) を表示します。Java API ドキュメントは、オンラインヘルプの目次からも表示できます。
Java 言語仕様 (英語) James Gosling、Bill Joy、Guy Steele の各氏による「Java Language Specification (Java 言語仕様)」の目次が表示されます。
Java WorkShop について ヘルプビューアに「Java WorkShop について」を表示します。以下の情報が表示されます。
  • シリアル番号
  • バージョン番号
  • 日本語版 Java WorkShop 開発チームの写真
  • 著作権情報
