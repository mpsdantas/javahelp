Exercice 1 : Affichage de la hiérarchie des classes

Dans cet exercice, vous apprendrez comment sont dérivées les classes du projet Blink.
  1. Vérifiez dans le gestionnaire de projet le nom du projet en cours.
    Si celui-ci n'est pas Blink, double-cliquez sur le nom de projet Blink.
  2. Choisissez Parcourir -> Afficher les classes.
    Java Development Environment ouvre l'afficheur de classe, qui liste les classes, méthodes et commentaires composant le projet Blink. Le projet a un fichier source, Blink.java.
  3. Cliquez sur Blink.java.
    L'afficheur de classe charge les classes dont est dérivé Blink.
    Blink est dérivé d'Applet, qui est dérivé de Panel, et ainsi de suite.

    Les informations de l'afficheur de classe sont dérivées de javadoc, le générateur de la documentation API Java. Javadoc analyse la déclaration et les commentaires du code source du projet et formate l'API en code HTML.
