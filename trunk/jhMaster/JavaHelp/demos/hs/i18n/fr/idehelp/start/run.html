Exercice 3 : Exécution du projet

Dans cet exercice, vous exécuterez le projet Blink à l'aide du fichier HTML créé par Java Development Environment.
  1. Vérifiez dans le gestionnaire de projet le nom du projet en cours.
    Si celui-ci n'est pas Blink, double-cliquez sur le nom Blink.
  2. Cliquez sur le bouton Exécuter de la barre d'outils principale de Java WorkShop.
    Java Development Environment génère le fichier HTML dans votre répertoire Blink. Ce fichier comporte le repère d'<appliquette>, qui spécifie les options à utiliser lors de la mise en place du projet Blink dans une page Web. Java Development Environment charge le fichier dans l'afficheur d'appliquette, et vous pouvez assister à l'exécution du projet Blink. Le mot "Blink" clignote en différentes couleurs.
  3. Pour arrêter l'appliquette en cours d'exécution, choisissez Appliquette -> Quitter dans la fenêtre de l'afficheur d'appliquette.
