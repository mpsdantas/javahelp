Didacticiel Quatre : Exploration d'un projet

Ce didacticiel vous apprendra comment explorer un projet. Pour tirer le meilleur parti de ce didacticiel, réalisez les exercices dans l'ordre de leur présentation.
  • Exercice 1 : Affichage de la hiérarchie des classes vous apprend à traverser la hiérarchie des classes de votre projet.
  • Exercice 2 : Visualisation des méthodes de votre projet vous indique comment lister les méthodes définies dans votre projet. A partir de cette liste, vous pourrez afficher le code source de chaque méthode.
  • Exercice 3: Recherche de chaînes dans votre code vous montre comment rechercher dans votre code source toutes les occurrences d'une constante chaîne.
