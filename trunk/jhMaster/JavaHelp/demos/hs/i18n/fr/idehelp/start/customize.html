Didacticiel Six : Personnalisation d'une page d'exécution

Ce didacticiel vous enseigne comment éditer le fichier HTML exécutant le projet. Pour tirer le meilleur parti de ce didacticiel, suivez les leçons dans l'ordre de leur présentation.
  • Exercice 1 : Edition du fichier HTML vous indique comment éditer le fichier HTML que Java Development Environment crée lorsque vous exécutez un projet.
  • Exercice 2 : Mise à jour de la page d'exécution décrit comment exécuter le fichier HTML et non le fichier par défaut Blink.tmp.html.
