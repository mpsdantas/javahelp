Menu Fichier

Nouveau Vide le contenu de l'éditeur de source. Vous pouvez entrer du texte et l'enregistrer comme nouveau fichier. Pour plus d'informations sur l'éditeur de source, voir Edition de code source.
Ouvrir Ouvre un fichier existant dans l'éditeur de source.
Nouvelle fenêtre éditeur Ouvre une nouvelle fenêtre d'éditeur de source sans aucun fichier chargé.
Fenêtre éditeur clone Ouvre une nouvelle fenêtre éditeur chargée avec le même fichier que celui contenu dans la fenêtre éditeur d'où le clonage a été effectué. Ce processus peut être répété plusieurs fois. Voir Edition élémentaire pour plus d'informations.
Recharger le fichier source Recharge le fichier actuellement chargé dans l'éditeur de source.
Enregistrer Enregistre le fichier actuellement chargé dans l'éditeur de source en utilisant son nom actuel.
Enregistrer sous Enregistre le fichier actuellement chargé dans l'éditeur de source en utilisant le nom fourni dans la boîte de dialogue du sélecteur de fichier.
Enregistrer tous les fichiers source Enregistre tous les fichiers actuellement chargés dans les fenêtres actives de l'éditeur de source.
Version Commandes de contrôle de version. Voir Contrôle de version pour plus d'informations.
Liste des fichiers Java Development Environment maintient une liste des fichiers chargés dans l'éditeur de source. Quand vous chargez un fichier dans l'éditeur, son nom est ajouté en tête de la liste. Sélectionnez un fichier dans cette liste pour le charger.

Cette liste répertorie les 12 derniers fichiers édités dans l'ordre de leur chargement dans l'éditeur. Cette liste est maintenue d'une session Java Development Environment à l'autre. Si un fichier est supprimé du système de fichiers, il est retiré de la liste.

Sortie Quitte Java Development Environment.
