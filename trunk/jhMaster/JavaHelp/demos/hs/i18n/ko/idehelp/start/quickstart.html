자습서 1: 프로젝트 만들기

본 자습서는 Java Development Environment을 처음 사용하는 사람을 위한 것입니다. 본 자습서에서는, 프로젝트가 무엇이며, Java Development Environment에서 프로젝트를 만들고, 빌드하고, 실행하는 방법이 무엇인지를 익힐 것입니다.

자습서 1은 다섯 개의 과로 나누어져 있습니다. 자습서에서 최대의 유익을 얻으려면, 각 과를 차례대로 익혀야 합니다.

  • 중요한 개념: 프로젝트와 포트폴리오는 프로젝트와 포트폴리오의 작동 원리를 완벽하게 설명합니다. 본 자습서를 계속 진행하기 전에 이러한 개념들을 이해해야 합니다.
  • 제 1과: Blink 프로젝트 만들기에서는 애플릿이 들어 있는 프로젝트를 만드는 방법을 보여 줍니다. 애플릿은 Java 언어를 지원하는 브라우저에서 실행하는 자그마한 프로그램입니다.
  • 제 2과: 프로젝트 빌드하기에서는 애플릿 프로젝트를 빌드하는 방법을 보여줍니다.
  • 제 3과: 프로젝트 실행하기에서는 프로젝트를 실행하고 애플릿 뷰어에 프로젝트를 나오게 하는 방법을 보여 줍니다.
  • 제 4과: 성능 분석 데이터 수집하기에서는 성능 분석 정보를 수집하는 방법을 보여줍니다.
