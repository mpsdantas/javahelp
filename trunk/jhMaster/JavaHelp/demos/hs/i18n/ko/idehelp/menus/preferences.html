[기본 설정] 메뉴

[프록시] 프록시 서버 (프록시)를 변경하려면 [프록시] 대화 상자를 사용하십시오.
[버전 제어] 이 메뉴 명령을 선택하면 Java Development Environment에서 사용할 버전 제어 시스템이 선택됩니다.
[소스 편집기] 이 메뉴 명령을 선택하면 다음과 같은 소스 편집기 동작 환경을 설정할 수 있습니다:
  • [들여쓰기/탭]
  • [구문 반전 표시 및 컬러]
  • [글꼴 선택]
[키보드] 이 메뉴 명령을 선택하면 Java Development Environment 내의 기능 그룹에 대한 키 할당 (바인딩)을 검토, 추가, 삭제할 수 있습니다. 더 자세한 내용은 키보드 환경 설정을 참조하십시오.
[콘솔 인풋] 이 메뉴 명령을 선택하면 명령행 방식 프로그램 입력을 [콘솔] 창에서 해석하는 방식을 결정할 수 있습니다. 더 자세한 내용은 [콘솔 인풋] 대화 상자를 참조하십시오.
[빌더] 이 메뉴 명령을 선택하면 Java Development Environment이 프로젝트를 컴파일하는데 사용하는 Java Compiler를 결정할 수 있습니다. 더 자세한 내용은 [빌더 기본 설정] 대화 상자를 참조하십시오.
[도구 모음에 레이블 표시] 이 메뉴 명령을 선택하면 도구 모음 단추 밑에 레이블을 추가합니다.
