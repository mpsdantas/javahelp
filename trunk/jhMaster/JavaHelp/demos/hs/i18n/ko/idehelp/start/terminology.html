Java Development Environment 용어

다음 용어들이 온라인 도움말에서 사용됩니다. 이 용어들을 이해하면 자습서를 보다 빠르게 익히는데 도움이 될 것입니다. (애플릿과 스레드와 같은) Java Development Environment과 관련된 다른 용어들은 그 용어들이 자습서에서 처음 사용될 때 설명합니다.

자습서를 진행할 때, 브라우저의 오른쪽 위 모서리에 있는 동화상 아이콘을 잘 살펴 보아야 합니다. Java Development Environment이 도움말 페이지를 로딩하고 있는 동안 이 입방체가 회전을 합니다.

Java Development Environment 주 도구 모음
창의 제일 위에서 메뉴 표시줄 바로 아래 있는 영역. Java Development Environment 주 도구 모음은 다음과 같은 모양입니다:

편집/디버그 도구 모음
소스 편집기 창의 위에 있는 도구 모음. 편집/디버그 도구 모음은 다음과 같은 모양입니다:
단추
마우스로 눌러 동작이 실행되게 할 수 있는 도구 모음이나 창에 있는 요소. 예를 들어, [프로젝트] 단추 를 마우스로 누르면 [Java Development Environment 프로젝트 관리자] 창이 화면에 나타납니다.

도구 모음 단추들에 대해 도구 도움말이 마련되어 있습니다. 도구 모음의 단추가 어떤 동작을 수행하는지 잘 모르면, 그 단추 위로 마우스 포인터를 이동시켜 해설이 나타나게 하십시오. 예를 들어, 마우스 포인터를 단추 위로 옮기면, "[문자열 찾기]"라고 화면에 표시됩니다.

메뉴
특정한 작업이나 객체를 선택할 수 있는 항목 목록. 메뉴는 브라우저에서 제목 표시줄과 Java Development Environment 주 도구 모음 사이에 나타납니다. 메뉴의 항목을 언급할 때는 [메뉴] -> [메뉴 항목]의 형태로 표현합니다. 예를 들어, [프로젝트] -> [빌드]는 "[프로젝트] 메뉴의 [빌드] 항목을 선택하십시오"라는 의미입니다.
탭
창에서 탭을 마우스로 누르면 화면에 나타나는 (대화 상자와 비슷한) 꼬리표가 붙은 페이지. 예를 들어, [프로젝트 편집] 창에는 [일반], [빌드], [디버그/찾아보기], [실행] 등의 탭이 있습니다.
