제 2과: 중단점 설정하기

중단점을 설정하는 것은 프로그램에서 문제가 발생하는 지점을 추적해 나가는데 도움이 됩니다. 이 과에서는, 프로그램을 실행한 상태에서 Blink.java에 중단점을 설정하게 될 것입니다. 중단점이 설정되면 프로그램이 중단점까지 실행한 다음 멈춥니다.

본 자습서에서 사용하는 소스 라인 번호는 사용자가 소스 코드 수정하기에 나오는 추가 코드를 추가했다고 가정한 라인 번호입니다.

  1. [디버그] -> [중단점]을 선택하십시오.
    [Java Development Environment 중단점] 창이 화면에 나타납니다.
  2. [클래스 이름} 란에 Blink라고 입력하십시오.
  3. [위치] 란에 45라고 입력하십시오.
  4. [중단점 설정]을 마우스로 누르십시오.
    소스 편집기에서 해당 라인이 반전 표시됩니다: 
         if (x + w > d.width) {

    이 라인이 이제 선택되었으므로 프로그램이 루프를 따라 진행할 때 변수의 값의 변화를 추적할 수 있습니다. 대부분의 경우, 프로그램에서 버그가 발생한다고 생각하는 라인에 중단점을 설정할 것입니다.

프로그램이 중단점에 도달할 때까지 프로그램이 실행됩니다. 새 중단점이 들어 있는 라인이 초록색으로 바뀌어 프로그램 실행이 멈춘 위치를 알려 줍니다. [스레드/스택] 창이 갱신됩니다.
