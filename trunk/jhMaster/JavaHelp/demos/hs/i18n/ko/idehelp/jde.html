Java Development Environment 온라인 도움말

Java Development Environment 도움말 시스템은 다음과 같은 주요 항목들로 체계화되어 있습니다. Java Development Environment [도움말] 메뉴에서 이 항목들로 곧바로 들어갈 수 있습니다.
  • [프로젝트 관리하기]--프로젝트와 포트폴리오를 만들어서 사용하는 법
  • [프로젝트 빌드하기]--Java 프로그램을 빌드하는 법
  • [프로젝트 디버깅하기]--Java 프로그램을 디버깅하는 법
  • [프로젝트 성능 분석하기]--Java 프로그램의 성능을 분석하는 법
  • [소스 편집하기]--소스 파일을 편집하는 법
  • [GUI 만들기]--그래픽 방식 인터페이스를 구성하고 생성하는 법
  • [소스 찾아보기]--소스 파일을 찾아 보는 법

[목차] 단추를 마우스로 누르면 도움말 항목들의 계층 구조 목록을 살펴 볼 수 있습니다. 닫힌 책 모양 아이콘을 마우스로 누르면 항목 계층 구조 표시가 확장되고 열린 책 모양 아이콘을 마우스로 누르면 항목 계층 구조 표시가 다시 축소됩니다. [목차]를 화면 이동 도구로 사용할 수도 있습니다--[목차] 단추를 마우스로 누르면 현재 커서가 위치한 페이지가 계층 구조 상에서 반전 표시되므로 그 페이지의 위치를 알 수 있습니다.
