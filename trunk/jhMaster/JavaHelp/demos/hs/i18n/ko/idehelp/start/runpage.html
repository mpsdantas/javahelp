제 2과: 실행 페이지 수정하기

이 과에서는, 기본 파일인 Blink.tmp.html이 아니라 새로 만들어진 HTML 파일을 실행하게 될 것입니다.
  1. 프로젝트 관리자에서 현재 프로젝트의 이름을 확인하십시오.
    현재 프로젝트가 Blink가 아니라면, Blink라는 프로젝트 이름을 마우스로 두 번 연속 누르십시오.
  2. [프로젝트] -> [편집]을 선택하십시오.
    [프로젝트 편집] 창이 화면에 나타납니다.
  3. [실행] 탭을 마우스로 누르십시오.
  4. [이미 존재하는 실행 페이지 URL]에 대해 [예]를 마우스로 누르십시오.
    창에서 [이미 존재하는 실행 페이지 URL]을 제외한 모든 필드가 흐리게 표시됩니다. [이미 존재하는 실행 페이지 URL]을 [예]로 설정하면, Java Development Environment은 프로젝트를 실행할 때 [실행] 탭의 정보를 무시합니다.
  5. [이미 존재하는 실행 페이지 URL] 란에, 수정된 HTML 파일에 방금 지정한 새 이름을 입력하십시오.
    전체 경로 이름을 입력할 필요는 없습니다. Java Development Environment이 파일에 도달할 수 있는 소스 디렉터리로 갈 것입니다.
  6. [확인]을 마우스로 누르십시오.
  7. 주 도구 모음에서 [실행] 단추를 마우스로 누르십시오.
    새 HTML 파일이 브라우저로 로딩됩니다. 이제 Blink 프로젝트가 보다 빠른 속도로 실행되는 것을 알 수 있을 것입니다.
