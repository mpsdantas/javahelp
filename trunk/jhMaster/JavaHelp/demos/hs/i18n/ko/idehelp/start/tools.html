Java Development Environment의 도구들

Java Development Environment의 도구들은 (메뉴 표시줄 밑에 있는) Java Development Environment 주 도구 모음에 단추로 표시되어 있습니다.

주 도구 모음 단추들은 다음과 같습니다:

[빌드]
현재 프로젝트를 빌드합니다. 프로젝트에 오류가 포함되어 있으면, [빌드] 탭에 해당 오류와 소스 코드로 연결되는 링크가 나타납니다.
[실행]
현재 프로젝트를 실행합니다. 프로젝트가 애플릿이면, HTML 파일이 만들어져 애플릿 뷰어에서 실행됩니다. 프로젝트가 독립형 프로그램이면, 콘솔로 출력이 될 것입니다. Java 패키지나 Bean은 실행할 수 없습니다.
[성능 분석]
성능분석기를 실행합니다. 각 프로그램 메서드에서 얼마나 많은 시간이 사용되었으며 각 메서드가 몇 번이나 실행되는지를 알려 주는 보고서를 만듭니다.
[검색]
지정된 소스 파일에서 텍스트의 문자열에 대한 참조를 찾습니다.
[프로젝트]
프로젝트를 만들고, 불러오고, 수정하고, 삭제합니다.

프로젝트는 Java Development Environment의 기초입니다. 프로젝트는 Java Development Environment에서 프로젝트를 빌드하고, 찾아보고, 실행하고, 디버깅하는데 필요한 여러 가지 파일들에 대한 기록을 가지고 있습니다. 프로젝트를 그룹으로 묶어서 포트폴리오에 넣습니다. Java Development Environment에서는 다음과 같은 프로젝트 유형을 사용할 수 있습니다:

  • 애플릿. Java 언어로 만들어져 HotJava와 같이 Java를 지원하는 브라우저에서 실행이 되는 프로그램.
  • 독립형 프로그램. 웹 브라우저 밖에서, 즉, 보통은 MS-DOS 명령창이나 UNIX 운영 환경의 쉘 도구에서 실행이 되는 Java 언어로 만든 프로그램
  • Java 패키지. 동일한 패키지 이름으로 선언이 된 Java 클래스 그룹
  • Java Beans. GUI 밀더에서 사용할 수 있는 재사용 가능형 소프트웨어 컴포넌트
[속성]
GUI 빌더 프로젝트에서 인터페이스 컴포넌트의 프로퍼티를 수정합니다.
[도움말]
Java Development Environment에 관한 정보를 알려줍니다. 여러 가지 방법으로 도움말을 볼 수 있습니다:
  • 주 도구 모음에 관한 정보를 보려면 Java Development Environment 주 도구 모음에 있는 [도움말] 단추(물음표)를 마우스로 누르십시오.
  • 편집/디버그 도구 모음 단추들에 관한 정보를 보려면, 편집/디버그 도구 모음에 있는 [도움말] 단추(물음표)를 마우스로 누르십시오.
  • [도움말] 메뉴에서, 도움말을 보려는 항목을 선택하십시오.
  • 대화 상자나 탭에서, [도움말] 단추를 마우스로 누르십시오.

단추가 어떤 기능을 수행하는지 잘 모르면, 마우스 포인터를 그 단추 위로 옮겨서 단추에 대한 해설이 나오게 할 수 있습니다. 해설은 브라우저의 메시지 표시줄에 나타납니다.
