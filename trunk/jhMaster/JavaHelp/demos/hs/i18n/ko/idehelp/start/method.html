제 2과: 프로젝트에 있는 메서드 보기

이 과에서는, Blink 클래스에 정의되어 있는 메서드 목록을 소개합니다. 이 목록에는 메서드의 소스 코드도 나옵니다. 이 과를 제대로 익히려면, 먼저 앞 과인 클래스 계층 구조 보기를 완벽하게 익혀야 합니다.
  1. 클래스 뷰어에서, [생성자] 색인 레이블까지 쭉 내려가십시오.
    생성자는 만들어질 때 객체를 초기화하는 특수한 메서드입니다. 클래스 뷰어에는 Blink 클래스 내에 정의되어 있는 하나의 생성자, 즉 Blink가 나옵니다.
  2. [Blink]를 마우스로 누르십시오.
    소스 편집기에는 생성자 Blink의 소스 코드가 나타납니다. 생성자의 첫번째 줄이 파랑색으로 반전 표시되어 있습니다.
  3. 클래스 뷰어에서, [메서드] 색인 레이블까지 쭉 내려가십시오.
    Blink 클래스에는 init, paint, start, stop, 그리고 run 등의 다섯 가지 메서드가 있습니다.
  4. paint를 마우스로 누르십시오.
    소스 편집기에 paint 메서드의 소스 코드가 나타납니다. 첫번째 라인이 파랑색으로 반전 표시됩니다.
