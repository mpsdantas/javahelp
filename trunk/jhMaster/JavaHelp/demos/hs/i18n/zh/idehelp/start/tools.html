Java Development Environment工具

Java Development Environment工具在Java Development Environment主工具栏（位于菜单栏的下面）上都用按钮来表示。

主工具栏中的按钮包括：

生成
生成当前项目。如果项目中有错误，“生成”制表项会报告错误并链接到源代码上。
运行
运行当前项目。如果该项目是一个applet小程序，则会创建一个HTML文件，并在applet小程序查看器中运行。 如果它是一个独立程序，其输出内容会出现在控制台上。您不可以运行Java包或程序节。
简要表
运行“简要表”。将生成一份报告，指出每个程序方法所耗费的时间，以及每个方法执行的次数。
查找
在指定源文件中查找字符串。
项目
创建、引入、编辑和删除项目。

项目是Java Development Environment的基础。它负责记录在Java Development Environment中生成、浏览、运行和调试项目所需的各种文件。 项目都被集中放入公文夹中。Java Development Environment可以使用下列项目类型：

  • Applet小程序. 一个用Java语言编写，并在能支持Java的浏览器 （比如HotJava）中运行的程序
  • 独立程序. 一个用Java语言编写，并在web浏览器以外运行的程序，通常它会从 MS-DOS命令窗口或UNIX操作环境下的shell工具中运行
  • Java包. 一组用同一包名进行声明的Java类
  • Java程序节. 可以在GUI生成器中操作的可再用软件组件
属性
编辑GUI生成器项目中界面组件的属性。
帮助
提供关于Java Development Environment的信息。有以下几种获取帮助的不同方法：
  • 单击Java Development Environment主工具栏中的“帮助”按钮（问号），可以获取有关主工具栏按钮的信息。
  • 单击“编辑/调试”工具栏中的“帮助”按钮（问号），可以获取有关“编辑/调试”工具栏按钮的信息。
  • 从“帮助”菜单中选择需要帮助的项。
  • 在对话框或制表项中，单击“帮助”按钮。

如果您对一个按钮所执行的动作不太了解，可以将指针指向它，以获取有关该按钮的描述。 描述信息将出现在浏览器的注脚中。
