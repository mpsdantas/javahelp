练习2：复制和粘贴项目

本练习将指导您学习如何把项目复制到个人公文夹中，以及如何把它 粘贴到公用公文夹中。如果您尚未创建公用公文夹，现在就应该着手进行创建了。 详细的说明请参看创建公文夹。

复制一个项目

  1. 选择“公文夹” => “选择” => “个人”。
    Java Development Environment将打开您的个人公文夹。如果您没有个人公文夹， 请参阅引入个人公文夹。

  2. 单击WorkShop工具栏上的“公文夹管理程序”控制 。

  3. 选择Blink项目。
    如果没有Blink项目，请参阅 创建项目中有关如何创建Blink项目的说明。

  4. 单击“公文夹管理程序”工具栏上的“复制”控制 。
    WorkShop会将Blink项目复制到剪贴板上。

粘贴一个项目

  1. 选择“公文夹”=>“选择”，然后选择公用公文夹的名字。
    WorkShop会打开您的公用公文夹。

  2. 单击“公文夹管理程序”工具栏上的“粘贴”控制 。
    WorkShop会打开“粘贴项目”对话框。Blink项目的路径名就出现在“项目文件”域中。

  3. 确保选中了位于“创建自”域右边的“复制”双值按钮。
    此按钮会通知WorkShop把所有项目文件 （Blink.prj、Blink.java、 Blink.class）都复制到目标目录中。 WorkShop不会复制缺省的HTML文件，Blink.tmp.html。

  4. 在目标目录中，输入用来保存文件的目录的路径名。
    这个目录必须是HTTP 服务器可以存取的目录。

  5. 单击“包括源代码”域右边的“是”双值按钮。

  6. 确保Blink出现在“新项目名”域中。

  7. 单击“粘贴”按钮。
    项目的名字和图象会出现在公用公文夹中。项目文件连同两个附加文件：Blink.jws.src和Blink.jws被一起添加到目标目录中。第一个文件包括此项目的源文件，而第二个文件包含所有其它的文件，比如.prj文件。如果有其他用户要复制这个项目，WorkShop就会复制.jws.src和.jws文件，而不是复制单个项目文件。
