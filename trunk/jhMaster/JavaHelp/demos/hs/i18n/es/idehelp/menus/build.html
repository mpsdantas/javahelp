Menú "Construir"

Construir
Construye el proyecto actual utilizando la información especificada en la definición del proyecto. Este comando también tiene un botón de acceso rápido en la barra de herramientas principal.
Construir todo Obliga a construir el proyecto, aunque los archivos estén actualizados.
Compilar archivo Compila el archivo actualmente cargado en la ventana principal del editor de código fuente. Véase Compilación de un solo archivo para más información.
Parar construcción Para la construcción en curso.
Siguiente error Desplaza la ventana del editor hasta la línea que contiene el siguiente error de construcción.
Error anterior Desplaza la ventana del editor hasta la línea que contiene el error de construcción anterior.
