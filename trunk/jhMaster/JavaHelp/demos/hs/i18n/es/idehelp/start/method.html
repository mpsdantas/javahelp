Ejercicio 2: Visualización de los métodos de su proyecto

En este ejercicio, listará los métodos definidos en la clase Blink. En esta lista, podrá visualizar el código fuente del método. Para completar este ejercicio con los resultados esperados, deberá haber completado antes el ejercicio anterior, Visualización de la jerarquía de clases.
  1. En el visualizador de clases, desplácese hacia abajo por la lista hasta la etiqueta "Índice de constructores".
    Un constructor es un método especial que inicializa un objeto cuando lo crea. El visualizador de clases lista un constructor, Blink, definido dentro de la clase Blink.
  2. Pulse en Blink.
    El editor de código fuente muestra el código fuente para el constructor Blink. La primera línea del constructor se resalta en azul.
  3. En el visualizador de clases, desplácese hacia abajo por la lista hasta la etiqueta "Índice de métodos".
    La clase Blink contiene cinco métodos: init, paint, start, stop, y run.
  4. Pulse en paint.
    El editor de código fuente muestra el código fuente para el método paint. La primera línea se resalta en azul.
