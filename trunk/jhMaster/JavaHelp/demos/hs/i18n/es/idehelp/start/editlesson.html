Ejercicio 1: Edición de atributos de ejecución

La etiqueta <applet> que generó en el Tutorial Uno especifica los requisitos mínimos para colocar el applet en una página Web. En este ejercicio, añadirá parámetros que le permitirán personalizar la forma de visualizar el applet.

Este ejercicio funciona con el proyecto Blink. Si no ha creado y construido aún el proyecto Blink, debe hacerlo ahora utilizando las instrucciones de Creación del proyecto Blink.

  1. Compruebe el nombre del proyecto actual en la ventana del administrador de proyectos.
    Si el proyecto actual no es Blink, pulse dos veces en en el nombre del proyecto Blink.
  2. En el administrador de proyectos, seleccione "Proyecto -> Editar".
    La ventana "Editar proyecto" muestra cuatro pestañas: "General", "Construir", "Depurar/Examinar" y "Ejecutar".
  3. Pulse en la pestaña "Ejecutar".
    Los atributos de esta pestaña se utilizan para generar la etiqueta <applet> en el archivo Blink.tmp.html.
  4. Busque el campo "Parámetros" (en la mitad de la ventana).
    Actualmente no hay parámetros definidos para el applet. Un parámetro es un atributo no estándar que define cómo se visualiza el applet. Un parámetro tiene un nombre y un valor. Añadirá un parámetro para cambiar las palabras del texto intermitente.
  5. Escriba lbl (una abreviatura de "label"), en el cuadro de texto "Nombre" a la derecha del campo "Parámetros".
  6. Escriba lo siguiente en el cuadro "Valor:"
    Java Development Environment es estupendo :-)
  7. Pulse en el botón "Añadir".
    La línea lbl=Java Development Environment es estupendo :-) aparece en el campo "Parámetros".
  8. Pulse en "Aceptar" para guardar los cambios que ha hecho.
  9. Pulse en el botón "Ejecutar" en la barra de herramientas principal de Java Development Environment.
    Java Development Environment incluye la línea siguiente: 
    <param name="lbl" value="Java Development Environment es estupendo :-)">
    en la etiqueta <applet> en el archivo Blink.tmp.html. El archivo se carga en el examinador, y ahora puede ver las palabras "Java Development Environment es estupendo :-)" de forma intermitente en diferentes colores.

    Para detener la ejecución del applet, seleccione "Applet -> Salir" en la ventana "Visualizador de applets".
