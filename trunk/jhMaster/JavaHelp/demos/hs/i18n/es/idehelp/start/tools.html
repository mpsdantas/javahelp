Herramientas de Java Development Environment

La herramientas de Java Development Environment están representadas por botones en la barra de herramientas principal de Java Development Environment (debajo de la barra de menús).

Los botones de la barra de herramientas principal son:

Construir
Construye el proyecto actual. Si el proyecto contiene errores, la pestaña "Construir" informa de los errores y enlaces con el código fuente.
Ejecutar
Ejecuta el proyecto actual. Si el proyecto es un applet, se crea un archivo HTML y se ejecuta en el visualizador de applets. Si el proyecto es un programa autónomo, la salida va a la consola. No puede ejecutar un paquete Java ni un bean.
Perfil
Ejecuta el Perfilador. Produce un informe que muestra cuánto tiempo se ha dedicado a cada método del programa y cuántas veces se ejecuta cada método.
Buscar
Busca referencias a cadenas de texto en archivos de código fuente especificados.
Proyectos
Crea, importa, edita y suprime proyectos.

Los proyectos son la piedra angular de Java Development Environment. Un proyecto hace el seguimiento de los diversos archivos necesarios para construir, examinar, ejecutar y depurar un proyecto en Java Development Environment. Los proyectos se agrupan en carpetas. Los siguientes tipos de proyectos están disponibles en Java Development Environment:

  • Applet. Programa escrito en lenguaje Java que se ejecuta en un examinador que soporte Java, como HotJava.
  • Programa autónomo. Programa escrito en lenguaje Java que se ejecuta fuera del examinador de Web, normalmente desde una ventana de MS-DOS o una herramienta del shell en el entorno operativo UNIX.
  • Paquete Java. Grupo de clases Java declarado con el mismo nombre de paquete.
  • Java Beans. Componentes de software reutilizables que pueden utilizarse en el constructor de GUI.
Atributos
Edita propiedades de componentes de la interfaz en proyectos del constructor de GUI.
Ayuda
Proporciona información acerca de Java Development Environment. Puede obtener ayuda de varias formas diferentes:
  • Pulsando en el botón "Ayuda" (el signo de interrogación) en la barra de herramientas principal de Java Development Environment para obtener información acerca de los botones de la barra de herramientas principal.
  • Pulsando en el botón "Ayuda" (el signo de interrogación) en la barra de herramientas principal de edición/depuración para obtener información acerca de los botones de la barra de herramientas de edición/depuración.
  • En el menú "Ayuda", seleccionando un elemento para el que desee ayuda.
  • En un cuadro de diálogo o pestaña, pulsando en el botón "Ayuda".

Si no está seguro de qué acción realiza un botón, puede obtener una descripción del botón colocando el puntero encima de éste. La descripción aparece en el pie del examinador.
