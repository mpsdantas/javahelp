Menú "Depurar"

Iniciar/Reiniciar Inicia o reinicia la depuración del proyecto actual.
Reanudar Reanuda la ejecución del proyecto actual.
Ejecutar hasta aquí Ejecuta el proyecto actual hasta una ubicación especificada.
Analizar por procedimiento Analiza un método procedimiento a procedimiento.
Analizar línea a línea Analiza un método línea a línea.
Evaluar
 Evalúa un valor de dato.
Mostrar método de llamada Desplaza la pila de llamada un nivel hacia abajo.
Mostrar método llamado Desplaza la pila de llamada un nivel hacia arriba.
Subprocesos/Pila Abre la ventana "Subprocesos/Pila".
Conmutar punto de interrupción
 Establece o elimina un punto de interrupción en la línea donde se encuentra el cursor.
Puntos de interrupción Abre la ventana "Puntos de interrupción".
Excepciones Muestra una lista de condiciones existentes que maneja el depurador.
Parar depuración Termina la sesión de depuración.
