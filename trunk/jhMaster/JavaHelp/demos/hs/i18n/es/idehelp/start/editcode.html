Ejercicio 1: Edición de código fuente

Como introducción al trabajo con el editor de código fuente, introducirá código en el archivo Blink.java para hacer subir y bajar las palabras.

Si no ha creado aún el proyecto Blink, debe hacerlo ahora utilizando las instrucciones de Creación del proyecto Blink.

  1. Compruebe el nombre del proyecto actual en la ventana del administrador de proyectos de Java Development Environment.
    Si el proyecto actual no es Blink, pulse dos veces en en el nombre del proyecto Blink.
  2. Escriba debajo las líneas de código fuente.
    Este código tiene errores intencionados porque la siguiente lección muestra la forma de corregirlos.

    Puede ir a un número de línea específico seleccionando "Editar -> Ir a línea". El número de línea del puntero aparece a la derecha del pie.

    lbl = (att == null) ? "Blink" : att;
