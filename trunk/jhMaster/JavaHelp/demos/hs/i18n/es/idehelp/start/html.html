Ejercicio 1: Edición del archivo HTML

En esta lección, editará el archivo HTML que Java Development Environment creó cuando ejecutó el proyecto Blink. Especificará un parámetro de applet y añadirá un enlace al código fuente. Una vez haya editado el archivo, lo guardará con un nombre de archivo diferente para que los cambios no se sobreescriban la próxima vez que ejecute el proyecto.

Si no ha creado ni construido aún el proyecto Blink, debe hacerlo ahora utilizando las instrucciones de Creación del proyecto Blink.

  1. Compruebe el nombre del proyecto actual en la ventana del administrador de proyectos.
    Si el proyecto actual no es Blink, pulse dos veces en el nombre del proyecto Blink.
  2. En el editor de código fuente, seleccione "Archivo -> Abrir".
    Java Development Environment abre un selector de archivos.
  3. Abra el archivo Blink.tmp.html en el directorio Blink.
    El archivo contiene una etiqueta de applet que tiene un aspecto similar al siguiente: 
    <applet
   name="Blink"
   code="Blink.class"

codebase="/su_directorio_principal/jws/Blink"
   width="500"
   height="600"
   align="Top"
   alt="Si tiene un examinador habilitado para java,
        aquí verá una applet."
> 
<param name="lbl" value="Java Development Environment es estupendo :-)">
<hr>Si el examinador hubiera reconocido la etiqueta de applet,
   aquí vería una applet.<hr>
</applet>

    Java Development Environment creó este archivo cuando ejecutó el proyecto. La información contenida en este archivo se basa en la información de la pestaña "Ejecutar" de la ventana "Editar proyecto". Si no ha ejecutado con anterioridad el proyecto, véase Ejecución del proyecto.

  4. Seleccione "Archivo -> Guardar como" en la barra de menús de Java Development Environment.
    Aparece el cuadro de diálogo "Guardar archivo actual".
  5. Seleccione el directorio Blink; después, escriba un nuevo nombre para el archivo HTML y pulse en "Aceptar".
    Asegúrese de cambiar el nombre del archivo HTML; de lo contrario, los cambios se sobreescribirán la próxima vez que ejecute el proyecto. Asegúrese también de guardar el archivo en el directorio Blink.
  6. En el editor de código fuente, cambie el valor de altura a 75.
  7. Añada el siguiente parámetro antes del parámetro lbl para aumentar la velocidad de la intermitencia de las palabras: 
    <param name="speed" value="50">
  8. Añada el siguiente enlace con el código fuente después de la etiqueta <applet>: 
    <p>
<a href="Blink.java">Source Code</a>
    La ruta del código fuente debe ser relativa al directorio que contiene al archivo HTML.
  9. Pulse en el botón "Guardar" en la barra de herramientas de edición/depuración.
