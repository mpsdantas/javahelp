Ejercicio 3: Ejecución del proyecto

En este ejercicio, ejecutará el proyecto Blink utilizando el archivo HTML creado por Java Development Environment.
  1. Consulte en el administrador de proyectos el nombre del proyecto actual.
    Si el proyecto actual no es Blink, pulse dos veces en el nombre del proyecto Blink.
  2. Pulse en el botón "Ejecutar" en la barra de herramientas principal de Java Development Environment.
    Java Development Environment generará el archivo HTML en su directorio Blink. Este archivo HTML incluye la etiqueta <applet> en la que se especifican las opciones que deberán utilizarse al colocar el proyecto Blink en una página Web. Java Development Environment cargará el archivo en el visualizador de applets, y aparecerá el proyecto Blink en ejecución. La palabra Blink parpadeará intermitentemente, en distintos colores.
  3. Para detener la ejecución del applet, seleccione "Applet -> Salir" en la ventana del visualizador de applets.
