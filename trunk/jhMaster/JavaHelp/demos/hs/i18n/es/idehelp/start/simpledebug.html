Tutorial Cinco: Depuración de un proyecto

Este tutorial es una introducción a la depuración en Java Development Environment. Aprenderá a examinar subprocesos en un programa, definir puntos de interrupción, y examinar los datos de su programa.

El tutorial consta de siete lecciones. Deberá realizar los ejercicios en el orden en que se presentan.

  • Conceptos importantes: Subprocesos define brevemente los subprocesos y los grupos de subprocesos. Deberá entender bien estos conceptos antes de pasar a los ejercicios.
  • Ejercicio 1: Inicio de una sesión de depuración le enseña a empezar a depurar un proyecto.
  • Ejercicio 2: Definición de un punto de interrupción explica cómo detener la ejecución de un programa en un punto específico del código.
  • Ejercicio 3: Examen de subprocesos muestra cómo visualizar los grupos de subprocesos, subprocesos, y tramas de pilas que constituyen un proyecto.
  • Ejercicio 4: Análisis de un método muestra cómo realizar el análisis del código.
  • Ejercicio 5: Evaluación de expresiones muestra cómo evaluar los datos de su programa.
  • Ejercicio 6: Finalización de una sesión de depuración muestra cómo detener una sesión de depuración.
