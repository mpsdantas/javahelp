Ejercicio 1: Inicio de una sesión de depuración

En este ejercicio se explica cómo iniciar la depuración del proyecto Blink. Si aún no ha creado y construido el proyecto Blink, deberá hacerlo ahora siguiendo las instrucciones que se indican en Creación del proyecto Blink.
  1. Consulte en el administrador de proyectos el nombre del proyecto actual.
    Deberá aparecer el archivo Blink.java en el editor de código fuente. Si el proyecto actual no es Blink, pulse dos veces en el nombre del proyecto Blink.
  2. Pulse en el botón "Reiniciar" en la barra de herramientas de edición/depuración.
    Se iniciará la ejecución de Blink.java y Java WorkShop ejecutará el proyecto Blink en la ventana del visualizador de applets.

Mientras está ejecutándose Blink.java, podrá definir un punto de interrupción seleccionando "Depurar -> Puntos de interrupción". También podrá suspender el programa seleccionando "Depurar -> Interrumpir programa".
