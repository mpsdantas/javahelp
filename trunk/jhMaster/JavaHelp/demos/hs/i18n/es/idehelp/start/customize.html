Tutorial 6: Personalización de una página de ejecución

Este tutorial le enseña a editar el archivo HTML que ejecuta un proyecto. Para sacar el máximo partido del tutorial, deberá realizar los ejercicios en el orden en que se presentan.
  • Ejercicio 1: Edición del archivo HTML enseña a editar el archivo HTML que crea Java Development Environment para el usuario cuando ejecuta un proyecto.
  • Ejercicio 2: Actualización de la página de ejecución enseña a ejecutar el archivo HTML editado, en lugar del archivo predeterminado Blink.tmp.html.
