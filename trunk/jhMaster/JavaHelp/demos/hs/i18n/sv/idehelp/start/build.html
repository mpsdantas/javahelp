Övning 2: Bygga upp projektet

Den här övningen beskriver hur du bygger upp projektet Blink. Du bygger upp ett projekt för att du ska kunna köra det. När du bygger projektet kompilerar Java Development Environment projektets Java källkod till en fil som benämns filnamn.class.

Om du ännu inte skapat projektet Blink bör du göra detta nu via instruktionerna i Skapa Blink-projektet.

  1. Sök i Java Development Environments fönster Projekthanteraren efter namnet på aktuellt projekt.
    Om aktuellt projekt inte är link dubbelklickar du på projektnamnet Blink.
  2. Klicka på knappen Bygga i Java Development Environments verktygsfält.
    Java Development Environment kompilerar programmet Blink.java till bytekodfilen Blink.class. Titta på fliken Bygga (under källredigeraren) när Java Development Environment bygger upp ditt projekt. Statusmeddelandet "Klar" visas i fliken Bygga när dina källor är färdigbyggda och kompilerade.
