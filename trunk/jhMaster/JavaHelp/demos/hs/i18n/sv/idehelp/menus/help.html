Hjälp-menyn

Innehåll Öppnar innehållsförteckningen till online-hjälpen.
Uppgiftsbaserad hjälp Visar en meny med hjälp om vanliga frågor.
Meny-hjälp Visar en lista med hjälpavsnitt om menyerna i Java WorkShop.
Java Development Environment Självstudie Startar självstudierna till Java Development Environment. De består av åtta interaktiva kurser som steg för steg visar hur du skapar, bygger, kör och felsöker miniprogram.
Versionsinformation Innehåller senkommen information om den aktuella utgåvan av Java WorkShop.
Serienummer/Licens Visar serienumret på och licensinformation om Java Development Environment.
Registrering Öppnar dialogrutan Registrering i Java Development Environment. I denna dialogruta kan du registrera din kopia av Java Development Environment elektroniskt.
Skicka kommentarer Öppnar dialogrutan Kommentarer. I den här rutan kan du på elektronisk väg skicka kommentarer och förslag till Java Development Environment-teamet.
Java API-dokumentation (1.1) Visar Java API-dokumentation 1.1 i hjälpfönstret. Du kan också hämta fram denna dokumentation via innehållsförteckningen till online-hjälpen.
Java språkspecifikation Visar innehållsförteckningen till "Java Language Specification" av James Gosling, Bill Joy och Guy Steele.
Om Java Development Environment Öppnar rutan Om. Denna ruta innehåller följande information:
  • Serienummer
  • Versionsnummer
  • Foto på Java Development Environment-utvecklingsteamet
  • Copyrightvillkor
