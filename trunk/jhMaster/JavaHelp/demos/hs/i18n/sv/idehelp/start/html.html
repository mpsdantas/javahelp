Övning 1: Redigera HTML-filen

I den här lektionen ska du redigera HTML-filen som Java WorkShop skapade åt dig när du körde projektet Blink. Du kommer att ange en miniprogramsparameter och lägga till en länk i källkoden. När du redigerar filen kommer du att spara den under ett annat filnamn, så att dina ändringar inte skrivs över nästa gång du kör projektet.

Om du ännu inte har skapat ett Blink-projekt kan du göra det nu, utifrån anvisningarna i Skapa Blink-projektet.

  1. Kontakta projektledaren för att ta reda på vad det aktuella projektet heter.
    Om projektet inte är Blink dubbelklickar du på namnet Blink.
  2. Välj Arkiv -> Öppna i redigeringsprogrammet för källan.
    Då öppnas en filväljare.
  3. Öppna filen Blink.tmp.html i katalogen Blink.
    Filen innehåller miniprogramkod som ser ut ungefär så här: 
    <applet
   name="Blink"
   code="Blink.class"
   codebase="/your_home_directory/jws/Blink"
   width="500"
   height="600"
   align="Top"
   alt="If you had a java-enabled browser,
   you would see an applet here."
> 
<param name="lbl" value="Java Development Environment is really cool :-)">
<hr>If your browser recognized the applet tag,
      you would see an applet here.<hr>
</applet>

    Java Development Environment skapade den här filen när du körde projektet. Informationen i filen baseras på informationen på fliken Kör i fönstret Redigera projekt. Om du inte har kört projektet än kan du få mer information i Köra projektet.

  4. Välj Arkiv -> Spara som på menyraden i Java Development Environment.
    Då visas dialogrutan Spara aktuell fil.
  5. Markera katalogen Blink, ge HTML-filen ett nytt namn och klicka på OK.
    Kom ihåg att byta namn på HTML-filen, för annars kommer dina ändringar att skrivas över nästa gång du kör projektet. Kom dessutom ihåg att spara filen i katalogen Blink.
  6. Gå till redigeringsprogrammet för källan och ändra värdet på höjden till 75.
  7. Lägg till följande parameter före parametern lbl för att öka ordens blinkningshastighet: 
    <param name="speed" value="50">
  8. Lägg till följande länk i källkoden efter koden <applet>: 
    <p>
<a href="Blink.java">Source Code</a>
    Sökvägen till källkoden måste vara relativ katalogen som innehåller HTML-filen.
  9. Klicka på knappen Spara i verktygsfältet för redigering och felsökning.
