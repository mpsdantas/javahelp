Övning 3: Köra projektet

I den här övningen kommer du att köra projektet Blink med hjälp av HTML-filen som Java WorkShop skapat.
  1. Kontrollera i Projekthanteraren efter namnet på aktuellt projekt.
    Om aktuellt projekt inte är Blink dubbelklickar du på projektnamnet Blink.
  2. Klicka på knappen Kör i Java Development Environments verktygsfält.
    Java Development Environment genererar HTML-filen i din katalog Blink. Denna HTML-fil innefattar fliken <miniprogram>, som specificerar alternativen som ska användas vid placering av projektet Blink på en webbsida. Java Development Environment laddar in filen i Applet Viewer och du kan se hur projektet Blink exekveras. Ordet "Blink" blinkar i olika färger.
  3. För att stoppa körningen av miniprogrammet väljer du Miniprogram -> Avsluta i fönstret Applet Viewer.
