Verktyg i Java Development Environment

Verktygen i Java Development Environment motsvaras av knappar i huvudverktygsfältet i Java Development Environment (under menyraden).

Huvudverktygsfältet innehåller följande knappar:

Bygg
Bygger det aktuella projektet. Om projektet innehåller fel visas felen på fliken Bygg, som också innehåller länkar till källkoden.
Kör
Kör det aktuella projektet. Om projektet är ett miniprogram skapas en HTML-fil som körs i miniprogramgranskaren. Om projektet är ett fristående program visas utmatningen i systemfönstret. Du kan inte köra Java-paket eller bönor.
Profil
Kör profileraren. Skapar en rapport som visar hur mycket tid som används i varje programmetod och hur många gånger varje metod exekveras.
Sök
Söker efter referenser till textsträngar i angivna källfiler.
Projekt
Skapar, importerar, redigerar och tar bort projekt.

Projekten är hörnstenarna i Java Development Environment. De håller reda på de olika filer som behövs för att bygga, granska, köra och felsöka projekt i Java Development Environment. Projekten grupperas i portfolio-filer. Följande typer av projekt är tillgängliga i Java Development Environment:

  • Miniprogram. Ett program som har skrivits på språket Java och körs i en webbläsare som kan hantera Java, exempelvis HotJava.
  • Fristående program. Ett program som har skrivits på språket Java och körs utanför en webbläsare, vanligtvis från ett MS-DOS-fönster eller skalverktyg i operativmiljön UNIX.
  • Java-paket. En grupp Java-klasser som har deklarerats med samma paketnamn.
  • Java-bönor. Återanvändbara programvarukomponenter som kan manipuleras i GUI-byggaren.
Attribut
Redigerar egenskaper hos gränssnittskomponenterna i GUI-byggarprojekt.
Hjälp
Ger information om Java Development Environment. Du kan få hjälp på flera olika sätt:
  • Klicka på knappen Hjälp (frågetecknet) i huvudverktygsfältet i Java Development Environment om du vill få information om verktygsknapparna.
  • Klicka på knappen Hjälp (frågetecknet) i verktygsfältet för redigering och felsökning om du vill få information om verktygknapparna i detta verktygsfält.
  • Välj det element som du vill ha hjälp om på Hjälp-menyn.
  • Klicka på knappen Hjälp i en dialoguta eller på en flik.

Om du inte är säker på vilken åtgärd en viss knapp utför, kan du få en beskrivning av knappen genom att placera pekaren på den. Beskrivningen visas i fottexten i webbläsaren.
