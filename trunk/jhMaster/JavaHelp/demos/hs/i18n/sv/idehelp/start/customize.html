Självstudie sex: Specialanpassa en Körsida

I den här självstudien lär du dig att redigera HTML-filen som kör ett projekt. För att få ut så mycket som möjligt av övningen bör du utföra uppgifterna i samma turordning som de presenteras.
  • Övning 1: Redigering av HTML-fil visar hur du ska redigera HTML-filen som Java Development Environment skapar åt dig när du kör ett projekt.
  • Övning 2: Uppdatering av Körsida visar hur du ska köra den redigerade HTML-filen, istället för default-filen Blink.tmp.html.
