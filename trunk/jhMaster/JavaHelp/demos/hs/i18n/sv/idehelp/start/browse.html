Självstudie fyra: Ögna igenom ett projekt

I den här självstudien får du lära dig hur du kan ögna igenom ett projekt. För att få ut så mycket som möjligt av övningen bör du utföra uppgifterna i samma turordning som de presenteras.
  • Övning 1: Titta på klasshierarkin lär dig hur du färdas genom klasshierarkin i ditt projekt.
  • Övning 2: Titta på metoderna i ditt projekt visar dig hur de metoder som definierats i ditt projekt kan listas. I denna lista kan du granska källkoden för varje metod.
  • Övning 3: Hitta strängar i din kod visar hur du kan söka din källkod i alla förekomster i en specificerad strängkonstant.
